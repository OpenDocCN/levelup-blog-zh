设计是令人困惑的，在你的后端摸索之后，这是你最不想担心的事情。这里有 3 个提示，可以给你的用户最好的体验！

# 1.保持简单！

没有人喜欢过饱和的界面，你的设计应该清晰地定义自己。

一个很好的例子是 Medium.com 的主页。

这很好，因为:

*   Navbar 使浏览网站变得很容易。
*   Navbar 清晰简洁。没有多余的！
*   一致的字体
*   清除消息。
*   微妙的颜色。
*   对读者和作者都有意义。
