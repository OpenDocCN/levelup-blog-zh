用番茄工作法更有效地管理你的时间

原文：https://levelup.gitconnected.com/manage-your-time-more-effectively-with-the-pomodoro-technique-2e1276a69cc0?source=collection_archive---------17-----------------------#2022-01-03

提高生产力和更快完成更多任务的秘密技术。

介绍

有效的时间管理有一个秘密。你必须用西红柿而不是小时来思考！这乍一看似乎很傻，但数百万人已经被番茄工作法改变生活的力量所折服。

这种流行的时间管理方法是由弗朗西斯科·西里洛开发的，它要求你在专注的工作时间(pomodoros)和频繁的短暂休息之间进行交替，以优化持续的注意力集中，防止精神疲劳。

试试番茄工作法，如果你:

  • 发现一点点干扰会影响你的工作
  • 总是在最佳生产率点之后工作
  • 享受游戏化的目标设定
  • 有很多开放式的工作，可能会花费无限的时间

番茄工作法

番茄工作法真的很简单。它包含五个简单的步骤，重复几次:

第一步:

创建一个待办事项列表并获取一个计时器

第二步:

把你的计时器设置为25分钟，从一个任务开始。专注于这项任务，直到计时器响起。

第三步:

当课程结束时，标出一个 番茄 并记录你所完成的内容

第四步:

休息5分钟。做一些体育锻炼。出去呼吸一下。

第五步:

吃完五个番茄后，你应该休息更长的时间，休息15-30分钟。

当你在这25分钟的冲刺工作中分解你的任务时，你必须遵循三个重要的规则来充分利用每一个番茄:

  1. 将复杂的工作量分解成不超过四个番茄的较小任务。
  2. 小任务一起去。比一个番茄小得多的任务应该组合在一起，用工作填满整个番茄
  3. 番茄红素设定好了，就得响！pomodoro启动后，不允许中断它查看邮件、聊天或信息。任何想法、任务或要求都必须记下来，以便在番茄大战之后阅读。

有两个重要的补充:

  1. 如果由于某种原因，某个事件是不可避免的，你可以休息5分钟，然后重新开始吃下一个番茄。当这些中断发生时，你应该跟踪它们，然后思考如何在接下来的会话中避免它们。
  2. 如果你在计时器响之前完成了任务，规则仍然适用。利用剩下的时间提高技能或者过度学习。

为什么波莫多洛技术有效？

有几个优点使该方法特别适合于提高您的工作效率

  1. 每个人都可以使用它。从它开始很容易。
  2. 对抗干扰，增加专注。专注于一项任务时，你不会分心。
  3. 你会意识到你的时间去了哪里。每个pomodoro有一个或多个任务
  4. 您可以提高工作效率。检查你能 捕到多少只柚子

调味的其他技巧

为了更好地使用Pomodoro技术进行优化，您可以遵循以下简单的技巧:

提前计划好你的柚子

在每个工作日开始时，您可以为一整天计划所有pomodoros。通常情况下，8小时工作日可以完成16份柚子，但您只需提前计划12-14份柚子即可。留下2-4个pomodoros作为缓冲区，让它装满白天会出现的意外任务，或者如果任务比预期的时间长，它装满必须完成的额外工作。

用柚子的长度做实验

有些工作需要延长期限。25分钟对于设计、编码和测试来说可能太短了。您可以尝试延长工作时间，增加休息时间。有两项重要研究已经考虑了不同的时间段:52-17和90-30

使用应用程序强制执行pomodoros

起初，可能很难遵守Pomodoro技术的规则。您应该使用一个应用程序，告知您在使用Pomodoro技巧时需要休息的时间。

你能使用的最好的应用应该可以根据你的工作时间和休息时间定制。或者，您想要一个在休息时间将您锁在电脑外的应用程序。

在Windows上，您可以使用:大拉伸

在浏览器中，您可以使用 pomofocus.io 或marinaratimer.com

远离你的电脑

如果你在电脑前工作，你应该在休息时间离开。散散步，出去或做一些运动。不要切换到社交媒体或在YouTube上看东西。如果你在家，也许打扫厨房或洗碗。

你做什么并不重要，但你应该远离电脑。这将比呆在电话上或在社交媒体上阅读更令人精神振奋。
