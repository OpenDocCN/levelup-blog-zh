# Python 有什么用？

现在你会看到大家都在夸大 Python 的用途。人们会说，python 可以用在这里和这里和这里。但是你应该到处使用 Python 吗？

简单的答案是否定的。我同意，Python 是一种令人惊叹的语言，但每种编程语言都有它的用例。Python 广泛用于以下任务:

1.  后端开发
2.  系统脚本/自动化脚本
3.  物联网
4.  数据科学和机器学习
5.  软件/游戏开发的一小部分

是时候做决定了。现在，看完所有这些之后，如果你认为 Python 对你有用，那么让我们开始吧。
