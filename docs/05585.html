1.Git fetch和Git pull有什么区别？

git fetch 是告诉本地git从原始文件中检索最新元数据信息的命令。

git pull 做 git fetch 做的事情，并从远程存储库中带来(复制)那些变更。

来源: freeCodeCamp

2.Git rebase是做什么的？

当我们想要从我们的特性分支中取出所有提交，并将它们移动到主分支中时，我们使用 git rebase 。

重定基准通常被用作合并的替代方案。通过将一个分支的提交应用到另一个分支的提交之上，将一个分支重置为另一个分支来更新一个分支。

来源:Rebase——最强大的Git命令之一

3.在HTML中，div标签和span标签有什么区别？

<div>:块级，主要用于组织页面的布局。

<span>:行内级，主要用于对文本的一部分进行样式化。

4.CSS厂商前缀是什么？

供应商前缀是浏览器让CSS开发人员访问尚不稳定的新功能的一种方式。

来源: CSS厂商前缀

5.JavaScript是单线程还是多线程？

单线程。这里有一个很酷的解释者关于JavaScript如何同时是异步的和单线程的。

6.在JavaScript中，什么是箭头函数？

箭头函数是ES6语法的一部分。它没有自己的“this”关键字。相反，箭头函数将使用创建时在then函数之外的“this”关键字。

你可以参加我设计的“这个”测试(双关语)来更熟悉this关键词。

7.在SQL中，主键和外键是什么？

主键:表中每行的唯一ID/地址。
外键:一个表中的一组列，引用另一个表的主键或唯一键。
