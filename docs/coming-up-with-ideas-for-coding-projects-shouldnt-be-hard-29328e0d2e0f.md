# 提出编码项目的想法应该不难

> 原文：<https://levelup.gitconnected.com/coming-up-with-ideas-for-coding-projects-shouldnt-be-hard-29328e0d2e0f>

## 从你的日常生活中汲取灵感

![](img/c8e3fc17e3fb1d3bdbaf04e65ebf676f.png)

扎克·沃尔夫在 [Unsplash](https://unsplash.com/photos/q5tQtPZ7j2k) 上的照片

作为一名软件开发人员，从事兼职项目是自我发展道路上至关重要的一部分，它允许您探索新的技术和做事方法，这些是您在朝九晚五的工作中可能无法做到的，更不用说花一些时间练习巩固您的编程基础知识了。

然而，人们常常很难想出点子，也不知道他们的副业实际上应该做什么，所以我将尝试给你一些关于如何立即想出项目点子的指点和提示！

1.  **尝试用你的项目解决问题**

不要只是开始你在 YouTube 上看到的随机项目，或者在阅读一篇文章时，试着把你的项目想象成一个问题的解决方案。

由于软件开发实际上是解决问题，你会发现想法会更自然地出现在你面前，你不会浪费时间只是开始项目，几天后又放弃它们，因为它们根本不会引起你的兴趣。

**2。反思你的日常生活**

想出点子的一个好方法是反思你的个人经历。当你在手机上工作或使用应用程序时，试着想想有哪些地方可以改进。

它甚至不需要改进，也许你想使用某个应用程序的功能，但它是付费的，所以你可以编写一个小应用程序来做同样的事情，而不是付费。

你可以和你的朋友做同样的事情。询问他们是否有任何想法，或者他们是否认为某个应用程序会对他们有益。例如，我的女朋友想以每年 70 英镑的价格购买一个健身应用程序，这让我想到“我可以自己编写代码”，这给了我一个兼职项目的好主意。

**3。利用编码社区**

那里有很棒的开源项目，你可以为之做出贡献，你甚至可以受到启发自己开始一个项目。

另外，GitHub 是一个寻找[趋势库](https://github.com/trending)的好地方，它能给你很多灵感。

**4。探索趋势技术**

作为一名软件开发人员，你可能每天都在使用相同技术的单一行业中工作。

但这并不妨碍你探索除了你的专业领域之外让你感兴趣的趋势技术和行业。

例如，如果你喜欢游戏，虚拟现实等，你可以探索像 Unity 这样的东西，并建立一个简单的游戏。

或者，如果你从未开发过移动应用程序，但你对此也很感兴趣，你可以去探索一下像 Flutter 这样已经流行了一段时间的东西。

## 结论

至少对我来说，最简单的方法是专注于解决日常生活中让你兴奋的问题，并探索可用于开发你喜欢的软件的技术，这取决于你的个人偏好。
