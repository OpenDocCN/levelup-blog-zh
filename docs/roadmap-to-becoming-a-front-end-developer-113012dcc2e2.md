学习成为一名前端开发人员需要一段时间。

在本文中，我们将探讨成为前端开发人员的途径。我们会专注于我们必须学习的东西。

# 饭桶

Git 已经成为中最流行的版本控制系统，因为它能够拥有本地和远程存储库。它还可以让人们通过分叉来制作远程存储库的副本。

拉请求也是让人们在合并代码之前检查代码的好方法。

如果我们学会了如何拉和推代码，发出拉请求，合并代码，那么我们就有了一个好的开始。

要了解更多关于 Git 的信息，我们可以去[Git 完整指南](https://click.linksynergy.com/deeplink?id=JVFxdTr9V80&mid=39197&murl=https%3A%2F%2Fwww.udemy.com%2Fgit-complete%2F)了解更多。

# SSH 和命令行基础知识

SSH 代表安全外壳。这意味着我们必须登录到服务器，并知道如何通过运行命令来进行操作。

它们只是一些像文件操作命令，改变`.bashrc`等命令。这是我们必须知道的。

常见的操作包括操作文件、创建链接、导航文件系统、向命令行添加选项以及编辑文本文件。

# **HTTP(S)**

大多数应用程序必须与外部通信。HTTP 是实现这一点的方法，还有它的安全对手 HTTPS。

我们应该知道如何发出 HTTP 请求、设置请求头、解析响应并使用它们。

# **数据结构和算法**

前端开发就是解决问题。数据结构和算法让我们用计算机程序解决问题。

程序由用算法操纵数据结构的代码组成。所以我们必须学习这两者来创建我们自己的程序。

我们必须了解各种算法的速度，这样我们才能创建速度足够快的程序供人们使用。

常见的算法包括排序、搜索、贪婪算法和优化。

[Geeks for Geeks](http://geeksforgeeks.org/) 是学习算法的绝佳资源。

# **字符编码**

我们必须学习字符编码，这样我们才能在屏幕上正确显示数据。

最常见的是 Unicode，所以我们必须了解字符集中有什么。另一种常见的编码是 ASCII。

# 远程 Git 主机

我们必须学习如何使用远程 Git 主机，如 Github、Bitbucket 或 GitLab，这样我们就可以将代码推送到远程存储库。

此外，我们必须知道如何合并代码和分叉库。我们可能还需要管理权限，创建和远程存储库。

# 网页设计

我们必须知道一些关于设计的知识，即使我们自己可能不做设计。

但是，我们还是要站在用户的角度考虑用户体验，把事情做好。

# HTML 和 CSS

HTML 是一种标记语言，用于在网页中放置东西，CSS 则用于对这些东西进行样式化。

这些是网页的基本构件，所以我们应该充分了解它们，这样我们才能创建自己的静态页面。

用 flexbox 和 grid 做布局也很重要。此外，我们必须学习语义 HTML 标签，如导航，部分，旁白等。

一定要避免 HTML 过时的特性，比如格式化标签和样式属性。

# Java Script 语言

JavaScript 是前端开发的主要编程语言。我们可以使用它通过操作页面上的元素并动态地设计它们的样式来使我们的页面动态化。

此外，我们必须使用它从服务器发出 HTTP 请求，并从它那里获得响应。

抽象、封装、多态和继承等基本概念都适用于 JavaScript。

此外，我们必须学习 JavaScript 的面向对象和函数式编程概念。

其他重要的概念包括运算符、字符串、循环和数组。

[Mozilla 开发者网络](https://developer.mozilla.org/en-US/)有我们需要学习的所有 JavaScript 知识。一定要学习 JavaScript 的标准库，这样在我们开发的时候可以节省我们的时间。

# 以打字打的文件

TypeScript 是 JavaScript 语言的扩展，它为我们提供了 JavaScript 所没有的类型检查和注释。

它检查数据的类型，这样我们就可以使用 JavaScript 而不用担心在运行时出现数据类型错误，因为它在我们的应用程序构建之前就进行了检查。

# 框架和反应

Angular 和 Vue 是比较流行的库，React 是比较流行的视图库，供我们学习。

一旦我们掌握了如何用 JavaScript 操作元素，我们就可以继续学习这些框架和库，让我们的生活变得更加轻松。

在所有这些中，Vue 是对初学者最友好的，所以我们可能想从它们开始。

# 结论

要成为前端开发人员，我们需要学习很多东西。因此，如果我们想成为前端开发人员，没有比现在更好的开始时间了。
