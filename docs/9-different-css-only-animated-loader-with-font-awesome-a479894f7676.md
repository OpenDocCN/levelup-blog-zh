# 介绍

动画很神奇。一个合适的动画可以帮助你更好地解释你的内容，让你的网站/应用程序更有趣。像许多开发者一样，设计图标不是我的拿手好戏。幸运的是，网上有很多免费资源，提供易于使用、可定制的矢量图标。

今天，我将使用 font-awesome(全是自由层图标)来探索 9 种不同的动画加载指示器(简称 loader！)
