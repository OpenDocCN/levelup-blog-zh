# 如何应对工作场所的混蛋

## 学习管理冲突和提高生产力的技巧

![](img/35f6849ed998e758168ab82455fa6371.png)

在软件工程职业生涯中，我们经常与态度恶劣的人一起工作。这些人有些真的很聪明，很有才华，有些只是无能的混蛋。不管是哪种情况，和态度恶劣的人一起工作都不是件愉快的事。

和怪人一起工作会给你的日常工作带来持续冲突的风险。这种人在与人交往时会产生不必要的摩擦。当这种行为持续很长时间时，它有可能影响你和团队的绩效。

然而，几乎没有一个软件开发团队不需要和一群人一起工作。你必须与你的开发伙伴、产品负责人、架构师、联锁团队成员等互动。当你在一个小组中工作时，很有可能会遇到至少几个你不同意其态度的人。

在这种情况下，正确的做法是学习如何在专业上与态度恶劣的人共事。这样，你可以避免不必要的冲突，这将有助于你成长为一个体贴的专业人士。所以，让我们深入探讨五种方法，你可以用它们在日常工作中成功地对付混蛋。

# 1.第一印象会持续很长时间，但它可能不是正确的

我们都会被工作场所刚认识的人的第一印象所迷惑。我们遇到一位新同事，可能在第一次互动时就不喜欢他们。我们会立刻对他们形成负面的看法。接下来，我们通过最初的感知观察他们所有的活动。

很多时候，我们甚至在与人真正互动之前就对他们形成了看法。我们这样做是基于我们从同事那里听到的关于这个人的信息。一旦认知形成，我们不会通过事实来验证它，也不会和其他人交叉验证。

然而，第一印象可能并不总是正确的。我们应该总是试图验证它，以确保它是准确的。为此，观察他们是否总是态度恶劣。弄清楚他们是对所有人无礼还是只对你无礼。你的观察将帮助你再次确认你是否有正确的感知。

我知道对他人形成快速看法的缺点。因此，即使我对一个人有正面的第一印象，我也会推迟形成观点。通过观察，我发现我的许多第一印象并不准确。后来，我感谢自己没有根据第一印象建立印象。

# 2.直接对话可能会奏效

假设你随后与一个人的互动仍然是负面的。通过观察，你确定这个人是个混蛋。他们对你的态度总是很糟糕。但是你看到他们对你的其他同事很好。

在这种情况下，试着理解是什么让他们为难你。很有可能他们已经对你形成了负面的第一印象。你可能做了一些让他们误解的事情。但他们可能还不够成熟，无法直接与你消除分歧。

你可以向这个人伸出手，看看你是否能帮助消除任何误解。让他们知道你发现他们对你的行为有点不欢迎，这关系到你。询问他们是否愿意就此进行公开对话。如果伸出援助之手对双方来说都意味着一个更好的工作环境，那么这并没有错。

根据我的经验，直接讨论我的顾虑可以消除很多误解。有时候，他们听到了一些关于我的负面消息，并形成了自己的看法。快速同步后，我们可以解决分歧，继续我们的工作，不再有任何敌意。

# 3.即使面对逆境也要专业

有时候人们真的很混蛋。改变他们的态度并不总是可能的。在这种情况下，你斡旋和平的所有努力都将是徒劳的。这种混蛋把你积极的态度视为弱点，不会回应你的和平提议。

然而，与每个人保持职业关系对你最有利。在和混蛋打交道时，你可能犯的最大错误就是表现得和他们一样。在这样的对话中，这会让其他观察你的人觉得你是个混蛋。

因此，不要让他们的态度毁了你的一天或改变你的行为。我们都有义务在工作场所表现得专业。同时，你应该保留那些最困扰你的事件的证明文件。如果事情失控，当你想把事情升级的时候，这些证明会帮到你。

# 4.向你的经理寻求帮助

如果这些混蛋的不良行为继续下去，肯定会影响你的表现。没有人喜欢在充满敌意的工作环境中工作。在这种情况下，你应该向你的经理寻求帮助。

公司训练他们的经理去应付难相处的人。他们的工作是解决团队成员之间的人际问题。管理者可以与双方交谈，了解问题并帮助解决问题。他们可以做得比你能做的更好来阻止这种威胁。

> 当你这么做的时候，要对结果保持开放的心态。不要指望经理会支持你。

当你这么做的时候，要对结果保持开放的心态。不要指望经理会支持你。只要陈述事实和你的担忧，这样他们就能做出适当的决定。此外，即使反馈与你期望的结果不同，也要准备好接受反馈。

经理可能会给你一些建议，告诉你如何更好地处理这种情况。有时看起来他们做得不够。但是总有一种可能性，你可能也不得不改变你的态度。如果你不能接受反馈，你也不能指望对方会听你的。

# 5.总有最后一个选择

好的，现在你做了以上所有的事情，仍然没有得到一个合适的解决方案。糟糕的态度会继续影响你和团队的表现。你做了你的经理要求你做的事，但这无助于缓解问题。

如果它影响了你，并且以上所有方法都没有产生效果，那么下一步，联系人力资源部。你的工作效率不应该因为其他人的行为而受到影响。公司有责任确保每个人都有一个有利于高效工作的环境。

如果你的经理同意你的决定，但自己无法处理这件事，他们会愿意安排人力资源讨论。这将是一个理想的场景。但是如果你的经理不同意，不要阻止自己去找人力资源部。

我知道绕过你的经理和人力资源部联系可能很复杂，但这是你最后的努力。在尝试之前不要离开公司。不能保证你在新工作中不会遇到另一个混蛋。因此，你至少应该知道，在最终决定找到一份更好的工作之前，你已经尽力了。

# 最后的想法

现在你知道如何解决这个问题，如何在你的工作场所对付混蛋了。如果你注意到了，我还没有定义谁是混蛋。这要看在职场上什么行为是你不能接受的。基于此，你可以有自己的定义。

此外，您是否对这些步骤中的许多步骤要求自省感到惊讶？在某些情况下，我们自己也可能是个混蛋。如果我们不愿意反省自己的行为，那么我们就不能期望别人也这样做。

争取拥有一个能让你发挥最佳水平的工作场所总是明智的。态度恶劣的人会直接影响团队的表现。我希望你能在自己的情况下应用这些步骤，成功地创造一个友好的工作场所，让每个人都能茁壮成长。
