Docker CLI不是非常用户友好，当我们在一个多容器环境中工作时，你必须明确许多选项在日常使用和调试中作为默认选项是有意义的。这篇文章的目的是定义一些函数、别名和JSON模板，使CLI对我们来说更容易。
