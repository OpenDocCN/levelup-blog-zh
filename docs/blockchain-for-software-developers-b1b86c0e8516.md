这项技术的使用是如何改变软件行业的，这是非常令人着迷的。很多系统都在采用区块链。嗯，许多人使用它只是因为它是一个时髦词。与此同时，也有其他人将它用于真正的用例。

# 什么是区块链？

简单来说，*区块链是一个去中心化的账本，维护着记录。这些记录一旦被记录就不能更改*。

这项技术中的去中心化是什么？您可以假设区块链是一个数据库，它不是集中维护的，而是分布在计算机网络的各个节点上。

区块链的一个非常著名的用例是维护加密货币的安全和分散的交易记录。

# 为什么是区块链？

这是这项技术的一些好处—

*   区块链是一个共享数据库，通过加密技术将数据存储在链接在一起的区块中。
*   它按时间顺序存储数据。一个数据块被链接到前一个数据块以形成一系列数据块。
*   虽然这一切都是从存储分类账或交易数据开始的，但区块链有能力存储任何类型的数据。
*   区块链确保没有任何单一实体、组织或人群控制该系统。通过以分散和共享的方式存储信息，它确保权力不属于单个实体或组织。
*   分散的区块链是不可变的，这意味着输入的数据是不可逆的。对于比特币来说，这意味着交易被永久记录，任何人都可以查看。

除了成为一个时髦词，区块链最大的吸引力是它通过创造不可更改的记录来保证系统中的信任。

# 好吧，它是怎么工作的？

我们已经听够了这项技术，但是它是如何工作的呢？让我们找出答案。

为此，我们需要了解事务是如何创建的。它发生在这些简单的步骤中—

*   新交易进入系统
*   网络中的每个对等体都被告知新的事务。
*   交易的有效性由计算机网络通过求解复杂的方程来确认。
*   这些事务中的一些被聚集到一个块中。
*   这些块被链接在一起形成事务的历史。这些块保存在网络的一个以上的节点中。这使得无法编辑/删除交易。
*   一旦块被写入多个节点，事务就完成了。

# 为什么要分权？

在这里，分散化意味着信息的分散化，从而意味着权威和权力的分散化。

想象一个场景，一家大公司拥有一个存储数百万用户信息的大数据中心。将所有数据放在一个屋檐下，由一个实体控制，会有数据被擦除或损坏的危险。如果发生自然灾害、停电或互联网故障，或者有人删除或更改数据，该怎么办？

另一方面，区块链确保数据库不是保存在一个地方。数据分布在不同地理位置的网络节点中。这造成了冗余，并使记录不可变。即使有人改变了一个节点上的事务，也不会改变其他节点上的信息。

# 区块链——下一件大事

该行业每天都在寻找区块链的新使用案例。如今，区块链有数万种加密货币在运行。但是区块链由于其可靠和不可变的存储，可以在许多应用中使用。

一些流行的用例是—

*   **银行业务** —使用区块链记录银行交易可以显著减少交易的处理时间，不受银行营业时间的影响。各个银行之间的货币交换也可以更加安全和快捷。
*   **财产记录**——如果你曾经和财产登记办公室打过交道，你就会知道整个过程是多么的低效。获得一个简单的证书，如产权负担证书，需要很多时间。扫描文档，然后从文档中检索信息是非常耗时的。区块链是解决这一问题的绝佳方案。记录是不可变的，没有机会更改记录，并且可以很容易地检索事务的顺序。
*   **医疗保健**——如今，患者的病历至关重要。随着越来越多的系统相互共享数据，正确的医疗记录的价值已经飙升。在区块链上存储医疗记录可以确保记录不会被篡改，并且记录可以被加密和存储，只有需要的个人才能访问。
*   **智能合约** —什么是[智能合约](https://en.wikipedia.org/wiki/Smart_contract)？它是一种计算机代码，用于促进、验证或协商区块链上的合同。像任何其他合同一样，智能合同在双方同意的一系列条件下运作。这些可能是任何类型的合同，如咨询、服务、租赁和销售。
*   **供应链管理** —区块链可以用来记录购买的原材料、购买的服务以及销售给供应链中下一方的来源。IBM 已经为这个用例构建了一个名为 IBM Food Trust 的解决方案。
*   投票——你一定听说过失败的政党在选举中叫嚣不公平。使用区块链技术记录选票可以完全消除舞弊，并可以在很大程度上提高投票率。区块链将使这一过程完全透明，减少对选举管理人员的需求。它可以提供即时结果，而无需计算。
*   **货币** —我们都知道这个用例。区块链允许加密货币在不需要中央机构的情况下运行。它消除了许多处理和交易费用。经济不稳定的人可以使用更稳定的加密货币。

# 有什么适合软件专业人士的？

区块链将为软件开发者开启无数努力。许多系统将转移到区块链。有赌博应用程序正在使用区块链。

最容易被遗忘的用例可以在银行和金融领域找到。除此之外，还有很多我上面提到的其他领域最有可能转移到区块链。

这是一场革命，引领潮流的软件开发人员将在几年内取得巨大成功。
