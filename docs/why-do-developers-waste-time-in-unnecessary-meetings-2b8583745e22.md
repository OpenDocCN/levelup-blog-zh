## 如何在会议中不浪费每个人的时间

> 如果你必须用一个词来说明为什么人类没有实现，也永远不会实现其全部潜力，这个词就是“会议”。
> 
> —戴夫·巴里

我们在会议上花了无数的时间讨论大量的话题，一些相关的，还有许多不相关的。

在很多会议上，我们会讨论为什么工作没有完成。讽刺的是，我们没有意识到，这些无休止的会议是工作无法完成的原因。

这些会议多年来一直是工作场所的干扰。现在，这些会议已经变成了在线电话，让你在 ***在家*** 的工作环境中忙碌。

为什么我们会在会议上浪费大量时间——以下是一些原因:

# 将会议视为一天的开始/结束

曾经有一段时间，我每天都是从上午 11 点的站立会议开始。我曾经想知道为什么我不能完成工作。

我们许多人有意识或无意识地把会议作为一天或一周的开始或结束。是的，有几次会议为一天/一周定下了基调。但是你不应该认为这些会议是你一天的开始/结束。

我通过在清晨站立前几个小时开始来固定这种观念。我很惊讶地看到我能在一天或一周内完成的事情。

# 没有议程的会议

> 你开会是为了做决定，而不是决定这个问题
> 
> —比尔·盖茨

我们经常参加没有议程的会议。没有明确议程的会议一定会浪费你和他人的时间和精力。

你总是希望会议得出一个结论。要得出结论，你需要有正确的问题。议程提出了这些问题，并为会议提供了方向。

有了会议议程，会议就有了可衡量的结果。你不会想参加一个没有结果的会议。

# 信息交流会议

> 会议是组织不良的表现。会议越少越好。
> 
> —彼得·德鲁克

人们如此沉迷于会议，以至于想不出任何其他的信息交流渠道。我发现自己好几次遇到这样的会议。

今天，有许多沟通方式——电子邮件、聊天、任务管理工具、bug 追踪器等等。任何可以通过这些渠道发送的信息或数据，如果接收者能够理解，就不需要见面。

我经常拒绝参加没有议程的会议。这些会议从一个话题转移到另一个话题，浪费了每个人的时间。

# 频繁的会议以同样的讨论结束，因为在会议之间没有取得多少成果

在我的职业生涯中，我见过这样的每日例会，每个人都在会上更新了他们昨天所做的事情。我常常意识到，在两次会议之间，我们许多人并没有取得多大成就。

大多数时候，我们所做的工作是复杂的，需要几天时间才能看到结果。如果你必须更新你所做的事情，这些每日状态会议只是浪费时间。

虽然我选择不谈论我所使用的策略的细节，但其他人只是说，因为他们认为他们应该这样做。他们传递对别人没用的信息。

我建议增加这种会议的间隔时间。

# 我们不认为我们需要时间来转换上下文

大多数经理和领导都不知道在任务之间切换会浪费多少时间。你正在着手一项复杂的任务，突然你有了灵感。打完电话后，重新把注意力集中到任务上需要大量的时间。

我们需要重视手头的任务，而不是随机的会议。如果你正在处理重要的事情，我建议你重新安排会议时间。

# 害怕错过(FOMO)

我认为这是所有不是真正需要的会议的根本原因。这也是人们参加根本不需要他们出席的会议的原因。

后疫情时代，FOMO 暴涨，有一半时间坐在远离他人的地方的人都在打电话。对于经理来说尤其如此。他们从团队中得不到信息，FOMO 开始介入，他们创建会议并强迫其他人参加。

# 包扎

人们开会可能还有很多其他原因。我不是说所有的会议都是无用的。我的意思是我们应该避免所有不必要的。

在这个问题上我一直保持警惕。我经常对会议说不，并要求其他人通过电子邮件/聊天进行互动。可以通过电子邮件/聊天解决的话题不应该带到讨论室。

我和我的团队严格遵守这样一个时间，除非有重大事情发生，否则我们不会打扰对方。
