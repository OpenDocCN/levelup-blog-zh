你的非功能性测试暂时是你的:非功能性测试的问题。

近年来，NFT 的炒作获得了难以置信的支持，吸引了数百万人的兴趣。持有独特数字令牌的想法吸引了加密社区，展示了所有权的新概念，同时为数字艺术家提供了一个发现和探索技术的平台。不管这个想法背后的意图有多好，就目前的情况来看，这项技术是在一个有缺陷的系统上设计的。让我们讨论一下 NFTs 有什么问题，以及为什么我们应该寻找其有缺陷的逻辑的替代方案。

# 什么是 NFT，为什么它如此受欢迎？

NFT 的意思是不可替代令牌(NFTs)，它是一种代表存储在区块链上的独特资产的令牌。这些通常是数字艺术品或 NFT 收藏品，因此它们可以卖出高价，其中一件藏品的售价超过 6900 万美元。然而，与实物资产不同的是，你不能将一种资产换成另一种资产。销售必须在另一端秘密进行。该技术流行背后的主要原因之一是它们提供了无可辩驳的数字资产所有权的确凿证据，这激发了社区对所有权到底是什么的兴趣。

# NFT 所有权

这个想法的独特之处在于 NFT 所有权，因为它为人们提供了拥有数字资产的能力。不是将它们储存在你的电脑上，而是实际上被标记为艺术品的唯一所有者。不可替换的令牌由几个部分组成，图像、元数据和所有权记录。这证明你是该资产的实际所有者，并授予你所有权和优势——前提是该收藏旨在为所有者提供额外津贴——允许你享受你的数字艺术。

它们也是市场上受欢迎的交易资产，这就是为什么它们有一个额外的功能。数字资产的原始创造者可以设置它，以便他们总是从每笔销售中获得特定的百分比。这为艺术家从第三方出售他们的原创作品中获利创造了基础。

这些都是很棒的东西，对吧？嗯，有一个警告。虽然区块链是永久的，*你的非功能性测试不是。*

# 你的 NFTs 不能持续，让我们来谈谈。

让我们来看看 NFT 目前是如何设计的。

当您购买一个时，它通常有 3 个组件:

1.  元数据:这是关于其属性和特征的信息
2.  所有权记录:这将确认你是资产的真正所有者
3.  图像文件:这是你看到的艺术作品。你购买 NFT 是因为你喜欢它的外观，它基本上是资产的一部分。

尽管它们越来越受欢迎，也有潜在的用途，但故事总有另一面。随着越来越多的支持，人们越来越担心它们的真正寿命

# 考虑到这一点，NFTs 的问题在哪里？

常见的 NFT 批评指出，它们的创建是为了分散艺术，并为艺术家和艺术爱好者提供一个永久的解决方案。虽然区块链是一个永久的空间，但 NFTs 存储不会发生在区块链，至少不会发生在你的全部数字资产上。

您的珍贵作品的元数据和所有权存储在区块链上，而实际的 NFT 本身存储在集中式文件托管服务上。这意味着，从技术上讲，你的图像有可能被更改，甚至被删除，留给你的只是数据&没有图像。

# 没有形象有什么大不了的？

许多爱好者和持有者喜欢炫耀他们对稀有 NFT 艺术品的所有权，因为 Twitter 和许多其他社交网络允许用户将他们经过认证的 NFT 设置为个人资料头像。这也不仅仅是一件审美的事情，因为这些可能是极其宝贵的资产！特别是当一些已经卖了超过 50 万美元，所以炫耀他们听起来吸引他们的持有者，几乎像一个奖杯。这就是为什么失去形象意味着失去 NFT。

*让我们想象一下纸牌屋倒塌的场景*

假设你花了 10，000 美元买了一件稀有的 NFT。你买它是因为你热爱它的审美，想成为这件艺术品的唯一所有者。一旦你买了它，你想在你的社交媒体上炫耀它，所以你把你的钱包链接到 Twitter，并作为你的头像加载它。你也在 Reddit 上做了同样的事情。谁来评判？你为你最近购买的昂贵而稀有的东西感到骄傲！

现在让我们假设图像文件存储在 Amazon Web Services 上，出于某种原因，该服务不得不关闭，为了便于讨论，让我们假设维护存储服务器的成本超过了它们的收入。

一旦服务关闭，你的 NFT 就没有图像了。对于 10，000 美元的购买，您需要展示的唯一内容是元数据+所有权记录，该记录带有一个指向“404 page not found”的过期链接。你会有什么感觉？这就是为什么 NFT 艺术需要紧急干预才能被认为是可持续的。

那么我们能做些什么来修复 NFTs 呢？

没有办法，他们需要完全去中心化。整个 NFT(元数据、所有权记录和影像)必须存储在区块链上。这是确保它们长期存在的唯一方法。

# NFTs 的未来:NFT 全面分权能实现吗？

是的，它可以，甚至可能在不久的将来。已经有解决这个问题的解决方案正在开发中，其中之一是 Everscale 的 Drivechains。这些是为 Everscale 网络构建的分散存储设备。就其核心而言，传动链是一个为存储大量数据而优化的工作链。这可用于在区块链存储所有 NFT 数据，避免使用集中式文件托管服务。通过这种解决方案，不仅元数据和所有权记录将存储在链上，图像文件也是如此，从而实现了 NFTs(完全分散的数字艺术)的最初目标，同时为不可替换的令牌提供了所需的基础设施。

这将解决 NFTs 目前面临的多个问题。其中之一是所有权。当您将图像上传到一些服务时，您在某种程度上向他们提供了您的图像的权利，因为将任何材料上传到他们的服务会授予他们非独家、免版税、永久、不可撤销的全球许可(拥有分许可和转让权利)，以使用、展示以及在任何现在或未来创建您的有价值的艺术品的衍生作品。这违背了加密社区提出的所有 NFT 所有权概念。你实际上可能并不拥有代表你的 MFT 的图像，这是一个主要问题。

Everscale 的驱动链解决的第二个问题是寿命。你将不再依赖文件托管服务来保存你的作品。区块链是永久性的，而集中式服务可能不是。保存在中央服务器上的文件可以被修改。如果有人可以访问服务的服务器，图像可以被替换、编辑甚至删除，而记录在区块链上的数据是永久的，不能以任何方式更改。这是因为区块链被设计为不可变的，这是通过利用混合了块相互关系的加密哈希来实现的，使输入区块链的数据永久化。NFTs 为艺术社区和艺术爱好者提供了一种半分散的基础设施，这种设施已被证明在使艺术相关方面是成功的，特别是对于倾向于数字的一代用户。剩下的唯一事情就是修复这个有缺陷的系统的问题，只有这样我们才能称 NFTs 为‘去中心化的艺术’。
