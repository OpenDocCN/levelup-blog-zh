信息模式数据库

我们可以使用有价值的INFORMATION_SCHEMA数据库并查询TABLES表中的任何AUTO_INCREMENT列值。

下面是一个查询示例，其中我将众所周知的实践Sakila数据库作为目标:

SELECT TABLE_NAME, AUTO_INCREMENT
FROM TABLES
WHERE TABLE_SCHEMA = 'sakila';
