## 4.字符类别

下表提到了对你处理 re 有用的基本字符类。

```
+---------------+----------------------------------------------+
| Metacharacter |                   Behavior                   |
+---------------+----------------------------------------------+
| **\d**            | Match any digit character [0-9]              |
| **\D **           | Match any non-digit character                |
| **\n**            | Match a new line                             |
| **\s**            | Match white spaces                           |
| **\S**            | Match non-white spaces                       |
| **\w**            | Match word characters (includes a-z and 0–9) |
| **\W **           | Match non-word characters                    |
|               |                                              |
+---------------+----------------------------------------------+
```

## 5.否认

## 6.^和美元

**^** 和 **$** 堪称正则表达式中最常见的锚点。 **^** 匹配一行的开始， **$** 匹配一行的结束。

此处示例匹配启用的多行标志。

![](img/8819d45da48aea0371dcf28b4e8883ad.png)

^指定了句子的开头

![](img/f466bb2fea622f2c27458a351ad57f59.png)

$指定句子的结尾

![](img/0494775cfe7b99153f44caf26cfb3937.png)

^和$都被用来表示整个句子“苹果是世界第四大个人电脑供应商”

## 8.基于运算符(*、+、？, .)

***** ， **+** ，**？**和**。**是用来处理 re 的运算符。

## 9.数值范围

用于使用**{低，高}** 指定数量范围

***{n，m}*** *—最小值为 n，最大值为 m*

![](img/f75c73e149d485c37cc0ff2b6ea4154d.png)

最小值应该有 4 个数字，最大值应该有 6 个字母

***{n，}*** *—最小值为 n，最大值未定义*

![](img/3f7fb71901e4dec671c7717ef808e38d.png)

最小值应为 4 位数，最大值未定义
