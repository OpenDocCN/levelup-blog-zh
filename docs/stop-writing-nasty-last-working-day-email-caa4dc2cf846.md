# 停止写讨厌的最后一个工作日的电子邮件

## 意见

## 不要做那个带着一封刻薄的告别信就离开的失败者

![](img/ac3e970ac99dc3d362ce9efcf6fd7c92.png)

米哈伊尔·尼洛夫摄于[佩克斯](https://www.pexels.com/photo/person-woman-relaxation-laptop-9159637/?utm_content=attributionCopyText&utm_medium=referral&utm_source=pexels)

我们大多数人都在最后一个工作日给同事写过某种电子邮件。它通常是一封告别邮件或“我会想念你”类型的信息。

最近，我在最后一个工作日收到了一封电子邮件，发件人是一位我从未共事过的同事。这是一封发给公司广大员工的长邮件。这封邮件对发件人的同事和老板充满了不愉快的情绪和严厉的言辞。

这封邮件总结了该员工在公司的整个任期。它关注的是工作环境有多不公平，以及这个人即使表现令人满意也没有得到好的评价。

发件人最后签了一张纸条，说任何人都不应该试图联系他们，因为他们已经在各种平台上阻止了他们的同事。

尽管发件人试图用讽刺的语气写邮件，但很明显他们受到了伤害。同样很明显的是，这位雇员缺乏专业精神。我想知道为什么他们在邮件收件人名单中包括了从未共事过的人。

如果你已经在你的职业中呆了几年，你一定也收到过一些关于末日的信息，从尴尬和不舒服到彻底的沮丧。

# 为什么员工会在最后一个工作日写讨厌的邮件？

在公司的最后一个工作日，大多数员工往往会被各种情绪淹没——有些是积极的，有些是消极的。大多数员工离开时情绪很高，而有些人对他们的公司怀恨在心。

当他们辞职时，高度情绪化的人倾向于用两种方式写最后一封工作日的信——生气或悲伤。

缺乏情商的愤怒的人无法再抑制他们对公司施加太多压力的愤怒。

需要注意的是，员工通常不会因为自己的挫折而责怪自己。他们觉得自己没做错什么，组织要对他们的状况负责。

这种无助感导致怨恨，并最终演变成发给他们团队和(在我的例子中)他们甚至不认识的人的讨厌的电子邮件。

# 为什么员工应该停止写讨厌的最后一个工作日邮件？

![](img/de491fead2cc5ff0f717191c0df1d423.png)

图片来自 [Pixabay](https://pixabay.com/?utm_source=link-attribution&utm_medium=referral&utm_campaign=image&utm_content=1206614)

不管内容背后的真相如何，在最后一个工作日写一封令人讨厌的电子邮件在很多方面都是一个坏主意:

*   你失去了一个解释你自己和你的需求的机会——不管它们多么不合理。
*   通过写这封邮件，你不会得到你想要的情感支持。邮件可能会在你写的时候提升你的自我，但是这种满足感是短暂的。
*   你让自己看起来又刻薄又不专业。你也证实了你缺乏情商。
*   你失去了一个展示自己韧性的机会。你必须学会平衡自己的情绪，才能在职业上取得成功。韧性帮助你应对生活抛给你的挑战。

# 除了写一封讨厌的最后一个工作日邮件，你应该做什么？

最好的离开方式是友好相处——表现出一些风度，而不是灾难。

如果你在公司的任期没有达到你的期望，你的沮丧是可以理解的。你没有像预期的那样成长。你觉得你应该得到更好的个人和职业。

然而，明智的做法是与你的经理或人力资源经理谈谈，而不是写一封普通的最后一个工作日的电子邮件。与他们分享你的不满，而不要指望他们会立即采取行动。讨论的重点是确保你表达了自己的观点，这可以减轻你的情绪负担。

讨论还会让双方分享故事，并帮助你友好地离开。

# 经理处理讨厌的最后一个工作日邮件的最好方法是什么？

如果经理收到这种性质的电子邮件，他们应该采取行动。这将有助于防止雇员和组织之间产生任何嫌隙。他们应该认识到，这是一个员工高度情绪化的时期，他们可能有合理的理由不高兴。

失望可能是由于来自绩效管理、处理办公室政治或在团队合作差的环境中工作的压力。

同时，经理们必须明白，他们对员工在最后一个工作日的精神状态不负任何责任，应该让过去成为过去。他们不应将此视为针对个人——将其视为对他们或公司的攻击。

相反，经理可以专注于如何处理这种情况。他们应该弄清楚是否需要与(前任)员工进行转换。他们还应该回想一下，他们是否可以做些什么来避免这种情况。这些行动可以帮助现有员工在未来不会有类似的经历。

# 最后的想法

有些人在结束与雇主的工作时感到沮丧或愤怒是可以理解的，如果他们离开时没有好的条件。但这并不意味着它应该反映在你的最终信息中。

在写最后一封工作邮件的时候，想想你想留下什么作为你的遗产。

你可以在结束前说些好听的话来创造积极的回忆。或者，如果你觉得没什么好写的，写一封简单的告别邮件，作为你在公司最后一天的信息。

让我们停止写那些令人不快的电子邮件，开始用善意、感激和欣赏来关心我们的同事。不要在最后一个工作日写一封讨厌的电子邮件，这是你的优势。
