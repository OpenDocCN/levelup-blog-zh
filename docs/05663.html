什么是网页开发者

恭喜你。你决定留在这里，这意味着你迈出了成为一名网站开发者的第一步，但是作为一名网站开发者到底意味着什么呢？

我不想在这里说得太明显，但是web开发人员就是利用编程语言来构建网页和应用程序的人。有三种类型的web开发者，前端开发者、后端开发者和全栈开发者。

前端开发人员

前端开发人员负责构建您在网页或应用程序上看到并与之交互的内容。专家将设计、图像或想法转换成代码，让网络浏览器显示网站。我们不应该低估前端开发人员所做的工作，事实上，随着web应用程序变得越来越复杂，前端开发人员的工作已经从简单地实现设计转变为实际构建在我们的浏览器中运行的完整的工作应用程序。对我来说，这是一个关键的角色，在这里你可以将技术、开发和设计结合起来，构建令人惊叹的体验，这也是我对入门者的建议。从前端开始比从后端开始更容易。

后端开发人员

这个角色负责设计和构建支持我们前端应用程序的系统。例如，当你点击“保存”按钮时，后端开发人员将编写负责接收你的请求的算法和逻辑的代码，处理它，将其存储在数据库中(如果需要)并发回结果，以便前端可以让你知道一切顺利。

后端是让应用程序在多种设备上工作的基础，也是让你把信息存储在“可能”中成为可能的基础。

全栈开发人员

这是一个旧角色的新术语，全栈开发人员是一个既能完成前端任务又能完成后端任务的人。

1.学习HTML和CSS

为了成为一名前端开发人员，你需要知道的第一件事是关于web是如何工作的。你有没有想过你的浏览器是如何知道如何在屏幕上显示的？如何摆放物品，使用哪些颜色？

超文本标记语言

HTML(超文本标记语言)是所有网络浏览器都知道的标准标记语言，它被用来描述你的网站内容。与其说是它的设计，不如说是它的结构和内容本身。HTML使用特定的标签和节点语法，看起来像这样:

<html>
    <head>
        <title>My first web site</title>
    </head>
    <body>
        Hello World!
    </body>
</html>

也许你还不知道那段代码是做什么的，但是我很确定你至少可以识别一些点，比如有一个标题，还有一些内容比如“Hello World！”那里。HTML的制作是为了让它既能被计算机阅读，也能被人类以一种近乎自然的方式阅读。

这是你想开始的地方，那里有很多课程，选择一门，在进入CSS之前学习概念和它是如何工作的。

这里有一些免费资源供您参考:

  • Learn-HTML

半铸钢ˌ钢性铸铁(Cast Semi-Steel)

现在你知道了如何在用户的屏幕上放置内容，我们需要让它看起来漂亮，这就是CSS可以帮助你的地方。CSS是一种代码语法，用于描述HTML元素在屏幕上的显示方式。它允许开发者将内容放在屏幕上的不同位置，添加颜色、背景、边框、字体、动画，任何你需要的东西来匹配你的设计，很可能CSS支持它。

学习CSS是非常重要的，现在没有人会使用一个没有设计的页面，这是一个杀手锏。有些人用CSS做了非常惊人的事情，从惊人的设计到艺术，动画，真是难以置信。

CSS看起来有点像:

.title {
  color: green;
}

有点可怕，但也许你知道它可能会做什么。

2.实践

在进入下一步之前，确保你大量练习。你现在可以建立静态网站，制作许多静态网站，从基础开始，当你习惯了之后，再进入更高级的功能。为你的狗或猫建立一个个人网站，添加颜色、背景、图片、标题、菜单等。

一旦你对HTML和CSS相当熟悉，这对于避免以后的挫折是很重要的，那就继续学习JavaScript。

3.学习JavaScript，改变世界

到目前为止，你已经建立了惊人的网页，但他们不能做得很好？你可以展示内容，设计风格，创建伟大的设计和动画，但是他们缺少交互元素，动态部分，这就是JavaScript的用途。

JavaScript是一种编程语言，事实上，它是当今可用的最流行的编程语言之一，它具有在web浏览器中运行的特殊性，因此可以用于构建web应用程序。

JavaScript允许您通过处理诸如用户点击元素、定时事件等事件来与HTML交互。它还允许您动态地更改页面的内容，这意味着它可以访问“HTML”，或者实际上可以访问页面的DOM。

如今，JavaScript从浏览器迁移到了许多其他应用程序。许多公司正在构建运行在JavaScript上的后端，甚至桌面应用程序现在也是用它构建的。

学习JavaScript将是实现你成为一名web开发人员的目标的巨大一步，这可能是最难的，所以花点时间学习它，这并不容易，但也不是不可能，而且有大量的免费内容，没有借口。

所以你要读书，练习，练习，再练习。

4.制作一个作品集网站

现在你有了技能，是时候展示它们了。使用作品集网站展示你的技能在开发者中非常流行。作品集网站基本上是你向雇主或未来客户展示的最佳项目的集合。

通常，开发人员会有2个不同的投资组合网站:

个人网站

建立你自己的网站并托管它，向世界展示你的才华。这个网站将是你的自我推销，它是开发者的终极课程，所以要让它给人留下深刻印象。这个网站的一个特色是有一个部分，你可以在那里链接或展示你的项目。

通常你不仅会显示结果，所以你的项目在互联网上运行的链接，而且还包括它的源代码。这是下一个选择非常好的地方。

开源代码库

GitHub是一个网站，它允许你免费托管你的项目的代码，并向所有人开放。GitHub是最重要的开源库，也是许多开放应用程序的家园。获得一个帐户是免费的，你可以在几分钟内开始托管你的代码。在他们的网站上了解这一切。

现在，你可以通过简单地拖放文件来使用GitHub，而不需要任何特殊的工具，然而，GitHub的特别之处在于它们运行了一种叫做GIT服务的东西，GIT是一种托管代码的技术，使团队协作成为可能，并且允许你在任何时间点查看你的代码，并从中看到所有的变化。这是一个神奇的工具，你需要学习它。

6.学个框架！

现在是时候加快步伐，进入框架世界了。框架是一组简化开发任务的函数和库。较新的框架是复杂的代码片段，有时甚至会重新发明我们编写web应用程序的方式。

其中一些框架本身就像一种编程语言，因为它们提供了非常具体的方式来完成事情。可能需要一些时间来适应它们，但是一旦你习惯了，你的工作效率就会提高。

这里重要的一点是，有很多很多的框架，你不需要学习所有的框架。选择一个并从那里开始，做一个好的选择，选择一个最流行的框架，并掌握它。也许你不喜欢它，没关系，选择另一个和开关，但不要试图在所有的开始学习。

我个人的建议是选择以下一种:

  • ReactJS (我个人的最爱)
  • 棱角分明
  • VueJS

这三者之间没有对错之分，根据项目类型的不同，有不同的偏好和便利。但他们是2020年的佼佼者。

7.面试准备！

好样的。你现在是一名网站开发人员，尽管你仍然需要找一份工作。与其他职业相比，作为一名网页开发人员进行面试是其独特的经历。你必须向你的面试官展示你的经历，这是你在学习过程中建立的附带项目。你还必须向面试官展示你知道如何编码，你对HTML、CSS和JavaScript有一定的了解，甚至可能对某个特定的框架有所了解。

如果面试中有一部分你实际上不得不编写一小段代码，或者如果你被派了一个家庭作业，你需要在稍后的面试中完成并展示你的成果，不要感到惊讶。这些都是面试开发人员职位时的常见做法。

确保你之前练习过你的技能，你会做得很好。

有专门为面试做准备的网站，检查一下，给他们一个机会。请记住你的水平，其中一些网站会变得非常复杂非常快，根据你的体验调整你的挑战。
