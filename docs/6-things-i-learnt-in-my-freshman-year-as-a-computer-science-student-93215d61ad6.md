# 函数式编程

![](img/ead2e78282d6abc039fe42da8cbb637b.png)

克里斯里德在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

世界上一些顶尖的计算机科学学校，包括牛津和剑桥，在第一年就教授函数式编程。我的大学选择的语言是 [Haskell](https://www.haskell.org) ，因为它是纯函数式语言的一个很好的例子，我的教授这门课的教授，Philip Wadler ，实际上是早在 1990 年这种语言的主要创造者之一。

为什么要学习函数式语言？说实话，我不太可能会再使用 Haskell。它们非常强大，这就是为什么像 Java 和 Python 这样的语言总是添加更多的功能特性供我们利用。

但这不是重点。函数式编程语言与命令式编程语言是完全不同的范式，它们更多地基于数学。简而言之，你可能会说像 Java 这样的语言是被发明的，而函数式语言是被发现的。以这种方式学习编程教会你数学地、逻辑地思考，在你知道之前，你会想为什么你需要迭代。

# 计算和逻辑

![](img/347cc13debb79a1e805bbb1cc7f80f4c.png)

[Tbel 施虐者](https://unsplash.com/@tbelabuseridze?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

毫无疑问，这是我上过的最抽象的课程。我们学习了形式逻辑，可以追溯到[亚里士多德的三段论](https://en.wikipedia.org/wiki/Syllogism)。这被用来创建一种语言，在这种语言中，我们可以很容易地评估逻辑表达式，以确定它们是真还是假。我们看到了很多真值表，[布尔代数](https://en.wikipedia.org/wiki/Boolean_algebra)，[顺序演算](https://en.wikipedia.org/wiki/Sequent_calculus)和 [lambda 演算](https://en.wikipedia.org/wiki/Lambda_calculus)(与我们在 Haskell 中学习的内容完美地结合在一起，它的标志是一个 lambda)。然后，这些简单的逻辑规则被建立起来，变得越来越复杂，直到我们到达[有限状态机](https://en.wikipedia.org/wiki/Finite-state_machine)，最终，计算机如何工作。

这东西很奇怪，很新，很复杂，有时似乎毫无意义。但是这个逻辑是计算的核心。学习它不仅是学习更高级的计算理论和系统设计课程的必要前提，也是一次令人惊叹的逻辑、哲学、数学和计算机历史之旅。

# 线性代数

![](img/eacdd4c1444fff8c6faff768141b9bf1.png)

照片由 [Antoine Dautry](https://unsplash.com/@antoine1003?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

随着逻辑和离散数学，微积分，概率和统计，线性代数在计算机科学中非常普遍。

线性代数的概念在计算机科学的许多方面都至关重要:图形(游戏设计)、图像处理、密码学、机器学习、计算机视觉、最优化、图形、量子计算、计算生物学和信息检索(网络搜索)。

本课程是一个介绍，从线性方程组和矩阵代数开始，然后转移到行列式和对角化、向量几何、向量空间、线性变换，最后是正交性和基的变换。一路上我们还谈到了证明和数学归纳法。

这门课到最后变得相当具有挑战性，但最棒的是我们必须自己证明一切，而不仅仅是被教授规则。

# 面向对象编程

![](img/c0186aef1432b7e08ac63f846aa2b52a.png)

乔纳斯·雅各布森在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

Java。在经历了非常抽象的一个学期(老实说，这有点像是一场火的洗礼)后，我们都很高兴能继续学习一些更熟悉的东西。(第一学期不教这个的原因，以及我前面提到的原因，是因为学生开始攻读计算机科学学位时的技能水平非常不同。有些人从小就开始编程，有些人整个高中都在学习编程，有些人从未写过一行代码。从没人做过的事情开始，创造公平的竞争环境。)

我们从最基础的开始，一步步发展到类设计、继承和所有那些好的面向对象的东西。我们在整个学期中进行了三次大的评估，这给了我们一个很好的机会来真正伸展我们的编程腿并构建令人满意的程序。

# 结石

![](img/fbfe514d7382fdc281fb3685900847da.png)

照片由[杰斯温·托马斯](https://unsplash.com/@jeswinthomas?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

对那些认为你不懂得很多数学也能成为一名优秀程序员的人嗤之以鼻。像线性代数一样，微积分是计算机科学家工作中不可或缺的一部分(没有双关语)。

微积分用于创建图形和图像、模拟、解决问题的应用程序、创建统计解算器以及算法的设计和分析。

我们涵盖了我在高中学过的所有东西，甚至更多:函数和极限、导数、反函数、微分的应用、积分、积分技巧、积分和级数的应用。再一次，这比线性代数要熟悉得多，所以我非常喜欢它。

# 商业

![](img/6cd8afd8f82ee723caea803880201a76.png)

[Adeolu Eletu](https://unsplash.com/@adeolueletu?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

与英国大学相比，去苏格兰大学(或美国大学)的好处之一是四年制学位而不是三年制。这让你有机会选修你研究领域之外的课程，几乎是你想去的任何学校。

由于我对商业感兴趣，我选修了商业导论课程，内容涵盖商业战略和政策、市场营销、经济分析、运营和管理、金融、商业分析和组织行为学(这是一门全年课程)。

这给了我一个急需的暂缓学习技术科目的机会。我喜欢有机会写论文和展示小组项目。我还发现这个主题——尤其是经济学——非常有趣。

明年我有算法和数据结构，数据科学，计算机系统，离散数学和概率，软件工程和我选择的选修课。非常期待。

人们经常问我，在一个计算机科学学位中，我们实际上学习了什么，仅仅是编程吗？事实是，编码只是我们所学知识的沧海一粟。在我的第一年，我接触了计算逻辑，大量的数学，少量的哲学，是的，编程。
