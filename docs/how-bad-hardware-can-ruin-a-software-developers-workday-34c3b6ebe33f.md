# 糟糕的硬件会毁掉一个软件开发人员的工作日

> 原文：<https://levelup.gitconnected.com/how-bad-hardware-can-ruin-a-software-developers-workday-34c3b6ebe33f>

## 锯子必须锋利才有用！

## 我最近从我的雇主那里得到了一个新的笔记本，因为我的旧笔记本变得相当慢。不可思议的是，我的快乐程度提高了这么多。

![](img/1c3643ef249deabb04eba2cd8a34a2ad.png)

克里斯·利维拉尼在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

一个软件开发人员要花很多时间等待。等待编译器构建软件，等待测试运行人员执行所有单元测试，或者等待涉众阐明需求。其中一些任务可以通过使用更多的计算能力来加速，我最近接受了这种升级。

我的旧设备用了 3 年左右，速度越来越慢。保修期已过，没人能找出实际问题是什么。这可能是一个抑制 CPU 的热传感器的问题，但我的老板不想花钱修理那个年龄的设备。

所以我不得不面对它。**老实说，每天的工作都令人沮丧。我有一些需要定期完成的任务，突然间它们比平时多花了两倍的时间。我能做的就是等待。这相当于每天一小时，令人难以置信。有人可能会说，反正我拿到了钱，那还有什么意义呢？我不喜欢每天花 8 个小时在令人沮丧和讨厌的事情上，我喜欢富有成效，我喜欢创造事物，我喜欢帮助人们更快地完成任务。这就是我成为软件开发人员的原因。如果我做不到这一点，我就会不开心，脾气暴躁。**

当然，我坚持要买一台新设备，但是，是的…比方说，我公司的购买流程肯定可以改进。我花了将近 4 个月的时间才让老板给我订购了一台新设备，又花了 2 个月的时间我才最终收到它。

那是一段艰难的时光。在家工作，与同事没有真正的联系(亲自喝咖啡休息对我来说完全不同于 Skype 电话)，我的慢速设备让日子永远持续下去。这也可能是一种轻微的冬季抑郁症，春天的阳光也有助于对抗它。

完成新设备的设置后，**我终于喜欢上了再次工作**。一切运行快速和顺利，没有更多的冻结或不合理的等待时间。甚至我的公司也从我提高的生产率中获益，因为我现在一天可以完成比以前更多的任务。我的建议是，如果需要，就用好的硬件。

如果你的笔记本很慢，你作为一个软件开发人员就不可能工作得很快。如果你的斧头不锋利，你就无法砍倒一棵树。如果你的鞋子穿破了，你就无法整天站着。

如果你是一名雇主，并且你正在阅读这篇文章，记得**总是为你的员工提供好的工具**。这会让他们更快乐，更有效率！
