# 前提

我的大学专业与机器学习相去甚远。它主要与设计自动化系统有关，带有一些编程和算法的软接触。虽然它对我以后的职业生涯没有直接的帮助，但它为我在计算机科学的各种方面自学打下了良好的基础。

虽然我从学生时代就知道一些不错的旧算法，但我对现有的框架和现代架构了解不多，所以这方面知识的缺乏决定了我对课程的选择。现在回想起来，我认为这是一个相当幸运和有用的选择，所以如果你是这条道路的起点，请随意复制它。

# HSE 大学的自然语言处理

【https://www.coursera.org/learn/language-processing】航向 : [航向](https://www.coursera.org/learn/language-processing)

**价格** : 49 美元/月

**预计通过**的时间:4-5 周

这可能是我花钱买的最好的课程，因为它让我对自然语言处理的主要原理有了坚实的理解，比如语言建模、词性标注、任务序列等。

它也是非常方便的结构与毕业实习作为建设有用的应用程序(电报机器人)在最后。在我自己关于数据搜集的课程中，我也有同样的材料布局，因为我认为在我们的时代仅仅学习东西是不够的——一个人最终需要得到一个工作工具。

我已经在我的第一篇中帖中对这门课程做了详细的回顾，我仍然会向新手推荐这门课程。

# 如何赢得 HSE 大学的数据科学竞赛

**课程:**[https://www.coursera.org/learn/competitive-data-science?专业化=反洗钱](https://www.coursera.org/learn/competitive-data-science?specialization=aml)

价格: 49 美元/月

**预计通过时间:**5-6 周

我从未参加过 Kaggle 竞赛，也从未真正以此为目标，但这门课程仍然让我受益匪浅。它由著名的 Kaggle 大师提供，其想法是教授数据学习任务的非常规方法。是的，有时候是“骗子”的意思。在实际的机器学习中，了解何时可以用足够好的 EDA 替换复杂的模型有时很重要。本课程给出了如何发现此类案例的提示。

# Fastai 课程 V3

**课程:**【https://course19.fast.ai/】T2

**价格:**免费

**预计通过时间:**8-10 周

这将是这个列表中第一个也是唯一一个免费的课程，也是对开源社区设计如此庞大作品的一种感谢。尽管它是由 fastai 组织创建的，但它并不像 ULMFiT 那样集中在他们的产品上，而是围绕常见的 ML 实践给你一个很好的解释。

尽管在 ML 分支中有许多进步。在本课程中，您将深入了解以下概念:

*   正规化
*   学习率搜索
*   动态学习率

和其他；

它对新手也很有用，因为它提供了关于谷歌云平台环境设置的教程。如果你家里没有足够好的资源，使用基于云的解决方案是最理想的，参加这个课程值得学习这个特定的主题。

# DeepLearning.ai 的自然语言处理专业化

**专精:**[https://www.coursera.org/learn/sequence-models-in-nlp](https://www.coursera.org/learn/sequence-models-in-nlp)

**价格:** 49 美元/月

**预计通过时间:** <每个疗程 4 周

最近的一个。让你了解像变压器这样的现代建筑。就像在大多数 deeplearning.ai 课程中一样，你必须自己动手做设计网络的“脏活”。就你必须通过的练习而言，这也是相当复杂的，所以如果你想做一些有挑战性的事情，请随意加入。
