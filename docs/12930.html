1。CSST7】动画

Animate css是一个随时可用的跨浏览器动画库，可用于您的web项目。非常适合强调、主页、滑块和引导注意力的提示。

2。 魔法效果

魔术CSS是一组简单的动画，包括在您的web或应用程序项目中。

3。Animista

Animista是一个CSS动画库，你可以在这里玩现成的CSS动画，并且只下载那些你会用到的。

4。 动画库

这是一个纯CSS动画的集合。您可以覆盖对象的动画类名，使其具有动画效果。您可以更改与元素相关联的类名来激活其转换。

5。 汉堡

汉堡包是一个CSS汉堡包动画的10+例子的集合，非常适合移动设备的导航。

6。 旋转

Whirl是CSS加载动画的一个策展。它减轻了开发者的痛苦，因为它已经有106个或更多的旋转。

7。 缺德CSS

对于那些希望用一些可爱的动画效果来增加网站趣味的人来说。在网站上找到您需要的所有内容，并下载CSS文件。然后，您就可以开始用简单的类向内容添加动画了。

8。 三个圆点

三点是一组CSS加载动画，只用一个元素制作。我认为该项目不仅可以提高你的CSS技能，还可以提高你的想象力。

9。 CSShake

一些CSS类来移动你的DOM！易于使用的类，将添加不同种类的动摇到你的网站的每个部分。

10。 VOV CSS

这个CSS库中包含了几个功能，包括淡入淡出、缩放、滚轮、摇动、旋转、模糊、滚动和滑动。

11。 模仿CSS

与框架不同，Mimic CSS是内联CSS..这个CSS框架非常好用。
