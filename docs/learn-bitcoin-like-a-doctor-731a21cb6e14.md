# 挖掘比特币

这是一本书，作者是热情的计算机科学家、比特币的支持者卡勒·罗森鲍姆。

他用一个多故障的恶心 Cookie Token 方案(就像今天的法定货币)来解释比特币。

他在每一章都让它防错，直到我们有了像比特币这样的系统。

这是**第一本书**如果你真的想了解比特币的工作原理，你应该读的！
