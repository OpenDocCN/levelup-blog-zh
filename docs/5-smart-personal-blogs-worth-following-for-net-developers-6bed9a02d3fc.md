优秀的开发人员不断地从各种可用的资源中学习:[、开源项目](/5-asp-net-core-open-source-projects-to-gain-practical-knowledge-24fbf9164230)、书籍、视频课程、会议。

另一个重要的知识来源是好的博客，成熟的开发人员、技术领导者或架构师在这里分享他们的经验，并准备回答您的问题。

我从我的个人列表中收集了 5 个最好的博客，这些博客我关注了很多年，我非常喜欢，因为它们对我的职业生涯有很大的积极影响。

# 1.企业工艺

博客[的作者](https://enterprisecraftsmanship.com/)撰写了深入的文章，主要是关于领域驱动设计、单元测试和 CQRS。

一些好文章:

*   [为懒加载辩护](https://enterprisecraftsmanship.com/posts/defense-lazy-loading/)
*   [验证系统边缘的协作](https://enterprisecraftsmanship.com/posts/verifying-collaborations-at-system-edges/)
*   [实体 vs 值对象:最终差异列表](https://enterprisecraftsmanship.com/posts/entity-vs-value-object-the-ultimate-list-of-differences/)

# 2.数据库之星

这个[博客](https://www.databasestar.com/blog/)完全致力于关系数据库(SQL Server，MySQL，PostgreSQL)。

文章都集中在 [SQL 路线图页面](https://www.databasestar.com/sql-roadmap/)上，这使得导航和学习过程变得更加容易。

一些好文章:

*   [SQL 事务和隔离级别](https://www.databasestar.com/sql-roadmap/)
*   SQL 中的分层数据:最终指南
*   [SQL CTE(带子句):终极指南](https://www.databasestar.com/sql-cte-with/)

# 3.卡米勒·格日贝克

[博客](http://www.kamilgrzybek.com/)作者主要写。NET 核心，干净的架构，领域驱动的设计，数据库管理技术。

一些好文章:

*   [勒死。NET 框架应用程序到。网络核心](http://www.kamilgrzybek.com/tag/net-core/)
*   [使用原始 SQL 和 DDD 实现简单的 CQRS](http://www.kamilgrzybek.com/design/simple-cqrs-implementation-with-raw-sql-and-ddd/)
*   [处理并发—聚合模式和 EF 核心](http://www.kamilgrzybek.com/design/handling-concurrency-aggregate-pattern-and-ef-core/)

# 4.太慢异常

[博客](https://tooslowexception.com/)有关于垃圾收集器、异步/等待操作符、异常、诊断和其他 C#/内部的深入文章。NET 概念。

一些好文章:

*   [你何必在意。NET GC…？](https://tooslowexception.com/why-should-you-care-about-net-gc/)
*   [对象如何。GetType()真的有效吗？](https://tooslowexception.com/how-does-gettype-work/)
*   [异步/等待和的危险。产生一张图片](https://tooslowexception.com/the-danger-of-asyncawait-and-result-in-one-picture/)

# 5.阿布哈梅

[博客](https://khalidabuhakmeh.com/)作者主要讲解各种 ASP.NET 核心、实体框架核心和 C#技术。

一些好文章:

*   [防止对 ASP.NET 核心的重复 Web 请求](https://khalidabuhakmeh.com/preventing-duplicate-web-requests-to-aspnet-core)
*   [用实体框架核心和 SQL Server 递归数据](https://khalidabuhakmeh.com/recursive-data-with-entity-framework-core-and-sql-server)
*   [通过静态内容提升 ASP.NET 核心性能](https://khalidabuhakmeh.com/boost-aspnet-core-performance-with-static-content)
