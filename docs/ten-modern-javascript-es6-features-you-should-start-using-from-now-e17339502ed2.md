你可能会问——为什么我们不能使用**“var”**关键字？

在 ES6 问世之前，流行的是**【var】**宣言。但是，事实是**【var】**关键字的用法有问题。一些值得一提的问题:
—用**“var”**声明的变量是全局范围的。
—用**“var”**声明的变量可以重新声明和操作。
—用**“var”**声明的变量被提升到其作用域的顶部，并用未定义的值初始化。

因此，建议停止使用**【var】**关键字，开始使用**【let】****【const】**。
