## 让我们揭示每种语言的真实面目。

并非所有的编程语言都是生来平等的，但是每种语言都是可以模仿的。在编程语言让我们成为它们噩梦般的语法要求的奴隶之后，是时候报复了。

以下是流行的编程语言，展示了它们的真实面目。

## 汇编程序又名技术莎士比亚

见见汇编语言，它是莎士比亚英语在科技界的翻版，备受憎恨。

## 也就是脾气暴躁的奶奶

见见 C，这种编程语言会为一个丢失的分号大发 70 次脾气。

## C++又名邪教领袖

见见 C++，它和 C 一样苛刻，但不知何故却喜欢它。

## 马尔博尔格又名《苦行僧》

认识一下 Malboge，这种语言认为模仿 C++的难度会使它成功，但走得太远而失败了。

## Fortran 又名麻烦制造者

遇见 Fortran，我们所有的麻烦开始的地方。

## 又名绝望的梦想家

见见编程语言 Go，它绝不是下一个大事件。

## HTML/CSS 又名假艺术家

来看看 HTML/CSS 吧，这两个冒牌的家伙会把你的整个网页弄得一团糟，即使他们已经确切地告诉了*该设计什么以及如何设计。*

## JavaScript 又名保姆

认识一下 JavaScript，这种语言的唯一目的是理解 HTML 和 CSS 留下的混乱。

## Java 也就是老师的宠儿

来看看 Java，这种挤进了所有算法教科书的语言仍然被 Python 打败了。

## Python 又名伪代码 2.0

认识一下 Python，这种梦想中的语言太简单了，看起来甚至不像真正的代码。

## MATLAB 又名书呆子

来看看 MATLAB 吧，这种语言除了用于顶级研究之外，拒绝用于任何用途。

## 露比又名崇拜者

认识一下 Ruby 吧，这种语言的名字会让人觉得它价值连城，但又会让人失望。

## 斯威夫特也就是刻薄的女孩

见见斯威夫特，编程界的雷吉娜·乔治，她吓走了除苹果之外的所有人。

## SQL 又名过度思考者

认识一下 SQL，这是一种存储大量数据的语言，它可以将过度思考提升到一个全新的水平。
