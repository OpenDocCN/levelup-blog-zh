# 这 7 条建议会让你在更短的时间内变得高效

> 原文：<https://levelup.gitconnected.com/these-7-tips-will-make-you-highly-productive-in-less-time-a28e1c1e840c>

## 使用行之有效的生产力技巧，在职业生涯中更快成长

![](img/45f04dc8df0b6463c077da91d845b2dd.png)

使用 Canva 创建

大多数软件工程师感觉他们总是没有时间。他们从来没有足够的时间来完成工作。

他们觉得他们需要比其他人花更多的时间工作才能更有效率。

然而，对更多时间的需求是永无止境的。

你需要找到更好的方法来提高生产率，而不是简单地增加工作时间。

如果这听起来对你来说很熟悉，这里有七个非常有效的技巧，可以让你在更短的时间内变得更有效率。不要相信我的话，也不要相信任何人的话。按照简单的步骤做几天，你就能看到自己的改变。

# 1.列出你需要完成的任务

提高效率的最好方法之一是列出你需要完成的任务。这将帮助你保持注意力集中，确保你不会忘记任何重要的事情。

当我们有很多工作要做的时候，一想到它就让我们不知所措。由于我们的非结构化方法，我们不知道从哪里开始。这种感觉降低了我们的能量水平，并把我们推向拖延。

然而，我们不必在同一天完成积压的每一项任务。但是由于我们对需要完成的事情没有一个清晰的认识，我们觉得有太多的事情要做而没有足够的时间。

因此，第一件事是列出你需要做的所有任务。该列表可以提供待办事项中所有事项的结构化视图，并帮助您可视化实际工作量。

# 2.对你清单上的项目进行优先排序

一旦你有了需要完成的工作清单，重要的是要对它们进行优先排序。并非所有的任务都是平等的。优先化可以帮助你专注于你需要首先完成的任务。

人们遵循各种各样的优先化技术。

有些人喜欢先完成最重要的工作。这有助于他们减轻自己的压力。

其他人喜欢先完成最小的任务。它很快提供了一种成就感，并增加了他们的动机水平。

你可以根据自己的情况建立自己的优先排序系统。然而，如果你需要一些帮助，[你可以使用艾森豪威尔矩阵来决定优先级](https://www.eisenhower.me/eisenhower-matrix/)。

建立一个有效的优先排序系统可以帮助你实现短期和长期目标。

# 3.从列表的顶部开始，一路向下

现在，您有了一个基于您的需求堆叠的优先任务列表。是开始工作的时候了。从清单的顶部开始，一次选择一项任务。

通常，人们认为多任务处理是生产力的关键。我有时也会陷入同样的陷阱。然而，[研究表明，一心多用会降低你的工作效率。](https://news.stanford.edu/news/2009/august24/multitask-research-study-082409.html)

当你试图同时做两件事时，你的大脑无法同时关注这两件事。最终，两者都没有做好。

你可能认为同时做两件事是有效率的。但是实际上，你将会花费比你独立完成它们更多的时间。这也会降低双方的产出质量。

因此，在进入下一个任务之前，先关闭第一个任务。

# 4.为每项任务设定一个时间限制，并努力坚持下去

在处理一项任务时，很容易偏离正题。

为了保持专注，为每项任务设置一个计时器。这将有助于你在规定的时间内按时完成任务。

通常，我在开始一项任务时都有一个明确的最终目标。但是在工作的时候，我开始思考各种方法来使它变得更好。我一直在做不必要的努力。

这种行为在软件开发人员中很常见——他们花费不必要的精力试图过早地过度优化他们的代码。

当有几项任务要做时，计时器会让我保持在正确的轨道上。它提醒我需要在给定的时间内完成每项任务。这让我小心翼翼，不要被其他想法或活动分心。

# 5.休息片刻，保持专注和动力

许多人认为，如果他们手头有大量的任务，他们需要连续工作才能更快地完成。他们觉得休息是失去的时间，他们可以用来取得进展。

实际上，情况恰恰相反。长时间连续工作会让你的大脑变慢。很难集中注意力。你最终会犯错误，导致返工。

为了更快地完成任务，你需要有一个清晰的焦点，并且第一次就做对。通过定期休息，你的大脑有时间放松和恢复活力。反过来，它可以帮助你一整天都保持专注和精力充沛。

你可以起来给自己冲杯咖啡休息一下。你也可以喝点水或者吃点零食。或者，你可以简单地盯着远处的一些绿色植物。

一旦你从休息中回来，你的大脑会工作得更好。你会感到精力充沛，工作效率更高。

# 6.尽可能委派任务

授权是一种技能，一旦掌握，可以成倍提高你的工作效率。

不是每一项你必须执行的任务都需要你的专业知识。您可以筛选出其他人可以执行的工作并委派给他们。这将有助于释放你的时间。你可以利用这些额外的时间去做更适合你能力的工作。

如果你在一个团队中工作，授权会相对容易。

但是如果你单干，你仍然可以授权。当然，在这种情况下，你可能不得不付钱给其他人做一些工作。但是你可以利用不做辅助性工作节省下来的时间去做一些能让你赚更多钱的事情。

授权是一项需要时间来掌握的技能。但是一旦你掌握了窍门，就没有回头路了。你会发现它是最好的生产力助推器之一。

# 7.记录你的进步

为了保持你的动力，你需要知道你是否有进步。做到这一点的唯一方法是跟踪它。

因此，开始每天记录你的工作。它可以简单到记下你什么时候开始一项任务，它的计划完成时间，以及你什么时候完成它。您可以根据优先级列表中的每项任务来跟踪它。

这会给你一个鸟瞰你的进步和每天能完成多少任务的视角。通过查看日志，您可以评估上述步骤是否有助于您的工作效率。

我确信，如果你真诚地按照这些步骤去做，你的生产力会提高得很快。有一个日志来再次确认同样会给你一种成就感。它会激励你继续做好工作，不断提高你的生产力。

万一你没有取得好的进展，你可以用数据来找出原因。然后你可以找到弥补差距和提高生产力的方法。

有各种复杂的工具来跟踪你的进展。我使用一个简单的 Excel 表格，它能很好地告诉我是否达到了目标。

不管你用哪种方法，关键是要定期跟踪进度。这可以帮助你找出哪里需要调整，并从长远来看提高你的工作效率。

# 拿走

现在你知道了——在更短的时间内变得更有效率的 7 个技巧。

遵循这些建议，你可以更好地利用时间，更快地实现目标。

然而，请记住，你不可能在一夜之间提高生产率。很难打破多年来形成的工作习惯。因此，对你的进步要有耐心，给自己时间去培养新的生产习惯。

有了时间和毅力，你可以变得高效，更快更好地完成任务。
