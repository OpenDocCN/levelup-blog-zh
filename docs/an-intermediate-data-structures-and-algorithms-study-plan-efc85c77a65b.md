# 查找峰值元素

**话题涵盖:数组，** [LeetCode 链接](https://leetcode.com/problems/find-peak-element/)

这个问题有一个非常简单的 O(n)时间复杂度的解决方案。挑战是在 O(log n)时间内解决这个问题。

当我看到 O(log n) time 时，我的脑海里会跳到“二分搜索法”在我的初学者文章中，我说过，“二分搜索法是一个基本的计算机科学概念，它远远超出了一组数字。”

好的，这可能仍然是一个数字数组，但是这是一个有趣的二分搜索法问题，因为这个数组是没有排序的，你不是在寻找一个特定的元素。
