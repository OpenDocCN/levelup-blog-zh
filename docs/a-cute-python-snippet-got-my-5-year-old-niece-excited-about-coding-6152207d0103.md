当 Python 遇到艺术

随着社会日益以计算机为导向，人们一致认为编码是未来技术的新语言也就不足为奇了。根据美国劳工统计局的[预测，在未来十年，STEM(科学、技术、工程和数学)职业的增长速度将比所有职业的增长速度快两倍多。现在，编程就像数学和科学一样重要。](https://www.bls.gov/opub/btn/volume-10/why-computer-occupations-are-behind-strong-stem-employment-growth.htm)

在很小的时候，孩子们就接触到了编程的世界，因为父母相信这将为他们一生的成功奠定基础。这种成功不仅仅是有利可图的职业机会，还有解决问题的技巧。有时，我会对自己说，让孩子们真正对编码感兴趣的最好方法是什么？**然后，上个月我侄女过生日的时候，一个灯泡熄灭了。**

她刚满 5 岁，在她的生日派对上，她让我画她的小跟班，这是她有史以来最喜欢的角色！虽然我很喜欢那些黄色的小家伙，但我当然不想因为我不会画画而破坏他们的形象！身边有一个 5 岁的孩子，你一定知道她没有立即得到她想要的东西会发多大的脾气。然后突然，我想到我可以画画，但是用 Python！
