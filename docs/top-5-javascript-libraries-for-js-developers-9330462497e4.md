# 介绍

当使用原始的 HTML、CSS 和 JS 编码时，构建轻量级的交互式前端 web 应用程序可能会非常繁忙。

为了使生活变得简单，开发社区引入了许多易于使用的 JS 库，以使高度交互式的 web 应用程序消耗更少的精力和时间。

让我们来看看前 5 名的 javascript 库，它们将把你的普通 web 应用程序变成一个令人兴奋的交互式应用程序。

# TiltJS

TiltJS 是一个 javascript 库，为您的 JQuery 提供了一个由 60+ fps 轻量级悬停倾斜效果支持的 requestAnimationFrame。可以使用`yarn add tilt.js`简单安装，也可以从[官网](https://gijsroge.github.io/tilt.js/)下载到本地机器。

关于安装和使用的更多细节，请参考他们的官方 Github 文档。

# 光滑的

Slick 是一个 jquery 插件，支持断点、CSS3 转换、触摸事件/滑动等等。

它的一些主要特点是:

*   完全响应，可随容器扩展
*   带断点的单独设置
*   功能齐全，使用 CSS3
*   启用或禁用刷卡的能力
*   无限循环
*   箭头键导航
*   添加、移除、过滤和非过滤幻灯片
*   自动播放、点、箭头、回调等…

有关安装和使用的更多详细信息，请参考他们的官方 Github 文档。

# 比迪奥

[Bideo](https://rishabhp.github.io/bideo.js/) 是一个 javascript 插件，可以让你在容器的主体或背景中播放视频。

它的主要特性之一是可以自动调整容器和浏览器屏幕的大小。

关于安装和使用的更多细节，请参考他们的官方 Github 文档。

# 洛蒂耶斯

LottieJS 是一个 javascript 库，它提供了各种各样的动画，您可以将它们包含在您的 web 应用程序中。这可以美化您的应用程序，为您的前端应用程序提供高级外观。

它提供的一些功能有:

*   控制动画速度
*   更改背景颜色
*   更改图层颜色
*   跨平台——HTML、WordPress、Webflow、Eliminator、Android 和 IOS

Lottie 是一种基于 JSON 的动画文件格式，支持设计人员像运送静态资产一样轻松地将动画运送到任何平台。它们也可以在没有像素化的情况下缩放大小。

为了更好地理解 Lottie 以及如何在您的 web 应用程序中使用它，请访问这里的。

# 阿尼伊斯

[AniJS](https://anijs.github.io/) 是另一个轻量级的 Javascript 库，提供 CSS3 动画并帮助构建交互式定制 UI 组件。

它提供了编码人员和设计人员之间更好的集成和协作

关于安装和使用的更多细节，请参考他们的官方 Github 文档。
