## **为什么选择 Python 中的面向对象编程？**

Python 采用面向对象的方法设计，具有以下优势:

*   提供清晰的程序结构和干净的代码
*   简化现有代码的维护和修改
*   因为类是可共享的，所以代码可以重用
*   因为类是可共享的，所以代码可以重用，还有许多其他类似的优点。

**一个物体有两个特征:**

*   属性
*   行为

**举例:**汽车是可以成为对象的，其特征如下:

*   颜色、尺寸、重量=属性
*   启动发动机、加速、停止=行为
