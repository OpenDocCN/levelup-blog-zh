在大学里，我经常听到一句名言:

> 小事带来完美，但完美不是小事。

在编程中，这句名言最适合用来启发灵感。灵感是你的大脑每天需要的红牛，为你的大脑提供能量。

> 成功是 1%的灵感，99%的汗水。

没有灵感，如果你被雇佣了，你很快就会变成一个累赘。你靠换几个 LOC，补好票，赶紧回家来维持。失业离经济衰退只有一步之遥。

如果你失业了，祝你转行好运。

没有学习，没有编程生涯。

# 如何获得每天编码的灵感:

然而，学习编程本身并不鼓舞人心。如果你想学习工具和技术，谷歌永远是你的朋友。你在 YouTube 上也有 1000 多个教程。但那是学习。灵感在哪里？

当你已经通过自己的激情完成了大量的工作时(几乎没有别人的指导)，灵感就来了。一两个 bug 已经不能阻止你的意志力了。

但是你如何达到那个水平呢？

除了 Medium，以下是我在日常生活中尝试获得灵感的一些方法。

# #1.TechCrunch + Crunchbase

TechCrunch 让您了解行业的最新技术趋势。它带来了最新、最前沿的创业公司的产品。

它的姐妹网站 **Crunchbase** 也保存了许多有价值的数据，如谁认识谁(社交资料)、谁做什么(首席技术官/首席执行官资料)、谁拥有多少(估值)。

如果你正在寻找一条创业之路，Techcrunch 就是你通往巅峰的阶梯。

# 第二名:安吉利斯:

Angellist 将投资者、初创公司创始人和员工联系在一起。

AngelList 成立于 2010 年，其使命非常成功。它不仅是一个个人资料聚合器，也是一个活跃的社区。你可以得到关于投资的最新消息。

最重要的是，你可以结识有影响力的科技领袖，以及他们的人生使命，就像你会浏览脸书与你的朋友和家人联系一样。你可以凭借你对社区的贡献在这里得到提升，也可以得到你想要的利益相关者的关注。

# #3: GitHub 探索

没有什么比一个很酷的工具更能激励程序员了。 [GitHub explore](https://github.com/explore) 每天为你带来多样的、每日流行的 feeds。

但是，您不能像消费文本内容一样消费它。你想试着在它的基础上发展，但在某一点上，你会感到疲倦。有时候，你可能也会羡慕其他开发者。
