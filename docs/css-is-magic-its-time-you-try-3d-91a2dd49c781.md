## 首先，翻译是如何工作的？

`translate`方法基本上将一个 HTML 元素从它的实际位置移开，而不会干扰布局树上的任何其他兄弟/父元素。总而言之，`translateX`方法将元素左右移动，而`translateY`方法将元素上下移动。

![](img/7e663d328edb51dde29faeb2644f01f6.png)

平移如何在 X 和 Y 轴上工作

> *`perspective`属性定义了对象离用户有多远。因此，较低的值会比较高的值产生更强烈的 3D 效果。*
