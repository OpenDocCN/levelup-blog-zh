在当今世界，我们都肯定可以使用更多的钱。幸运的是，作为开发人员，我们有一项可以用来赚钱的宝贵技能。

在这篇文章中，我们将看看作为开发人员，我们如何赚取额外的钱。

# 作为开发人员工作

以我们的软件开发技能来说，做一名开发人员可能是最常见的赚钱方式。

成为一名雇员比自由职业者更安全一点，但是他们都有风险，所以我们必须有其他赚钱的方法，我们在下面概述。

# **卖数码产品**

有很多东西是数字化的。除了我们的时间，数字产品不需要任何金钱。

我们可以写独家内容，放在付费会员网站后面。

此外，我们可以为 Angular 和 Vue 等内容管理系统和框架销售主题和插件。

我们也可以制作自己的组件库，并向人们收费。

# 软件即服务

软件即服务是我们可以构建并向使用它的人收费的软件。

唯一的成本将是托管。然而，如果我们想节省员工成本，那么我们必须自己做很多事情，如客户支持和维护。

因此，成本可能很高，而且可能非常耗时且压力很大。

此外，我们必须自己构建它，并通过构建原型来验证我们的服务有市场。

所有这些都需要付出很多努力，所以它肯定不适合所有人。我们做的事情越多，托管费用也越多。

# 移动应用

我们也可以开发自己的移动应用程序，并通过订阅或直接付费购买应用程序来销售。

这与软件即服务是一脉相承的。我们首先必须建立一个原型来验证我们的产品是否有市场，然后我们可以加倍努力将其投入生产。

此外，我们在 SAAS 方面也遇到了同样的客户支持和维护问题。

要么我们雇佣员工使我们更容易，这是很昂贵的，要么我们自己做所有的事情。

# 博客

我们可以创建自己的博客，通过在上面放广告或附属链接来赚钱。

此外，一旦我们的博客受欢迎，我们可以接受赞助商的帖子。

这是一个长期的努力，因为我们必须等待搜索引擎给我们一些搜索流量。此外，我们需要不断推广我们的博客。

# 书

编程书籍总是很受欢迎。人们需要从书本中学习。通常情况下，它会以电子书的形式出现，而不是实体书。

我们可以为像奥赖利这样的大出版商写作，也可以自己出版。

# 在线课程**课程**

教编程比我们自己做编程更容易，所以如果我们想要一种更轻松的方式从我们的技能中赚钱，我们最好创建一个在线课程。

我们可以制作视频或者写一个基于文本的课程。此外，我们可以在我们的在线课程中增加一对一辅导课程或小组辅导课程。

很多人被困在基本的编程任务中，所以我们肯定可以通过为他们解决这些问题来赚钱。

# **捐款**

我们可以在博客或社交媒体上接受来自 Patreon 或 Buy Me A Coffee 的捐款。

这样，如果有人捐款，我们可能会得到一些额外的现金。然而，这都是抽奖的运气，因为大多数人不会为他们已经可以免费得到的东西捐款。

# **制作我们自己的工作板**

我们可以制作自己的求职公告板，让人们付费在我们的公告板上发布信息。

当然，这是另一种软件即服务，因此它伴随着支持和维护软件即服务的所有障碍。

此外，我们必须努力推广，以获得任何牵引力，我们必须从搜索引擎获得流量，以便人们可以看到我们的网站。

# 结论

作为开发人员，有很多方法可以赚外快。我们可以全职工作或自由开发者的工作。

此外，我们可以将自己的软件作为服务或移动应用程序。然而，这些都是艰难的选择。

更容易的选择可能与写作或教学有关，如辅导、博客和写书。
