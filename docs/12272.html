门户网站

React 门户是一种将子组件放入DOM节点的方法，该节点存在于父组件的层次结构之外。尽管门户可以安装在DOM树中的任何位置，但它们在其他方面的行为就像普通的React孩子一样。上下文也将与门户和任何其他React组件一起工作。门户的典型用例包括模态、弹出菜单、祝酒词等，在这些情况下，您需要在DOM three中的更高级别上安装组件。

高阶组件

反应高阶组件(HOC)是一种重用组件逻辑的模式。hoc是将一个组件作为参数并返回一个新组件的函数。典型的组件将props转换成DOM中的节点，而高阶组件将一个组件转换成另一个组件。
