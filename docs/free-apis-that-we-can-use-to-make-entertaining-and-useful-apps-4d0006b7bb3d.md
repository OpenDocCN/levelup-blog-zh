# 笑话 App

有了[查克·诺里斯数据库](http://www.icndb.com/api/) API，我们可以制作自己的应用程序来展示关于查克·诺里斯的笑话。

我们可以随心所欲地展示它们。我们可以随机展示它们，按类别获取笑话，以及我们能想到的任何其他方式。

不需要认证，所以我们不需要注册就可以深入研究。

# 琐事游戏

[Open Trivia](https://opentdb.com/api_config.php) API 有一个通过他们的 API 提供的问题列表。每个问题都有正确和错误的答案。

所以，我们可以用它来轻松制作一个琐事游戏 app。我们可以只显示问题和选项，并对照列出的答案进行检查。

不需要认证，所以我们不需要注册就可以深入研究。

# 唐纳德·特朗普引用应用程序

Tronald Dump API 为我们提供了唐纳德·特朗普成为名人后说过的话。

有很多方法可以使用它们，并以有趣的方式展示报价。

不需要认证，所以我们不需要注册就可以深入研究。

# 漫画应用

有了 [xkcd](https://xkcd.com/json.html) API，我们可以毫不费力地从 xkcd 中获取所有漫画。

我们只需通过 ID 对每个漫画提出请求，然后我们可以在自己的应用程序中显示作为图像 URL 和元数据提供的漫画。

不需要认证，所以我们不需要注册就可以深入研究。

# 必应地图应用

[必应地图](https://www.microsoft.com/maps/)是地图数据的绝佳资源。我们可以用它来显示地图、获取地理数据、街景、路线信息、位置数据等等。

这一点经常被忽视，因为谷歌是主要的地图提供商，但必应也免费或低价提供大量数据。

为了使用这些数据，我们必须注册一个 API 密匙来访问它。

# 谷歌地图应用

[Google Maps](https://developers.google.com/maps/) 就是大家找地图 API 时想到的地图 API。

它还有免费和付费两个层次，提供的信息与 Bing Maps API 提供的大部分相同。

我们所要做的就是注册一个 API 密钥。但是，请注意，注册免费层也需要信用卡。

# 冠状病毒跟踪器

[新冠肺炎](https://covid19api.com/) API 拥有来自约翰霍普金斯大学的最新数据，关于过去几个月在世界各地蔓延的新冠肺炎病病例。

它记录了过去的数据和最新的感染病例数据。

如果我们想知道我们什么时候可以再次走出家门，那么就用这个建立一个应用程序，自己查看趋势。

不需要认证，所以我们可以开始使用它。

# 美国健康保险应用程序

Healthcare.gov 的[API 有关于美国保险市场的数据，我们可以马上得到。](https://www.healthcare.gov/developers/)

这些都以 JSON 的形式提供，所以我们可以用我们喜欢的方式显示它们。它有教育内容，像解释什么是保险市场，所以我们可以从他们那里学到更多。

# 求职应用

除了他们的常规网站之外，各种工作网站还通过他们的 API 提供他们的数据。

它们包括 [Github Jobs](https://jobs.github.com/api) 、 [GraphQL Jobs](https://api.graphql.jobs/) 、[事实上](https://www.indeed.com/publisher)、 [ZipRecruiter](https://www.ziprecruiter.com/publishers) 以及更多任何类别的工作，包括公共和私营部门。

有大量的数据，如描述、名称、位置等等。此外，我们可以通过关键字或位置来搜索它们，这非常方便。

要访问它们，我们只需注册一个 API 密钥，然后就可以开始玩这些 API 了。

要制作一个让我们显示和搜索公共部门工作数据的应用程序，我们可以使用[Search.gov 工作](https://developer.usajobs.gov/) API。

它有一个完整的端点列表，我们可以调用它来访问联邦工作数据，包括按位置搜索、招聘路径、谁有资格做哪份工作等等。

# 结论

我们可以用免费的公共 API 构建许多应用程序。它们包括求职应用、地图应用和健康信息应用。

如果我们想要一些娱乐，我们也可以构建显示的应用程序，并用一些琐碎的 API 制作一个琐事游戏。
