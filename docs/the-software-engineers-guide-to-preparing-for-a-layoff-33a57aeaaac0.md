# 软件工程师准备裁员指南

> 原文：<https://levelup.gitconnected.com/the-software-engineers-guide-to-preparing-for-a-layoff-33a57aeaaac0>

## 我现在这样做是为了未雨绸缪。

![](img/77f2dacf387e593b53bda6b79cb48229.png)

阿洛拉·格里菲斯在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

十多年来，科技行业的气氛从未如此紧张。

所有的科技公司都在疯狂裁员。

条纹[裁员](https://www.cnbc.com/2022/11/03/stripe-plans-to-lay-off-14percent-of-workers.html) 14%的员工。脸书解雇了 11，000 名员工。Lyft 和其他科技公司也大幅裁员，试图削减成本。

工程师们一直在谈论更多的公司将进行裁员。

《华尔街日报》称[人才大战已经结束](https://www.wsj.com/articles/tech-workers-face-new-reality-talent-wars-layoffs-meta-twitter-11668167520)。

招聘表已经从工程师选择他们想加入的公司，变成了公司选择雇佣的工程师。

安静的辞职改变了我们对工作的看法，在这次经济衰退浪潮之前，它有它的时刻。

许多工程师现在听到这个消息感到焦虑。他们晚上很难入睡，比平时更加努力地工作，以证明自己对公司的价值。

如果你目前有工作，但不确定宏观经济即将发生的变化，你该怎么办？

我现在这样做是为了未雨绸缪。

# 🔪通过面试提高你的面试技巧

一直坚持面试。

面试就像公开演讲。每次做的时候都会有那种蝴蝶和焦虑的感觉。

面试的坏处是占用了你大量的工作时间和私人时间。当你经常这样做时，你会发现它花费的时间越来越少，因为你的简历总是最新的，你会更好地剔除你不想为之工作的公司。

你面试的次数越多，你解决面试问题的能力就越强。你可以当场回答问题，并向面试官提出好问题。

> 这来自于我们大脑中的一个现象——一段记忆被回忆得越频繁，它的神经网络就变得越强大。随着时间的推移，通过持续的回忆，记忆会在海马体和大脑皮层中被编码。最终，它独立存在于大脑皮层中，并被长期储存起来。

你可以练习并深入了解面试问题。你总是知道自己的价值。

我们经常认为想要继续面试是一个危险信号，应聘者也经常认为这是在“浪费公司的时间”然而，从另一方面来说，即使已经有提供给他们自己的安全网的机会，公司还是会继续面试候选人。此外，他们一直公开招聘信息，以防任何招聘渠道或工程资源出现问题。

更新你的简历。一旦你被解雇，再采取行动就晚了。

# 🏋️‍♂️不要过度劳累

多工作是对你感受到的压力的自然反应。

如果你需要采取行动来减轻风险，那么准备面试是最好的方式。

随着许多公司解雇软件工程师，你可能会努力工作来证明你在公司的价值。

工作时间越长，裁员的可能性就越大。

绩效可能是决定解雇谁的众多因素之一。然而，你无法控制的其他决定也会影响裁员决定。

做一些工作之外的事情。这可能违反直觉，但当你感觉到风险时，你的自然反应是投入更多时间，每周工作 60 小时，这让你更加偏执，如果真的发生裁员，打击会更大。

记住，在雇主的眼里，你只不过是他们会计账本上的一个数字。

# 🤙联系你以前的同事，整理你的社交网络

联系你以前联系和共事过的同事/经理/技术主管/导师。

如果有一天你需要推荐，他们至少会记得你。一般来说，即使没有裁员，这样做也很好。

这就是你如何走在新闻的前面，了解新的计划、失败的项目等等。目标是永远不要感到惊讶。

试着和被解雇的人保持联系。除了成为一个好朋友和人类，你还可以判断他们多快能找到新工作，在哪里找到，这样如果你想跳槽，你就知道谁在招聘了。

当你接触你的人际网络时，真诚地试着了解他们的现状。

与老同事、老上司重新联系时的一个错误就是过多地谈论自己。

> 通过对别人感兴趣，你可以在两个月内交到比两年内试图让别人对你感兴趣更多的朋友。——戴尔·卡内基

真诚地对他人感兴趣比试图变得感兴趣要好。

*为什么是*？因为所有人类都有一种天生的被欣赏的欲望。

当我们一直在说的时候，我们就剥夺了其他人执行这一重要行为的机会。上一次刚认识的人不停地说自己有多棒多成功，你是什么感受？至少，你可能对这个人产生了低度的嫉妒和厌恶。

不要让对话安全，停留在表面层面。闲聊是在你不熟悉的网络中打破僵局的好方法，但是只有当你深入交谈时，你才能了解一个人。在前 10-20 分钟内保持闲聊，以供参考。

我们通常不想听起来对我们不知道的事情一无所知。然而，对方很可能会把你的沉默当作你不感兴趣的表示。听录音，问问题。

# 关闭思想

经济衰退可能令人望而生畏。在经济低迷时期，准备是关键。你的准备可以帮助你在找新工作时增强信心，让你的大脑休息一下，这样你晚上就能睡得更好。

你可以根据以下三条建议为自己的失业做好准备:

*   提高你的编码技巧和面试技巧。市场仍然很好，但是除了丰厚的薪水之外，还是要期待一些东西。数字工资仍将高于大多数行业。确保你比你旁边的人更有价值，但不要背后捅别人一刀。它可能会回来困扰你。
*   过度工作是焦虑的表现。多工作并不一定意味着你不会被解雇。它只会每天削弱你的精神健康。试着忘掉你的烦恼，为将来储存能量。
*   联系你的旧关系网有助于你和老同事重新联系，也有助于你的未来规划和求职机会。
