# STL 算法概念

C++ STL 提供了执行多种任务的函数:排序、搜索、复制、修改、执行数值计算和重新排序等等。这些算法的一个有趣的方面是，它们不是附属于各种 STL 容器的成员函数。相反，这些函数是在特定容器内创建的迭代器上执行的全局函数。

这种实现的一个很大的优点是，您只需了解一种算法是如何工作的，然后就可以将它应用到您选择的容器中。许多算法将在多个容器上工作。一个更强大的特性是，您可以在自己的、用户定义的容器上使用这些算法。

这种实现显然不属于面向对象的编程范式，而是一般函数式编程的一个例子。这意味着容器和算法是不同的，它们通过一个接口进行互操作。这种设计选择的结果是一些算法将不能与一些容器一起工作。更糟糕的是，一些算法，即使它们可以与容器一起工作，也可能效率低下。程序员有责任决定哪些算法适用于哪些容器。

# STL 算法类别

STL 中有几大类算法。以下部分将解释这些类别是什么，以及类别中算法的一些简要描述。

# 非修改算法

非修改算法不会改变容器中元素的顺序，也不会改变容器中的任何值。例如，`for_each`函数对容器中的每个元素执行操作。由于范围`for`循环现在可用，该功能不再像以前那样有用。计数元素的函数有:`count`和`count_if`；并且找到最小和最大元素，`min_element`和`max_element`。

搜索数据有几个功能:`find`、`find_if`和`find_if_not`。有两个搜索功能用于子范围中的第一次出现:`search`和`search_n`。还有一个查找最后一次出现的功能:`find_end`。

对于数据范围的处理，有两个函数:`equal`，用于确定两个范围是否相等，以及`is_permutation`，用于确定两个无序范围是否包含相同的元素。

最后，尽管并不详尽，有三个函数可以确定容器的元素是否匹配某个标准:`all_of`、`any_of`和`none_of`，它们的作用正如它们的名字所暗示的那样。

# 修改算法

这类算法改变容器中元素的值。一些算法直接改变容器的值和/或在它们被复制到不同范围的元素时修改这些值。

修改容器元素主要有两个功能:`for_each`和`transform`。`for_each` 函数对容器的每个元素执行修改操作。另一方面，`transform`函数可用于修改容器的元素，并将它们复制到不同的目标范围。

有两个复制容器元素的函数:`copy`，它从一个开始元素复制，和`copy_if`，它复制匹配一个标准的容器元素。

您可以使用两个函数来移动容器中的元素:`move`，从起始位置开始移动元素，以及`move_backward`，从最后一个元素开始移动容器元素。

替换容器元素有两个函数:`replace`，用另一个值替换某个值的元素，和`replace_if`，用另一个值替换匹配某个标准的元素。

在这一类别中，我要提到的最后一个函数是 merge，它接受两个范围并将它们合并在一起。

# 移除算法

这个类别实际上是修改算法的一个子类别，包含删除容器元素的函数。第一个是`remove`，它删除匹配给定值的容器元素。一个类似的函数`remove_if`删除符合指定标准的元素。`remove_copy`函数复制不匹配指定值的容器元素，而`remove_copy_if`复制不匹配指定标准的容器元素。

`unique`函数将删除任何重复的相邻容器元素。还有`unique_copy`函数，它复制容器元素，同时也删除相邻的重复元素。

# 变异算法

下一组算法修改了元素的顺序，但没有改变它们。`reverse`函数反转容器的顺序，而`reverse_copy`在复制容器元素的同时复制它们。

`rotate`函数将使用迭代器位置来移动容器的元素。`next_permutation`函数改变容器元素的顺序，这样如果元素的顺序是(1，2，3)，下一个改变可能是(1，3，2)。

`shuffle`函数随机打乱容器的元素，`partition`函数将那些符合标准的元素移动到容器的前面。

# 排序算法

STL 中有几种算法用于对容器的元素进行排序。主要功能是`sort`。该函数将对基于迭代器的容器元素范围进行排序，通常使用*快速排序*算法来实现。sort 函数使用的排序算法类型取决于实现。

如果你有一个包含重复元素的容器，并且你想保持这些重复元素的顺序，你可以使用`stable_sort`函数。如果您只想对一定数量的元素进行排序，可以使用`partial_sort`功能。

我在上面提到的作为变异函数的`partition`函数也可以用于对容器元素进行排序，因为排序顺序可以作为使用的标准。

如果您想查看容器是否已经分类，您可以使用`is_sorted`功能。

# 排序范围算法

这些是您可以在元素已经排序的容器上调用的一组函数。`binary_search`函数在容器范围内搜索一个元素。includes 函数确定一个范围中的每个元素是否也是另一个范围中的元素。

`lower_bound`函数查找大于或等于给定值的第一个容器元素，这意味着这是一个较小的值可以插入到一个范围的地方，而`upper_bound` 函数查找给定值可以插入到一个排序范围的最后一个地方。

`merge`功能将一个范围合并到另一个范围。容器范围有三个基于集合的函数:`set_union`用于处理两个范围的排序联合，`set_intersection`用于两个范围的排序交集，`set_difference`用于查找一个范围中不属于另一个范围的所有元素。

# 数字算法

有一些算法用于数字处理。其中最常用的是`accumulate`函数，例如，它用于计算总和与乘积。该组中的其他函数`inner_product`、`adjacent_difference`和`partial_sum`并不常用，将在以后的文章中讨论。
