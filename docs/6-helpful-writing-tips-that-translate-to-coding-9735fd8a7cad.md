# 转化为编码的 6 个有用的写作技巧

> 原文：<https://levelup.gitconnected.com/6-helpful-writing-tips-that-translate-to-coding-9735fd8a7cad>

将这些写作技巧应用到你的编码中，以克服你的编码障碍

有时候，绕过作家的瓶颈可能是一个挑战。你可能会发现自己被困在一个似乎不可能逃脱的地方。有一些方法可以让你试着克服你的障碍，但是有时候更简单的方法是结束一天的工作，离开，然后当你精神焕发的时候再回来。

有一些方法可以帮助你克服编码障碍。

其中一些方法包括关注你的能量水平，找出如何优化获取想法，以及休息一下。

让我们看看其余的。

1.  **考虑你的能量水平**

我无法准确地告诉你什么时候是为你写作或编码的最佳时机。但是我能建议的是，在你精力最充沛的时候考虑你写作或编码的时间。

我自己是个早起的人，因为那是我最有活力的时候。因此，我会在早上完成大部分的写作或编码工作。

**2。立即写出想法**

想出创造性想法的最好方法是把你的潜意识想象成一个储存你多年来积累的所有知识和灵感的空间。

我们都希望我们有创造力的头脑能在我们最需要的时候随时为我们提供必要的想法。然而，事实往往并非如此。不管你有多努力，你很可能会注意到你的想法通常出现在最不方便的时候。因此，最重要的是有一个记事本或其他设备来写你的想法。否则，创意的火花将在你的脑海中永远消失。

**3。地点变更**

如果你在努力寻找表达自己的声音或合适的词语，你可能需要改变一下环境。找到灵感的一个方法是花时间去旅行。

地点可以是任何地方，从经常有人进进出出的咖啡店，只要人足够多，不会太分散注意力，但也不要太多，以至于让人不知所措，或者可能是天气好的时候公园外面的长椅。有时候，你所需要的只是离开办公桌 20 分钟，接触水、咖啡(或茶！)、纸和笔。

在当今这个忙碌的世界，不断有干扰降低生产力，自我反省变得比以往任何时候都更重要。它让我们客观地评估我们的斗争，并通过摆脱困境来克服它们

**4。休息一下**

写作之余休息一下是很重要的。这些休息时间可能短至几分钟，也可能长达几天或几周。

休息可以帮助你重新充电，摆脱写作瓶颈。为了让自己专注于手头的任务，改变你的物理环境也很重要。

一些人发现从他们的项目中休息一会儿，做一些完全不同的事情是有帮助的；无论是去散步，培养一个有趣的爱好，还是与朋友和家人共度时光。

**5。利用环境获得灵感**

几乎日常生活中能看到的很多东西，都能给你提供急需的灵感。寻找新想法可能就像阅读 Medium.com 或福布斯(或 GitHub 或 StackOverflow)上的一些出版物一样简单。你会惊讶地发现，你可以找到多少不同的角度来获得各种主题的独特而有趣的视角。这将有助于你对你的工作有足够的想法，你永远不会陷入几个小时试图弄清楚你应该写什么。或者代码。

**6。太多故事**

作家经常发现很难决定到底写什么样的故事。解决这个问题的唯一方法是通过消除的力量。当选择一个故事的主题时，一个人应该总是考虑以下几点:一个有强烈情感和故事的主题，人们关心的东西，或者会让他们好奇的东西。

这也可以说是有许多功能需要添加。幸运的是，您有像优先级和利益相关者这样的标准来帮助决定，但是说实话，您可以提出建议。想想用户、开发者或受众想要什么，并推荐他们。从其中简单的部分开始。
