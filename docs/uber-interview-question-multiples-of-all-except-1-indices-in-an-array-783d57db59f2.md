# 问题

给定一个名为`numbers`的整数数组，返回一个新数组，其中在索引`i`处，`numbers`数组中所有整数的倍数，除了在索引`i`处的整数。下面是一些例子，例子 1:

输入:`[1, 2, 3, 4, 5]`

答案:`[120, 60, 40, 30, 24]`

示例 2:

输入:`[7, 8, 9]`

回答:`[72, 63, 56]`

示例 3:

输入:`[12, 2, 6]`

答案:`[12, 72, 24]`

# 解决办法

在这里，我们遍历数组一次，以创建数组中所有数字的总倍数。一旦我们有了它，我们就可以再次遍历输入数组，通过简单地划分索引处我们不需要的值来创建输出数组。搞定了。这很简单。由于我们遍历了两次数组，这里的时间复杂度是`O(n) + O(n)`，还是 O(n)。解决方案的空间复杂性也是如此。
