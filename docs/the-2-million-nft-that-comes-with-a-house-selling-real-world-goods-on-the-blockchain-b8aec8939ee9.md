我坚信区块链技术的潜力。作为一个相对较新的人，我热衷于扩展我的“初学者思维”,并一直试图学习更多，同时接受需要说服的人的怀疑和天真。

# NFT 包括什么？

NFT 的关键原则是它们是不可替代的——完全独一无二的。虽然创作者在很大程度上无力阻止任何人获取数字媒体文件(如图像)的非官方副本，但他们可以为该文件铸造一个 NFT，该文件可用于无可辩驳地表示一个原始副本。这种代币允许他们出售原作并赋予其所有权。

当我在毕普拍卖后开始研究 NFT 时，我天真而错误地认为买家会收到艺术品的电子版作为价格的一部分——谁知道呢，也许他们收到了。当我列出我的第一张 NFT——今年早些时候拍摄的一张数码照片——时，我决定将 NFT 的原始图像文件包括在内，并删除我笔记本电脑上的所有其他副本。我认为令牌本身应该附带艺术品。

我后来了解到，虽然 NFTs *可以*包含令牌所代表的‘东西’,但是它们*没有*可以。

莱昂国王乐队已经出售了他们最新专辑的 NFT，其中一些包括终身前排音乐会门票。其他的包括该唱片的限量版。至关重要的是，NFTs 不包括音乐的主录音或版权。NFT 本身就是*事物*。

NBA 热门视频是可收藏的体育卡片的现代等价物，包括 NBA 篮球重要时刻的数字视频。但是与 NFTs 相关的视频剪辑并不是现存动作的*唯一*拷贝。

然而，在谢恩·杜尔杰洛夫的《千橡市的房子》的 NFT 中，他把房子本身也算进了售价中。

他不需要这么做，他本可以卖掉他委托制作的数字艺术品。但他认为他看到了一种利用 NFT 技术出售房子的创造性方法，谁知道呢——也许会成功呢？

# 现实世界和网络世界的复杂交集

区块链和加密货币领域的监管格局是最大的未知数之一。政府热衷于施加控制和影响，但是[在现阶段似乎不确定如何实现这一点。](/why-are-governments-choosing-to-remain-ignorant-and-short-sighted-about-bitcoin-e2e62bfcb71e)

截至目前，美国政府采取的唯一决定性步骤是对买卖加密货币并从中获利的加密货币交易商的资本收益征税。拜登政府正在推进的即将到来的增税将使最高税率的资本收益从 20%增加到 39.6%。

这种上涨是最近提出的解释比特币价格短期下跌的众多原因之一。据推测，在预期的增税之前，卖家热衷于将近年来最知名的加密货币价格飙升带来的收益具体化。

虽然不可能说这是否解释了暂时的下降，但这无关紧要。它为涉及区块链的交易和在现实世界中进行的交易之间的交叉提供了一个有用的线索。同样的资本利得税的增加也将适用于房地产销售的收益和利润。

对于史蒂夫·杜尔杰洛夫来说，这只是事情变得有点复杂的开始，他试图用 NFT 卖掉一所房子。

# 谁执行法律？

有许多法律法规适用于房地产的买卖。无论是通过 NFT 还是传统渠道进行交易，这些规则都需要作为交易的一部分来遵守和应用。

资本利得税仍将到期——考虑到这所房子是以 74.6 万美元的价格购买的，杜尔杰罗夫曾一度设想售价约为 2000 万美元，这并非微不足道。他真的在计划中考虑到这一点了吗？我敢肯定，他会很高兴从 2000 万美元的销售中获得 60%的利润，但税收仍然应该缴纳。

拍卖过程中还出现了其他问题，表明买卖双方需要克服的障碍。

*   所有权如何从卖方转移到买方？什么时候会发生？
*   法律上的财产转让是如何进行的？谁来资助？
*   如果潜在买家没有现金，他们将如何安排融资？NFT 是否将需要从传统银行贷款人处获得抵押贷款的买家排除在外？

值得注意的是，拍卖完成后，一份智能合同将在支撑 OpenSea 平台的以太坊区块链上自动执行。这将把购买价格从买方的加密钱包转移到卖方。艺术品本身的 NFT 会向另一个方向移动。这是自动发生的，无需人工干预，这就是区块链智能合约的本质。

另一项资产——房子本身——的转让会发生什么？

接下来是房地产经纪人通常对房产的描述，描述工艺质量、浴室数量等等。更重要的是，它提到买方被认为愿意遵守谢恩网站上描述的条款和条件。

这些大概是以非常规方式管理房产销售的基本法律条款和条件。这些真的能被执行吗？如果买方或卖方决定不遵守这些条款，加利福尼亚或联邦法律系统会愿意执行这些条款吗？即使法院试图这样做，他们也能这样做吗？

谢恩·德勒格洛夫的 NFT 会卖出去让我们知道吗？我希望如此。

# 谁的规则适用？

如果找到了买家，我们将拭目以待，看看 OpenSea 同意的(并永远记录在区块链埃瑟伦博物馆的)出售是否最终从头到尾都完成了，同时完全遵守所有适当的法律法规。在我看来，这在很大程度上取决于买方和卖方的遵守情况，以及他们自愿参与 OpenSea 常规系统售后服务的意愿。

## 预付一栋房子的费用是不寻常的

大多数传统的购房者在交易开始时都会同意一个销售价格，但是钱本身直到交易结束才从买方转移到卖方。有时，它可能会由双方托管，直到交易完成并移交钥匙。

然而，据我所知，随着 NFT 号在 OpenSea 的房子被拍卖，拍卖一结束，钱就会从买家手中转移到卖家手中(在所有的法律还没有开始之前，更不用说完成了)。

有许多利害攸关的事情，要到以后很久才会处理。

# 该合同将如何执行？

那些不幸成为涉及传统货币的金融诈骗受害者的人通常会通过警察和当地执法机构求助。当交易涉及分散和不受政府控制的密码货币时，当局很少感兴趣，也只有有限的能力(或愿望)来执行法律，除非其他法律已经被违反。

这将表明，这栋房子的买方将依靠卖方更好的本性来兑现他们交出房子的承诺——考虑到所涉金额，这似乎是一场相当大的赌博。

至少就比普尔的 NFT 而言，在拍卖完成后，买方知道他会得到什么，他们之间的聪明的合同将资金送往一个方向，NFT 送往另一个方向。预期不需要任何第三方强制执行或参与后续活动来完成或交易销售。

## 信任的问题？

同样的问题也可能适用于 NFT 销售中包含的其他现实世界商品的销售。除非货物转让以某种方式写入在 NFT 拍卖会上执行的智能合同，否则这是买卖双方之间的善意问题，以确保双方都能从买卖中得到预期。

莱昂国王将不得不兑现他们的承诺，为他们的 NFT 的购买者提供前排音乐会门票。大概买家足够信任乐队(作为一个超级粉丝),并相信乐队会做他们说过的事情。他们也在赌，只要买家拥有他们的 NFT，乐队就会继续在一起，继续表演，并在未来提供合适的门票。

当然，如果乐队没有履行对购买了他们的 NFT 的粉丝的承诺，他们在粉丝中的声誉会受到损害。仅此一点就可能是鼓励他们兑现承诺的动力。

# 使用 NFTs 销售其他真实世界的商品

当我研究使用 NFT 销售房子背后的逻辑时，它使我得出这样的结论:在 OpenSea 上完成 NFT 销售后执行的基本智能合同可能不足以自动处理现实世界商品的销售。

当然，非功能性食物的主流市场最终可能会适应和发展，以迎合这类东西的销售。例如，任何在易趣上购买或出售汽车的人都可能经历过这样的情况:虽然刊登广告和商定价格的过程与出售低值物品(如一件衣服)非常相似，但接受付款和完成物品交付的过程却完全不同。

与 NFTs 相关的真实世界商品的销售也是如此。如果市场能够为卖方提供一种在销售前修改智能合同的方法，以启用额外的现成条款和条件，那么它可能在智能合同中是可行的。

然而，就目前的情况来看，似乎存在固有的限制，这意味着销售不是在区块链上端到端进行的。仍然依赖于各方之间的信任，以及传统的法律和金融系统和实体的参与来执行和处理交易的某些要素。

# 价格溢价？

一些最著名的非功能性测试价格的一个重要因素是新颖性因素——该技术仍然相对较新。这些投资者中的许多人可能会提前进入并购买 NFT，希望之前在加密货币投资中看到的模式会重复——随着需求增加和供应稳定，早期采用者从价格上涨中受益最多。

许多铸造自己的 NFT 的人给它们定价很高，希望利用公众利益。这一点在千橡市的房子的 NFT 案例中显而易见——根据 CNN 文章的报道，该房产价值约 81 万美元，现在以约 140 万美元的价格出售(尽管附有数字艺术品)——这是一个相当高的价格。

非功能性交易的使用是否能让卖家像艺术品一样对真实世界的商品收取额外费用？艺术品的定价是主观的，支付的溢价反映了许多因素，包括艺术家作品的地位、稀缺性和可取性。

就房地产而言，溢价通常已经包含在价格中。这是基于物业本身，它的位置，设施，周围的设施，可出租性等等。即使房子本身也配有一件数字艺术品，仅仅为了通过 NFT 执行交易的特权，就指望买家支付巨额溢价，这种想法可行吗？

如果谢恩·杜尔杰洛夫成功卖出(或不卖出)NFT，这一考验大概就会到来。

# 可替代性的概念

最后要考虑的一个因素是可替代性的概念。请记住，非功能性标签允许创建者在所购买的商品中标明不可替代性和独特性。

一幅数字图像、一个音频文件、一段视频、一个迷因或一段代码都可以被复制、修改和重复使用——但 NFT 仍然是独一无二的，允许卖家注册所售商品原件的所有权。有人可能会质疑将这种技术应用于实体房屋的价值。

另一个人可以使用蓝图和相同的材料很容易地建造一个一模一样的*复制品*。它可以用同样的方式装饰，它的花园景观看起来一模一样，等等。但是它永远不会和 T4 完全一样，因为它不会占用相同的土地，也不会从相同的景色中受益，即使它建在原址的隔壁。

从这个角度来看，NFT 对于单个建筑来说有些多余——这些建筑在某种程度上总是独一无二的。房子已经不可替代了，不是吗？

NFT 变得与物理、现实世界的项目更加相关，这些项目可以被那些希望将副本冒充为原始文章的人复制和伪造。通过这种方式，NFT 可以成为实物真实性证书的等价物，但被锁定在区块链中，作为追踪所有权和证明其出处的手段。这可能适用于实物，如稀有的葡萄酒瓶、绘画、珠宝或类似物品，其价值不能单独作为 NFT 存在。

# 关键外卖

很明显，使用非功能性交易销售真实世界的商品存在一些实际的挑战，但这些挑战有望及时克服。从理论上讲，我们可以看到任何东西的市场，从稀有和定制的汽车、雕塑、艺术装置、美酒、古董收藏品或任何数量的其他实物商品，所有这些都通过 NFTs 在区块链上出售和登录。

在这一点上，我的感觉是，通过区块链使用 NFTs 处理此类销售的优点，并不超过在尝试完成售后交易时可能遇到的潜在困难。这在很大程度上是由于现有法律体系眼中的监管格局以及涉及区块链和加密货币的交易的地位。

目前，这两个世界看起来是分开运行的，没有足够的测试案例证明它们如何成功地交互和互操作。克里斯蒂拍卖行出售毕普的艺术品可能标志着这一步的开始。另一步可能是出售 Shane Dulgeroff 在 OpenSea 的房子(如果它最终在那里出售)，前提是交易成功且合法。

这种工作的关键似乎是允许卖方以一种可访问的方式创建和修改与他们的非功能性交易相关的智能合同，这种方式在法律上也是可执行的。目前，这需要了解编程语言，如[Solidity](https://docs.soliditylang.org/en/v0.8.3/)——以太坊区块链上的一种类似 javascript 的语言，可用于在智能合同中设置合同条款。

随着技术的成熟和平台的发展，区块链用户无需编写代码，就可以快速、轻松地设计和实现绑定智能合约中的附加条款，理论上，更复杂的销售可以只在区块链执行和处理。这些销售可能最终不得不将明确和结构化的移交合并到现实世界的法律和金融系统中。

这将是一个要点，使用 NFT 也许会成为一种更合适的现实世界商品所有权转移的工具。就目前而言，看到事情的发展是令人着迷的——我将饶有兴趣地观看谢恩·杜尔杰洛夫的 NFT 拍卖会！
