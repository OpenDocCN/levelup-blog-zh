你可能一直在想一个你可以从事的令人兴奋和有趣的兼职项目。虽然你想建造一些东西，但你并不确定要建造什么。这样的结果就是，你觉得无聊。

为什么不建立一些有趣的东西，而是学习一些新的东西呢？

提高你技能的最好方法是实际建造一些东西。因此，让我们提高你的技能，同时建立一个有趣的和具有挑战性的事情，这是在这一块列出的。

# 1.自动驾驶乐高汽车

![](img/0a7c1ca6d8af2c589ac3fe4ee1dbb03a.png)

照片由 [Praveesh Palakeel](https://unsplash.com/@praveesh?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

几年前，我做过一个类似的项目，就像这个。目标是创造一辆乐高汽车，它可以在一张纸上写下某个单词。

这个项目旨在制造一辆使用乐高或乐高技术的汽车，可以在不撞到任何物体的情况下行驶。在这个项目中，你需要一个树莓派(或 Arduino)，一些乐高，和一些超声波传感器，这样你就可以避开任何障碍。

软件和硬件的结合是这个项目如此有趣的原因。如果你从未接触过 Raspberry Pi(或 Arduino ),我会向你强烈推荐这样一个项目。

您将学到的内容:

*   Arduino(或 Raspberry Pi)的基本用法。
*   读取传感器。
*   软件和硬件之间的相互作用。

# 2.目录应用程序

![](img/cb0427750274ff3f881e397ab4b70306.png)

[Hilthart Pedersen](https://unsplash.com/@h3p?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

到目前为止，创建目录已经做了很多次。然而，让这个项目如此伟大的是，你可以使用 Flutter。

Flutter 是谷歌最新的 UI 工具包，允许你只用一个代码库创建一个原生移动应用。它使用 Dart 编程语言。

这些天来，Flutter 很热，这绝对引起了我的注意！

您将学到的内容:

*   摆动
*   Dart 编程语言
*   应用程序开发

# 3.九宫格游戏

![](img/9e63cab6d0a631c730ba8a75e7c4af9a.png)

[来源](https://nl.m.wikipedia.org/wiki/Bestand:Sudoku00.png)

数独拼图不需要介绍。这些有趣的谜题也是获得构建算法经验的好方法。目标是创建一个生成有效数独谜题的算法。

如果这对你来说太简单，你也可以建立一个算法来解决这些数独难题。一种方法是创建一个回溯算法。

您将学到的内容:

*   学习和实现数独算法
*   你可以为你的数独游戏增加几个难度

# 4.汽车的分类

![](img/df8af3e6924c475b29747898e6300f5a.png)

在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上由 [Carlo D'Agnolo](https://unsplash.com/@carlodagnolo?utm_source=medium&utm_medium=referral) 拍摄的照片

机器学习的一部分是聚类和分类。这个项目的目标是根据汽车的数据将汽车分为安全或不安全。如果你是机器学习领域的新手，那么这个项目是了解机器学习的一个很好的方式。

你需要做的就是开始使用[数据集](https://archive.ics.uci.edu/ml/datasets/Car+Evaluation)。

您将学到的内容:

*   你将掌握机器学习
*   分析数据

# 5.2D 游戏

![](img/cb7752684db458f1486f6c879b2cd41a.png)

[来源](https://www.flickr.com/photos/bagogames/14073389239)

如果你一直想涉足游戏开发，那么创建一个 2d 游戏绝对适合你。当这将是你第一次体验游戏开发时，你会学到很多新东西。总的来说，这将对你的编程技能有很大的帮助。

你不需要为一个游戏想出一个全新的主意，一个很好的开始项目就是重建 Flappy Bird——无论是在手机上还是在桌面上。

如果你有自己想要制作的游戏的想法，那是非常好的。请记住从小处着手。

你将学到的东西:

*   实体的移动
*   侧向滚动
*   冲突检出

# 6.渐进式网络应用

![](img/c6b56a4ef1b8899e2fc9e6659e95ad2c.png)

[卡洛斯·穆扎](https://unsplash.com/@kmuza?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

渐进式网络应用程序很热门。由于控制渐进式 web 应用程序的范围非常容易，所以它非常适合作为一个辅助项目。

当你计划构建一个渐进式 web 应用程序时，将它与周围的顶级 JavaScript 框架之一结合起来是一个完美的时机——Angular、React、Vue。

对这些框架做一些研究，然后选择一个你最喜欢的。这样，在构建渐进式 web 应用程序时，您将获得最佳的学习体验。

您将学到的内容:

*   让 web 应用程序进步所需的一切
*   您选择的 JavaScript 框架
*   Web 开发基础，如 HTML 和 CSS

# 7.大数据

![](img/72e179039d5c3a206442e57d1cc00456.png)

乔希·索伦森在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

当寻找一个具有挑战性和趣味性的大数据项目时，你绝对应该尝试一下芝加哥犯罪数据集。这是一个多分类问题，对于更有经验的数据科学家来说非常好。

这个问题很简单，但是由于这个数据集有超过 600 万个观察值，数据管理将会困难得多。

剩下给你的就是下载[数据集](https://data.cityofchicago.org/Public-Safety/Crimes-2001-to-present/ijzp-q8t2)并开始！

您将学到的内容:

*   大数据方面，如数据建模
*   处理大型数据集
