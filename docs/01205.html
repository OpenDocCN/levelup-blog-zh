背景故事:我的web应用程序用户界面遇到了挑战。我正在渲染一个数组中的卡片，但是这些卡片的行为并不是我想要的。

让它工作的小TAD算法

不用担心名字，LOL。

所以我决定使用已知的，即我需要来自每个父数组的3个数组。1和positive infinity之间任意数的n % 3为0、1或2 (其中 n 为数组长度，3为我需要的数组个数)。这是算法的基础。

去执行它；

我知道1 % 3 = 1.外推，所有的模1都需要去arr1。2 % 3 = 2，所以所有模2需要去arr2，所有模0需要去arr3。
