# 面向程序员的 9 大 YouTube 频道

## 从互联网上所有的优质资源中选优

![](img/0bab5fa2d5c5750a75295e7932093c8d.png)

詹姆斯·哈里逊在 [Unsplash](https://unsplash.com/s/photos/code?utm_source=unsplash&utm_medium=referral&utm_content=creditCopyText) 上拍摄的照片

书籍对于开始您的编程之旅非常有帮助。事实上，上个月我分享了一篇关于[最佳书籍](https://betterprogramming.pub/as-a-software-engineer-here-are-7-books-you-should-always-have-at-your-desk-e0262bccf09a)的文章，以培养成为一名优秀程序员的必要技能。但是 YouTube 也是一个定期更新和提高自己技能的极好资源。

今天有几个频道在平衡传统语言和现代问题方面做得很好。然而，并非所有这些节目频道都是平等的。有些对初学者来说很棒，充满了有用的技巧和诀窍，有些对经验丰富的开发人员来说很棒，而其他的可能完全是垃圾。

以下是我经常访问并从中获益良多的 10 个最佳频道，你也可以。

# [编码序列](https://www.youtube.com/c/TheCodingTrain)

## 订户:140 万

编码列车是一个有趣的互动频道，由计算机程序员丹尼尔·希夫曼运营。该频道有关于构建游戏、利用机器学习、模拟、JavaScript、框架等的创造性编码教程。他甚至有一个哨子来证明编码列车将在他的视频开始时离开。

拥有数学艺术学位和哲学硕士学位，希夫曼的交叉学习可以在他的算法艺术、生成诗歌、制作不和谐机器人或小屋编码的教程中看到。

# [树屋](https://www.youtube.com/user/gotreehouse)

## 订户:34.8 万

这是[团队树屋](https://teamtreehouse.com/?utm_source=google&utm_medium=cpc&utm_campaign=Brand&utm_term=teamtreehouse&gclid=Cj0KCQjwgMqSBhDCARIsAIIVN1XvR8Vu-HhnfVqP6NSO1ZY3KFk2eVnWVReCH_WK6VCa1SHOTLswjHwaAhQXEALw_wcB)的官方 YouTube 主页。Team Treehouse 是一个学习资源，旨在提供负担得起的技术教育。Teamtreehouse 提供许多主题的课程，包括 Python 和 C#编码。

各种各样的主持人以轻松易懂的方式播放视频。该频道的内容还包括树屋秀，这是一个网页开发和编程新闻以及有用的提示和采访的综述。

# [**dev tips**](https://www.youtube.com/user/DevTipsForDesigners)

## 订户:35.9 万

对于那些已经了解基础知识并希望快速提高的人来说，DevTips 是一个增强技能的绝佳渠道。在这里你可以了解到 web 开发，编码 CSS，HTML5 等等。频道创建者 Travis Neilson 教授大部分教程，并偶尔邀请客座技术专家。

如果你对软件和网络开发感兴趣，你会在这个频道找到很多有用的信息。DevTips 还定期发布视频，为打算在科技领域开始职业生涯的人提供建议。

# [CS 道场](https://www.youtube.com/channel/UCxX9wt5FWQUAAz4UrysqK9A)

## 订户:180 万

CS Dojo 由 YK·杉经营，他曾在谷歌担任软件开发员。现在，他通过白板上的循序渐进的教程和简单易懂的演示，与 180 多万订户分享他的专业知识。

该频道有九个播放列表，可以很容易地划分视频，这样想学习某项技能的观众可以观看他的所有视频，就像看完整个课程一样。也有很多适合初学者的视频。

# 德里克·巴纳斯

## 订户:117 万

Derek Banas 带来了针对初学者和高级开发人员的关于不同编程语言的大量 web 教程。这个网络发展 YouTube 频道已经上传了 1000 多个视频，每种语言都有单独的播放列表，并且每周定期更新两次。

任何人都会对频道上的互动视频横幅、视频主题和有趣内容着迷。他的视频是关于 PHP、Java 和 MySQL 等编程主题的详细课程。他因在 30 分钟播放列表中学习 JavaScript 而闻名。

# [Learncode.academy](https://www.youtube.com/user/learncodeacademy)

## 订户:75.4 万

Will Stern 清晰而简单的教程专注于与 web 开发相关的编程。在 Learncode.academy，您将会以 Stern 高效而全面的风格，了解 JavaScript 和 jQuery 等主题。该频道的预告片是斯特恩比较受欢迎的视频之一，并提供网络开发职业建议。这个有用的视频概述了你成为一名 web 开发人员的道路，以及你应该学习的顺序

# 新波斯顿

## 订户:260 万

TheNewBoston 由 Bucky Roberts 于 2008 年创建，拥有超过 200 万用户，是 YouTube 上最受关注的学习编程语言的网络开发频道。该频道是许多初学者免费网络学习的热门来源。它有大量的后端和前端语言教程，旨在为新手到高级学习者。

对于任何想成为软件开发人员的人来说，NewBoston 可以说是唯一最好的资源。Bucky Roberts 采用简单有趣的教学风格授课。他的编码教程对任何年龄的人来说都很容易掌握

# [用 Mosh 编程](https://www.youtube.com/user/programmingwithmosh)

## 订户:240 万

Mosh 不仅为您提供了大量关于流行 web 语言的教程，还提供了大量关于新兴 web 技术的教程，这些技术正在开发领域迅速获得一席之地。首先，该频道提供长视频(长达数小时),详细讨论语言的每个主题和话题。

Mosh Hamedani 的目标是培训程序员和软件工程师，并考虑招聘事宜。他训练个人破解面试问题，并谈论市场、开发人员的工资、流行语言等。，在频道上还有他的[博客](https://programmingwithmosh.com/)。

# [编程知识](https://www.youtube.com/user/ProgrammingKnowledge)

## 订户:150 万

在这里你会发现各种针对新程序员的视频。这里的教程从面向初学者的 Android 教程到 Java、Python 和 C++教程。参考资料中有一个 C 编程教程，它被认为是该语言中最全面的课程，你可以在任何地方找到。该频道(以及编程知识博客)归 Yogesh Patel 所有。

面向初学者的大量视频(超过 1，000 个)使得这成为任何想进入编码领域的人应该收藏的第一个渠道。你一定会找到你需要的一切。

你什么时候开始学习编程并不重要——当你还是一个孩子，还是一个成熟的成年人和专业人士。学习编码可以拓宽你的职业机会，让你的大脑保持敏锐。最棒的是，多亏了 YouTube 上的这些教育频道，你可以足不出户就能做到这一切
