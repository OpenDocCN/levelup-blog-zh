# 如何使用 Vercel 部署您的网站

> 原文：<https://levelup.gitconnected.com/how-to-deploy-your-website-using-vercel-831c4fac2d41>

![](img/d0b16f3b9422237d43f5e2d298362876.png)

Vercel 将最佳开发人员体验与对最终用户性能的专注结合起来。我们的平台使前端团队能够做到最好。

# 1.首先，如果您没有帐户，请创建一个。但是如果你这样做了，那么转到第 4 步:

![](img/a82c74b40343821e7e5f1c488ed333f1.png)

# 2.您将被重定向到仪表板。然后，点击下拉菜单并选择“添加 GitHub 组织或帐户”:

![](img/69f1273d824ae99d1ea0d99080b19df3.png)

# 3.然后，你会看到一个弹出的窗口。选择“安装”选项。不过，这不会在您的设备上安装任何应用程序:

![](img/fc95ef913fac2f9776c4e66eea037cd7.png)

如果它成功了，那么你会得到一个这样的窗口:

![](img/c99360944f6d44628955775c76e95f09.png)

# 4.现在，回到你的仪表板，点击“新项目”:

![](img/87eaacfd045b0f3029a2e432acc2f858.png)

# 5.现在，在下拉列表中选择您的 GitHub 用户名:

![](img/cbf542bf5fef1b1b7e9fd9956d009346.png)

# 6.现在选择您想要托管的 GitHub 存储库:

![](img/110cf7d2948a3fe02f7709013da14f01.png)

# 7.现在，选择您的帐户

![](img/1b44510e61267b0ad07d27f816a46226.png)

# 8.然后，点击下一步继续:

![](img/848558a2d3891812a0ac2987828e9aba.png)

# 9.接下来，输入详细信息并单击部署:

![](img/4b9c8cabfc06fef8afc9f43666b869e7.png)

# 10.然后，需要一些时间来处理:

![](img/5459408907ccd98cf6886ed37fb1c9dd.png)

# 11.过一会儿，您将能够看到一个祝贺屏幕:

![](img/714b5a2c1d39c7f83add703e7444caf8.png)

# 12.如果您想访问您的网站，请单击“访问”,它会将您带到以下网站:

![](img/a99b0e431b32c30a1a1593ebbc02a387.png)![](img/9e8b45fb5549f9aa9471c6ce195a9260.png)
