# 一个 Web 3.0 财团筹集了 4600 万美元，试图购买美国宪法

> 原文：<https://levelup.gitconnected.com/a-web-3-0-consortium-raised-46-million-and-attempted-to-buy-the-us-constitution-92b63960fb60>

## 宪法道是未来的征兆——即使它失败了

![](img/7d606b2c3ccd4140eeded008e6a9c218.png)

图片来自 [Shutterstock](https://www.shutterstock.com/image-photo/united-states-constitution-rolled-scroll-on-1103285504)

昨天，当美国宪法仅存的 13 份原版中的一份被拍卖时，一个分散的个人团体几乎成功地买到了它。苏富比拍卖行的以超过 4300 万美元的价格将这件罕见的艺术品[卖给了一位匿名竞标者。](https://www.sothebys.com/en/buy/auction/2021/the-constitution-of-the-united-states-sold-to-benefit-the-dorothy-tapper-goldman-foundation/the-official-edition-of-the-united-states)

它的新主人可能会对他们的购买感到高兴，如果他们被迫支付超过两倍于 2000 万美元的预售估价的话。价格是由*宪法道*推高的——在拍卖前几天成立的分权自治组织(简称道)，其明确目标是购买历史文献。

获胜的买家也应该感谢他们的祝福。虽然刀已经从成千上万的捐助者那里筹集了大约 4600 万美元，但在拍卖的后期被裁定没有足够的资金来储存、保存和保险这件文物。这件拍卖品的出价刚刚超过 4300 万美元。

虽然宪法道未能实现其最初的目标，即确保宪法文件的所有权并将其“为人民”而展示，但它实际上取得了很大成就。

从这一事件中得到的教训告诉了我们很多关于未来可能的筹资方式、运动的构建和组织以及生意的开展。

# 宪法道

这场运动是由亚特兰大的两位金融专业人士——奥斯汀·凯恩和格雷厄姆·诺瓦克发起的。虽然这个倡议是他们的主意，但这个集体是大约 18，000+个人的财产，他们做出了贡献，有效地购买了仅存在于以太坊区块链上的道的一部分。

来源:[奥斯汀·凯恩在推特上](https://twitter.com/j_austincain/status/1458850733495918596?s=20)

作为一个实体，对 DAO 的治理和监督由 DAO 内置的逻辑负责，并在存储在区块链上的计算机代码中自动执行。没有臃肿的董事会和执行董事做出(潜在的)自私的决定，或试图取悦或安抚股东。没有艰难的公司注册过程，也没有任何股票上市或公司筹资来支付成立成本和日常开支。

贡献者只需选择他们想要的捐赠规模(ETH-ether eum 代币)并获得[治理代币](https://news.artnet.com/market/sothebys-constitution-cryptocurrency-sale-2037156)作为他们捐赠的回报。这些代币是以大约一百万美元的比率发放的。治理令牌为决定 DAO 的命运和行为带来了投票权。

“道”被设定为一种工具来完成它的初衷——以最小的代价筹集资金来购买宪法。仅在几天之内，它就从渴望支持这项事业的参与者那里筹集了 4000 多万美元。正如我们后来看到的，他们也差一点就成功了。

在这一点上，他们最终失去了购买宪法文件的机会在很大程度上是无关紧要的。《道宪法》的迅速建立以及该项目理所应当地得到的曝光，将产生几个持久的影响。

我是这样看的。

## 它展示了 Web 3.0 支持集体行动的灵活性

如果你的任务是发起一场集体运动，并筹集 4000 多万美元来支持它，在正常情况下，这可能需要几天以上的时间。即使在智能手机和社交媒体无处不在的时代，这也不是那么容易做到的。

当自然灾害发生时，慈善机构会立即行动起来，呼吁捐款。但这些慈善机构是预先存在的注册公司，拥有银行账户、大批筹款人和营销人员、知名捐赠者名单，以及筹集、处理和支付资金所需的所有后台基础设施。

宪法道在 Twitter 上宣传了一点，通过 Discord Chat 推出，并利用名为 [Juicebox](https://juicebox.money/#/) 的早期 DAO 筹款平台建立。几天之内，它就筹集了超过 11，000 ETH，放在一个加密钱包里，随时可以使用。苏富比同意，加密货币的资金证明足以让道的代表在拍卖中出价。

这整个过程的影响是巨大的——不仅仅是对于那些想为他们脑海中可能出现的任何异想天开或放纵的计划或突发奇想筹集资金的秘密组织。

它表明，被认为有价值的项目可以迅速动员起来，几乎没有摩擦，几乎没有前期成本就能筹集到资金。捐赠者和支持者能够安全地为该事业捐款。然后，DAO 可以通过对等加密传输或在转换为传统货币后轻松使用它。

宪法道可能展示了一个令人信服的替代商业模式的慈善筹款，也为个人众包运动。这些公司在历史上一直局限于使用传统渠道进行公司注册、筹资和金融基础设施建设。

## 它向新人们介绍了加密货币和 Web 3.0

道宪法的许多私人捐助者是因为宪法文件本身所代表的意义而积极参与的。区块链计算的分散化理想与美国创始人构想的美国理想之间存在重叠——自主、自由、不受约束和压制等原则。

一些捐赠人在捐赠信息中分享了这样的理想主义信念:

来源: [NLW 在推特上](https://twitter.com/nlw/status/1460301879896907780?s=20)

这项倡议值得注意的是，它似乎不仅仅是一群密码爱好者试图颠覆或颠覆现有体制。许多人的动机是希望名义上拥有一份具有历史意义的开创性文件。

因此，他们被 DAO 的动机和它要做的事情所激励，而不是出于参与一个古怪项目的愿望。这可能解释了为什么在许多情况下，这似乎是他们首次涉足加密货币领域。

[对区块链交易的分析](https://dune.xyz/ilemi/ConstitutionDAO-Funding-Tracker)表明，在该项目的所有捐助者中，13%是首次捐助者，44%的捐助者钱包地址的 ETH 交易少于 40 次。

教训？许多参与者都是相对加密和区块链新手。这次经历可能让他们感到失望，因为刀没有赢得拍卖，但它证明了许多人愿意接受新技术，并克服最小的技术障碍，如果他们相信这一事业或被机会所迫的话。

# 接下来会发生什么？

由于未能赢得拍卖，这笔钱目前仍留在 DAO 的加密钱包中。成员们大概会投票决定下一步该做什么，是竞拍其他有历史意义的物品还是把钱退还给捐赠者。

能够考虑这种选择，也证明了"道"的功能和区块链的技术能力，在这种情况下，交易是透明的，可以很容易地追踪。

接下来发生的事情与道宪法所证明的事情相比似乎有些微不足道:

*   Web 3.0 技术使得大规模运动的快速、无摩擦动员成为可能。
*   在筹集和处理大笔资金时，区块链和加密货币可以用来绕过与传统金融系统相关的障碍和束缚。
*   像成立于 1744 年的苏富比这样的百年机构能够轻松地与道交易，并愿意承认加密货币等同于传统货币。
*   如果有令人信服的理由，公众不会考虑与 Dao 打交道，也不会考虑使用加密货币作为交易的障碍。
