## 非常适合在你自己的项目中使用

您可以围绕公共 API 中可用的数据构建许多酷炫的应用程序。有很多非常有趣的 API 可供使用。

API 经常被用来提供有价值的信息，作为其他应用程序的输入。使用外部 API 有一些好处。例如，你不需要管理数据。API 还使得创建应用程序中还没有的新功能成为可能。除此之外，如果您愿意，还可以对数据进行不同的表示。

这个列表中的所有 API 都是完全免费的，所以您应该可以毫无困难地开始使用。

# 1.天气

[OpenWeatherMap 天气 API](https://openweathermap.org/api) 是存放所有天气相关数据的完美地方。这个 API 的免费版本提供了当前天气以及 5 天/3 小时的预报数据。

它有每分钟 60 个请求的限制，这应该足够用它做一些好的事情了。你可以使用[地理定位 API](https://developer.mozilla.org/en-US/docs/Web/API/Geolocation_API) 构建一个基于你的位置获取天气的应用程序。

# 2.CoinAPI

[CoinAPI](https://docs.coinapi.io/) 是一个平台，将所有加密货币交易所整合在一个 API 下。有两种类型的 API 可用——REST 和 WebSocket API。这个 API 的免费版本每天提供多达 100 个免费请求。

您可以构建一个应用程序，一旦达到某个价格就购买加密资产。或者你可以保持简单，制作一个每十五分钟更新一次的跑马灯。这样你就不会超过每天的最大请求量。

# 3.美国国家航空航天局

NASA 有一个很棒的 API，可以让你利用他们的数据和图像。有不同的 API 包含非常有趣的数据。其中一个 API 给你关于火星温度的数据。其他一些 API 向您展示了 DSCOVR 的地球多色成像相机(EPIC)仪器收集的日常图像。这个 API 中有非常有趣的东西！

请注意，每小时有 1000 个请求的限制。

# 4.电影

《纽约时报》有一个[电影评论 API](https://developer.nytimes.com/docs/movie-reviews-api/1/overview) ，可以让你深入了解它所做的所有电影评论。有了这个 API，你可以访问从今天到 1924 年的超过 22，000 篇纽约时报影评。

有了这些数据，你可以根据某些关键词和评级建立一个观察列表。知道你终于知道看什么电影了。

# 5.未开发的

对于每一个喝啤酒的开发者来说，都有一个[未开发的 API](https://untappd.com/api/docs) ，可以让你获取所有你想要的关于啤酒的数据。从啤酒厂和场地信息到您最喜爱的啤酒的更多详细信息。

这个 API 可以很好地获取您最喜欢的啤酒列表。它还可以与你自己的用户信息进行交互，因此你可以创建尚未在 Untappd 应用程序中提供的新功能。

# 6.Skyscanner

Skyscanner 现在不需要介绍了。但是你知道有一个 [API](https://skyscanner.github.io/slate/) 可用吗？您可以用这个 API 中的数据做很多事情。例如，您可以跟踪某段时间内某次航班的价格。当你的机票价格足够低时，你可以给自己发一个通知。

但是，在开始使用 API 之前，您需要注册以获得 API 密钥。

# 7.发送网格

Sendgrid 为您提供了通过 [API](https://sendgrid.com/docs/API_Reference/api_getting_started.html) 发送电子邮件的能力。前 30 天你可以发送 40，000 封免费电子邮件。之后，你可以永远每天发送 100 封免费邮件。不错吧？

您可以在任何需要偶尔发送电子邮件的低流量应用程序中使用这个 API。如果你不喜欢 Sendgrid，你可以使用 [Mailgun Email API](https://www.mailgun.com/email-api/) 作为替代。

# 8.IEX

据其网站介绍，IEX API 是一个免费的基于网络的 API，为移动、桌面和交易应用程序提供 IEX 报价和交易数据。API 的文档可以在[这里](https://iextrading.com/developers/docs/)找到。

您可以使用这个 API 的数据构建的一个东西是您最喜欢的股票的报价器。你甚至可以建立一个机制，一旦股票达到一定的价格，就通知你。

# 9.口袋妖怪

对于所有的口袋妖怪粉丝来说，有一个奇妙的 API 可以用来获得你需要的关于口袋妖怪的所有数据。这个 API 非常有趣，可以用在你自己的项目中。

由于 API 包含了所有口袋妖怪的精灵，动作和能力，你可以构建一些口袋妖怪迷你游戏，在那里你可以与随机的口袋妖怪战斗。

# 10.运动

体育数据可以做很多有趣的事情。来自[体育开放数据 API](http://sportsopendata.net/) 的数据非常适合用于对体育数据进行各种分析。这个 API 包含不同足球联盟的数据，这些联盟的赛季排名，最佳得分者，甚至关于裁判的数据。

你可以做的一件事是制作一个仪表板，上面有你最喜欢的球队的统计数据！
