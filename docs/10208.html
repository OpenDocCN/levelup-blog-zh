如果你是开发者，你必须订阅的9个YouTube频道

原文：https://levelup.gitconnected.com/9-youtube-channels-that-you-must-subscribe-to-if-you-are-a-developer-d7ac881c9798?source=collection_archive---------0-----------------------#2021-11-07

强烈推荐给软件开发人员的YouTube频道，帮助他们在软件开发职业生涯中脱颖而出。

截至2020年，YouTube在全球拥有超过23亿用户。这些数字每年都在增加。根据Alexa 的统计，它在全球互联网参与度中排名第二。由于该平台是免费的，它邀请世界各地的创作者在旅游、新闻、博客、金融、音乐、教育、健身和其他不同的领域创建内容。

平台上最受欢迎的领域之一是编程或技术。为什么？编程或编码是任何人都可以在任何地点、任何年龄学习并获得工作的少数工作之一。我见过人们从医疗专业人员、律师、前军事专业人员等转行。成功的软件开发者。

根据美国劳工统计局的数据，在2016年至2026年间，软件开发人员的数量预计将以24%的速度增长，远快于该国其他任何职业。这只是一个估计，随着每天新的未知技术的出现，这个数字还会增加。

许多有抱负的开发人员和程序员使用他们最好的编码YouTube频道来学习和提高他们的编程能力。这些渠道不仅提供编码技巧，还提供基本的求职建议，如协助撰写漂亮的简历和准备面试。

YouTube上有数百个这样的节目频道，你可以从中学习和提升自己的技能。然而，我分享了一些最好的YouTube频道，排名不分先后，我相信每个软件开发人员都必须订阅。

卡勒·哈尔登

频道链接:https://www.youtube.com/c/KalleHallden
订阅者: 578k

卡勒·哈尔登是一名软件开发人员，他喜欢探索各种技术，并制作概念验证视频。他已经使用Python automation制作了大量视频，他的所有视频都在多个平台上受到欢迎和分享。他目前在瑞典的家中工作。他的视频质量极高，会让顶级YouTubers嫉妒。

卡勒最好的一点是，他是一个自学成才的开发人员，不仅做了大量的项目，还解释了他如何以及做了什么来完成这些项目。

霍马理工大学

频道链接:https://www.youtube.com/c/JomaOppa
订阅者: 1.47M

霍马是一名熟练的开发人员，他以技术、软件工程和数据科学方面的娱乐和教育视频而闻名。他的频道也有一个名为创业系列的7集优秀系列，以幽默的方式处理创业材料。

该频道还分享了霍马在不同公司的采访经历，包括FAANG公司。FAANG是美国表现最好的五家科技公司的首字母缩写，即脸书、亚马逊、苹果、网飞和谷歌。另一个流行的缩写是FAAMG，其中M代表微软。霍马是第一批推广这些首字母缩写词的YouTubers用户之一。

自由代码营

频道链接:https://www.youtube.com/c/Freecodecamp
订阅者: 4.5M

昆西·拉森(Quincy Larson)创办了freeCodeCamp，这是一个非营利组织，专注于让每个人都可以免费使用web开发。该非政府组织包括一个互动学习网络平台、一个在线社区论坛、聊天室、在线出版物、地方组织和一个YouTube频道。

如果你是初学者，你可能已经听说过他们令人难以置信的YouTube频道。这个频道没有任何广告。最好的部分是它是开源的，因此任何有深度的开发者都可以贡献给这个频道的播放列表。他们有各种技术和编程语言的全长课程，如 JavaScript、Java、Python、全伦理黑客、数据科学，甚至机器学习。

越来越多的课程进入了这个可以免费使用的平台。每当有人问我编程课程时，我都会毫不犹豫地向他们推荐这个频道。

爱德华卡。

频道链接:
订阅用户: 2.99M

T18】edu reka！ 是一个讲师指导的平台，有一个YouTube频道，强烈推荐给不断想提升自己技能的职场人士。该频道有令人印象深刻的5000多个视频，并有关于 DevOps、人工智能、数据科学、数字营销、云计算、和认证课程的免费教程。该频道对那些正在学习新技术的人非常有益，无论他们是新手还是经验丰富的开发人员。

新波斯顿

频道链接:https://www.youtube.com/user/thenewboston
订阅用户: 2.58M

这个通道是OG，这是一切开始的地方。学习编程在YouTube上变得流行和免费，因为有人决定定期上传视频，以便世界各地的人们可以学习编码。那个人就是巴奇·罗伯茨。巴基早在2008年2月就创办了这个频道，甚至早于广受欢迎的科技频道 【马克斯·布朗利】 。

对我和其他人来说，从书本上学习编程极其困难，但Bucky制作的视频非常受欢迎，以至于他们激励了一代人建立自己的编程频道。频道有J ava、Python、C++、区块链、Angular、React、Docker、Django 等热门语言和框架的教程。有4000多个视频。

Bucky最近创建了一个开源的 thenewboston 区块链，目前正在向其上传视频。因此，如果你想了解区块链技术，那么你必须查看视频。

娜娜的科技世界

频道链接:https://www.youtube.com/c/TechWorldwithNana
订阅者: 348k

Nana Janashia可能是名单上最新的频道，她在2019年开设了这个频道，两年内她已经获得了34.5万名订户。这是因为她免费上传了关于DevOps技术趋势的教程。迄今为止，她已经为 Docker、Kubernetes、Jenkins、Ansible、Prometheus Monitoring、Terraform、YAML、等工具提供了教程。娜娜是一名码头工人船长， AWS集装箱英雄 & CNCF大使。

如果您是一名软件开发人员，那么您应该知道这些DevOps技术在IT市场非常热门。如果你查看Udemy或其他平台，他们的教程很贵。不过娜娜用完全动手的方式，把这些教程做得极其专业，简单易懂。如果你想在DevOps的事业上出类拔萃，那么这个渠道就是你的金矿。

旅行媒体

频道链接:https://www.youtube.com/c/TraversyMedia
订阅用户: 1.68M

另一个OG频道，由Brad Traversy在2009年创办。这个YouTube频道有网络开发人员需要知道的一切。该频道有关于编码技能和语言的教程，如HTML、CSS、Python、PHP和JavaScript，以及前端框架和库，如React、Angular和Vue。

该频道还有布拉德和其他嘉宾导师的各种视频，如“香草JavaScript搜索应用”、“开源入门& GitHub ”、“用Kotlin 构建简单的Android应用”、“Ace求职面试的7个技巧”和数百个这样有趣的教育教程。

威尔·关

频道链接:https://www.youtube.com/c/WillKwan
订阅者: 133k

2016届计算机科学毕业生Will Kwan创建了一个令人难以置信的频道，他在那里使用人工智能、数字游牧者、网络开发课程等创建了关于他的实验的视频。

我最近通过YouTube的推荐找到了这个频道，看了他所有的视频，我非常惊讶。他的视频确实启发了我学习更多关于机器学习和人工智能的知识。

Java大脑

频道链接:https://www.youtube.com/c/JavaBrainsChannel
订阅者: 531k

毫无疑问，YouTube上关于Java及其框架的最佳渠道之一。LinkedIn的员工工程师Koushik Kothagal是这个频道的创建者，他在这个频道上传关于Java开发人员在软件行业工作时所需的一切的视频。他对概念的解释是一流的，他经常被一些观众称为平台上最好的Java导师之一。

Koushik还上传了各种有趣的视频，如“用Spring Boot和Java构建冠状病毒跟踪器应用程序”、“OAuth到底是什么”、“编码面试准备的5个常见LeetCode错误”、“API gateway到底是什么？“还有很多其他的。频道还有 Angular、NodeJS、OAuth、等框架的视频。

YouTube一直是学习任何东西的最佳平台之一，每天都有成千上万的视频上传，学习编码从未如此容易。
