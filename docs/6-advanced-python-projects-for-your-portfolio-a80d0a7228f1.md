# 1.坐立不安剪刀游戏

这是一个基于 UI 的 python 项目。我们将使用 python 中的 turtle 模块来制作 GUI。其逻辑是当我们按下空格键时增加坐立不安的次数，当我们松开空格键时就会减慢。

这个项目会考验你的逻辑。很多时候，我们会用到海龟库提供的函数。

**项目来源:** [项目链接](https://thecleverprogrammer.com/2020/12/23/create-a-game-with-python/)

# 2.视频音频转换器

这是我个人最喜欢的。我通常会搜索任何可以从我的视频中提取音频的网站。现在，我不需要去任何地方；相反，我可以运行我的 python 代码来完成同样的任务。我还可以引入一些桌面自动化来以我想要的方式保存所有提取的文件。😅

我们将使用一个名为`pytube`的 python 库。这个包将视频的 URL 作为输入。作为输出，它可以下载视频文件或音频文件的视频。

**项目来源:** [项目链接](https://thecleverprogrammer.com/2020/12/08/video-to-audio-converter-with-python/)

# 3.井字游戏

这是一个漫长的项目，但我可以向你保证，一旦你完成这个项目，你会对 python 充满信心。这个项目将测试你的逻辑构建能力，并使你在使用类和不同的库方面更强。

我们将使用`Tkinter`来制作我们游戏的 GUI。

**项目来源:** [项目链接](https://thecleverprogrammer.com/2020/12/06/tic-tac-toe-gui-with-python/)

# 4.图像格式转换器

这也是现实生活中的场景。我们通常需要将图像转换成其他格式。有时，我们需要减小尺寸或改变维度。这个项目是关于转换你的 PNG 格式的图像到 JPG。

我们将使用`Tkinter`从该位置选择图像，最后，我们将把它转换成 JPG。

您可以扩展这个项目，添加从 URL 获取图像的功能，然后将其转换为其他不同的格式。此外，您可以添加邮件功能，将邮件发送给您自己。

**项目来源:** [项目链接](https://thecleverprogrammer.com/2020/12/02/image-converter-gui-with-python/)

# 5.人脸检测

这是一个简单但有效的项目。python 的几行代码可以帮助你检测你的脸，并在你的脸周围制作一个方框。

我们将使用 cv2 python 库。这个库有一些预先训练好的模型，可以用来解决图像问题。其中一个模型是帮助我们检测人脸的`face_detector`。

**项目来源:** [项目链接](https://thecleverprogrammer.com/2020/10/09/face-detection-with-python/)

# 6.OTP 验证系统

这是另一个有趣的用例。在这里，您将学习使用 python 中的 SMTP 模块。任务是使用 python 生成一个六位数的 OTP，然后使用 python 将生成的 OTP 发送到用户的邮箱。稍后，用户将需要从邮件中复制 OTP，并将其与我们使用 python 生成的六位数 OTP 进行匹配。

代码很简单，但是任务的流程很有趣。您可以稍后自动执行从邮件中读取 OTP 的手动任务——通过让 python 程序直接读取该 OTP 来使其更有趣。

**项目来源:** [项目链接](https://thecleverprogrammer.com/2021/04/14/otp-verification-using-python/)
