## 一场喜剧

![](img/49bf17b14b26c0e6cc3ef69aa1699d1b.png)

汉斯·雷尼尔斯在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 拍摄的照片

虽然全能的希腊神统治着古希腊的思想和心灵，确立了自己在希腊社会的中心地位，但编程语言在很大程度上不仅是每个开发人员生活的中心，也是当今每个人生活的中心。科技产品已经成为我们日常生活中必不可少的东西，随之而来的是编程和编程语言几乎无处不在。

当然，没有什么可以取代标志性的希腊神，然而，人们不禁想知道，如果每种编程语言都上升到如此神圣的地位，会发生什么。所以，我们在这里，调查每一种流行的编程语言最适合从哪个希腊神开始。

# 作为雅典娜的 Java

![](img/8e1ace813b7d9f934f87093fc2a3b538.png)

照片由[杰瑞米·贝赞格](https://unsplash.com/@unarchive?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

奥林匹亚的智慧和战争女神雅典娜受到每个人的尊敬。

承认 Java 的地位是教授们编写的教科书中最具特色的编程语言，人们不能否认 Java 本身看起来相当聪明。与此同时，看到 Java 如何征服了技术世界，成为迄今为止最流行的编程语言，人们不能质疑她的战争策略。

雅典娜和爪哇是天造地设的一对(或者更准确地说，是在奥林匹斯山山顶上的一对)。

# c 如厄里斯

![](img/0e8ebc636237cdf71e9a8d71de7e6f4c.png)

照片由[叶戈尔·迈兹尼克](https://unsplash.com/@vonshnauzer?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

没有一种编程语言能像 C 语言那样轻而易举地毁掉程序员的生活。C 语言混乱的语法规则到处都是分号*和*，任何程序员犯下的哪怕是一个简单的错误都会造成绝对的混乱。

因此，毫不奇怪，这种编程语言只可能以混乱女神厄里斯开始，她就像 C 一样，碰巧非常古老，即使对于希腊神来说也是如此。

# JavaScript 扮演酒神

![](img/728a48f5e9a19c310ed2f316106f6edf.png)

照片由[金莎·艾利斯](https://unsplash.com/@kymellis?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

狄俄尼索斯是声名狼藉的酒和快乐之神，本质上是奥林匹斯山的铁杆派对者。

一些人可能没有意识到他的重要性，但仔细看看酒在希腊社会中的作用以及狄俄尼索斯对农业的重要性，这对所有地方的所有社会都是必不可少的，揭示了狄俄尼索斯比看起来更重要。

虽然 JavaScript 经常被嘲笑，尤其是被那些宣誓效忠于 Java 的开发人员嘲笑，但最终，每个人都必须承认 JavaScript 的作用是至关重要的。毕竟，如果没有它的帮助，你甚至可能读不到这个网页上的文字。
