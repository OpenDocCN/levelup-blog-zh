程序员需要练习来增强他们的逻辑思维。这里有几个 python 程序员的实践例子。本文专门只关注简单的程序，供初学者尝试。先有一个问题，再有解决方案。我建议先自己解决，然后再研究解决方案。

# 1.文件中的行数:

***写一个 python 程序，求给定文件中的行数。***

> 线索:`newline`转义序列(\n)

用几行代码创建一个简单的文本文件，并将其保存在与源文件相同的目录中。使用 Python 中的文件处理机制读取文件内容。使用 help count 变量，计算文件末尾的行数。当有换行符转义序列(\n)时，假定文件结束

# 2.执行程序所用的时间:

***用 Python 写一个程序，计算执行代码所经过的时间。***

> 线索:时间模块

我们可以使用时间模块来查找执行 Python 程序所需的时间。因为时间模块不是内置的，所以您需要导入该模块。这个程序非常简单。您需要将程序开始和结束的时间存储在一个变量中。执行时间是开始和结束变量之间的差值。

# 3.在终端上彩色打印:

***写一个 Python 程序在终端上用颜色打印。***

> 线索:使用 colorama 模块

在终端上进行彩色打印的一个简单方法是使用 colorama 模块。使用这个模块，可以打印多种样式，如前景色、背景色。请注意程序中的 reset 语句；必须以程序中设定的先前颜色停止打印。

# 4.洗牌:

***用 Python 写个程序洗一副牌。***

> 线索:随机模块

使用 itertool 和 random 函数，我们洗牌并显示它们。在程序中，我们从一副牌中随机打印五张牌。

# 5.从字符串中删除特定字符:

***用 Python 写一个程序从字符串中删除特定字符。***

> 线索:使用切片运算符[开始:停止:步进]

要从字符串中删除字符，我们可以使用字符串切片操作。首先，使用输入函数，我们获取字符的索引，它必须删除字符。然后使用这个索引，我们在索引前后分割字符串。这些字符串的连接将产生所需的输出。

# 6.不带换行符打印:

***写一个 Python 程序在终端上打印，不用换行。***

> 线索:使用 end 语句

当程序中有多条打印语句时，每条打印语句将从新一行开始打印。为了避免这种情况，您可以在程序中使用 end 语句。

# 7.倒计时钟:

***用 Python 写一个程序给一个给定输入的倒计时钟。***

> 线索:时间模块

时间模块用于倒计时时钟。程序员可以用输入语句请求倒计时所需的时间，该时间被转换成秒。`os.system("CLS")`清除终端，这给人一种倒计时时钟的错觉。

# 8.Python 中的乘法表:

***用 Python 写一个显示乘法的程序。***

要显示乘法表，请使用 For 循环和范围。这个程序采用任何给定的整数，并显示其乘法表。

# 9.修剪空白:

用 Python 写一个程序来修剪字符串中的空格。

> 线索:使用 strip()

有许多方法可以从字符串中删除空白。然而，使用 strip 函数是删除空白的最直接的方法。

# 10.密码生成器:

***编写一个程序，生成一个包含字符串、数字和符号的简单密码。***

> 线索:随机模块

在这里，您可以使用 random 或 secrets 模块来获取随机数据。字符串模块获取小写字符串、大写字符串、数字和符号。将获得的字符连接起来形成密码。
