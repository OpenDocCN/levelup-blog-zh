# 维基百科搜索应用

维基百科有一个 API 来返回维基百科网站的内容。

这太棒了，因为我们可以用它马上创建一个维基百科应用程序。

我们的应用程序可以从用户那里获取关键词，并在维基百科 API 中搜索结果。然后可以相应地显示结果。

# Github 应用

Github 有一个 API，我们可以用它来创建自己的 Github 应用。它支持几乎所有我们可以用 Github 做的操作，比如获取、推送、拉取、合并、发出拉取请求、浏览器存储库、文件等等。

因此，如果我们在 Github 上托管我们的代码，我们可以使用它来自动化我们能想到的任何事情。

如果我们使用 Github，我们应该利用它来使我们的生活变得更容易。

# Zomato 应用程序

Zomato 是一个网站，上面有用户对许多餐馆的评论以及它们的信息，如位置、菜肴类型、菜单、照片等。

我们可以使用他们的 API 来制作我们自己的应用程序，来显示我们想要的任何东西。

如果一个 API 密匙。我们可以搜索餐馆，查看菜单，位置，类别，城市等等。

# 身体质量指数计算器

身体质量指数代表身体质量指数，即我们的体重相对于我们的质量的比率。

身体质量指数告诉我们一个人是否体重不足、正常体重、超重或肥胖。

我们可以制作自己的应用程序来计算一个人的体重指数，并显示他们是否属于上面提到的任何体重类别。

# 重量转换应用程序

我们可以制作自己的应用程序来转换重量单位。单位转换总是一件痛苦的事情，所以我们总是可以做一个程序来简化不同重量单位之间的转换。

我们的应用程序可以让用户选择不同的单位之间进行转换，然后我们向用户显示结果。

# 字数统计工具

手工计算一页的字数是不可能的。数了几个字就数不清了。

因此，字数统计工具很有用，它让我们练习如何操作字符串。

为了计算一段文本中的单词数，我们可以用分隔符将单词分成一个数组，然后找到一种方法来计算数组中的单词数。

完成后，我们向用户显示总字数。

# 联系方式

我们可以制作一个联系表格给其他人发信息。它对于嵌入我们自己的个人网站也很有用。

像 SendGrid 这样的电子邮件提供商允许我们通过他们的 SDK 发送电子邮件。只要确保我们不会通过发送垃圾邮件来滥用它。

在表单中，我们获取用户名、电子邮件和一条消息，然后创建一个后端应用程序来发送电子邮件。

# 图像滑块应用程序

使用图像滑块可以很好地显示照片。我们将照片放在滑块中显示图像，并通过添加自己的导航按钮来导航图像。

此外，我们可以添加一个功能来自动浏览图像。为此，我们添加了一个播放和停止按钮来开始和结束自动导航。

# 加法游戏

我们可以通过创建加法游戏来练习加法和编程技能。

我们的游戏为问题生成 2 个数字，并让玩家输入他们的答案。

每次用户回答时，问题都会随机生成。

为了使我们的游戏更具挑战性，我们可以添加一个计时器来显示时间，并在时间到时转移到下一个问题。

此外，我们可以保留玩家答对问题的分数。

# 结论

实践项目可以有多种形式。我们可以制作简单的游戏和方便的计算器。

此外，我们可以制作与各种有用的 API 对话的应用程序，以免费获取有价值的数据，就像 Zomato 和维基百科 API 一样。
