热重装SwiftUI应用程序

原文：https://levelup.gitconnected.com/hot-reloading-a-swiftui-app-77ba6ade1df7?source=collection_archive---------15-----------------------#2022-06-14

无需编译您的SwiftUI应用程序，即可在iOS模拟器中更改代码并立即看到更新！

你需要

  • Inject，由Krzysztof zaocki创建的Swift包
  • InjectionIII，这是一个macOS应用程序，用于执行繁重的工作(即，监视修改后的源代码并插入新功能，就好像它已被编译到SwiftUI应用程序中一样)

您可以在以下文章中了解更多关于Swift产品包的信息。

有大量的阅读材料，所以我想让你看看热重装的行动！在我的YouTube视频中，我向您展示了测试SwiftUI应用程序所需的所有步骤。

  1. 添加Swift包
  2. 在包含SwiftUI视图的源文件中导入其模块
  3. 将@ObservedObject private var iO = Inject.observer变量添加到SwiftUI视图中
  4. 在你的视图体定义的末尾调用.enableInjection()
