“你如何无中生有？！?"。这是我的兄弟问的问题，他是一名液压技工，试图弄清楚我到底是做什么工作的。

从外部来看，编程听起来像是巫术或只有车库里少数天才书呆子才能理解的东西。如果你已经学习了一两件关于编程的事情，那么就没有那么神秘了，但是当你让所有的事情一起工作的时候，仍然有一些顿悟的时刻。在这篇文章中，我打算帮助那些几乎没有编程经验、可能有兴趣转行或者只是有兴趣了解更多编程技巧的人。

什么是编程？

我喜欢把软件工程师或软件程序员看作是把业务需求翻译成一台或多台计算机能够理解的指令的翻译者。有许多不同的语言是专门为不同的问题或计算机系统而构建的，如Java、C#、SQL、JavaScript等。事实上，有几百种不同的语言可以用来开发软件！

大多数编程都是通过在您喜欢的文本编辑器或集成开发环境(IDE)中修改文本来完成的。文本编辑器的一个例子是Notepad++、Visual Studio代码、Sublime Text或Atom。IDE在文本编辑器上有额外的功能，如在其中运行代码，设置断点以在某些代码行停止代码执行，以及语法自动完成。IDE的一个例子是Visual Studio、Eclipse、IntelliJ或RubyMine。

一旦你写好了代码，你需要把它转换成计算机能理解的指令。这个过程在某些语言中被称为编译，你将文本表示转换成计算机或虚拟机可以执行的字节码或二进制码。把二进制代码(1010110)想象成计算机可以通过CPU(中央处理器)发送的一串开或关的电信号。

最后一步是将那些二进制或字节码指令部署或放置到将运行您的应用程序或程序的一个或多个系统上。对于web开发，这通常是AWS或其他云环境上的web服务器。对于移动开发，这是上传到应用商店，然后让您的最终用户下载应用到他们的手机上运行。

就是这样！写一个文本文件，编译成二进制指令然后上传到服务器。我在这里忽略了一些细节，但通常这是任何软件工程师为一个应用程序或系统所经历的工作流程。

节目的类别/流派

如果您想更深入地了解编程世界，可以更深入地了解细节，并为您提供从哪里开始的更多细节。有许多不同类别或类型的节目

手机开发

移动开发旨在为iPhone或Android手机开发应用程序，然后部署到各自平台的应用程序商店。对于Android开发，您将使用Java或Kotlin进行编程，并利用Android Studio作为您的IDE。一旦你建立了你的应用程序，你可以通过插入你的手机并在物理设备上运行应用程序或者使用模拟器(手机的软件版本)来测试它。在确认一切工作正常(在许多不同的手机上看起来都不错)后，你将把你的代码打包成一个zip文件，并提交到App store进行审查。一旦它在几天后被谷歌批准，那么它将被列出并可供任何有你支持的手机的人下载。

对于iOS开发来说，过程非常相似，你开发你的代码，用手机或模拟器测试，然后提交给苹果公司审查。您将使用XCode作为您的IDE，Objective C或Swift作为您的开发语言。要成为苹果的开发者，你需要支付99美元/年，而安卓开发者需要支付25美元/年。

网页开发

典型的web开发使用后端服务器来运行API，前端服务器向客户端提供JavaScript，API连接到数据库来存储用户数据。有许多框架可以用来快速构建web应用程序，其中包括React、Angular 2和VueJS作为使用JavaScript或TypeScript的前端框架。后端框架包括。Net Core使用C#，Spring Boot使用Java或Kotlin，Ktor使用Kotlin，Express使用Node，Ruby on Rails。典型的数据库包括Postgres、MariaDB、MySQL和Microsoft SQL Server。

还有其他技术可以利用，特别是当应用程序与移动应用程序集成或者需要扩展到越来越多的用户时。有时开发人员只关注前端或后端技术，被称为“前端”或“后端”开发人员。跨前端和后端系统构建的开发人员被称为“全栈”开发人员。

硬件开发

我不太熟悉嵌入式系统工程师的典型工作流程，但我猜他们在Linux或Unix系统上使用Emacs或Vim，使用C或C++或FPGA编程语言进行开发。这种类型的编程是在定制硬件(想想F22或737飞机)或需要定制特定电路或用途的系统上完成的。

游戏开发

游戏开发可能是学习编程时最受欢迎的搜索类别之一。现在有许多不同的框架可以用来开始构建图形、利用物理引擎和处理碰撞检测。一些例子包括Unity、虚幻引擎、Cry引擎和Xamarin。代码可以在你的Xbox、PC、平板电脑或Playstation上运行，可以用C/C++/C#/Python编写。多人游戏也将连接到服务器来集中管理游戏的运行，以减少所有玩家之间的延迟。

物联网发展

这一类别正在以令人难以置信的速度增长，随着我们的家庭变得越来越“智能”，我们看到越来越多的联网设备创新需要编程。这些设备中的一个例子是机器人真空吸尘器，其可能与使用C/C++或FPGA编程语言的硬件设备共享类似的编程。如果设备需要与后端服务器通信，那么该服务器将类似于利用web框架和诸如Spring Boot和Java之类的伴随语言的web服务器。

如何开始编程

  • 缩小你想做的编程类型。你想开发移动应用吗？你更愿意建一个网站吗？你有兴趣创造一个游戏吗？做一些研究，直到你能弄明白，甚至尝试用这些语言进行一些基本的编程
  • 选择一种语言。一旦你缩小了你想从哪种类型的编程开始，选择一种语言，并开始获得专业知识和经验。
  • 在Youtube上观看教程，阅读博客文章，搜索StackOverflow，学习几门Udemy课程或阅读书籍。无论你学得多好，都要从那里开始。
  • 如果你想转行，可以考虑参加一个编码训练营。我见过许多人在不到一年的时间里，从不太懂编程到找到一份年薪7.5万英镑的软件开发工作。
