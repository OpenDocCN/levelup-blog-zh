什么是CSS？

CSS是一种描述HTML文档风格的语言。它代表级联样式表。

CSS用于定义网页的样式，包括设计、布局以及不同设备和屏幕尺寸的显示变化。

添加CSS的三种方法

将CSS放入文档有三种方法:

  1. 外部样式表
  2. 内部样式表
  3. 内嵌样式

外部样式表

这是将CSS附加到文档的最常见和最有用的方法，因为您可以将CSS链接到多个页面，从而允许您使用相同的样式表来设置它们的样式。

这意味着你可以通过改变一个文件来改变整个网站的外观。

内部样式表

内部样式表是指没有外部CSS文件，而是将CSS放在包含在HTML <head>中的<style>元素中。

内嵌样式

内嵌样式是只影响一个元素的CSS声明，包含在一个style属性中。

基于 CSS特异性解释文章。这种样式的特异性值为1000。在我看来，在开发过程中，有些情况下我们仍然必须应用内联风格。然而，因为它通过选择器覆盖了CSS访问，所以在使用它时要非常小心。

五种选择器

css中有5种类型的选择器:

  1. 简单的选择器包括HTML标签名，HTML标签ID，类名。
  2. 组合选择器
  3. 伪类选择器
  4. 伪元素选择器
  5. 属性选择器

简单选择器

简单的选择器有:

  1. 类选择器选择具有特定类属性的HTML元素。要选择具有特定class的元素，编写一个句点.字符，后跟类名。Ex .my-class {}
  2. id选择器使用HTML节点的id属性来选择特定的元素。元素的id在页面中是唯一的，因此id选择器用于选择一个唯一的项目。要选择具有特定id的元素，请编写一个散列字符#，后跟元素的id。Ex #my-id {}
  3. 通用选择器 *选择页面上的所有HTML元素。
  4. 元素选择器选择指定元素名称的所有元素。Ex div {}或span {}

组合选择器

CSS中有四种不同的组合子:

  1. 后代选择器匹配指定元素的所有后代元素。
  2. 子选择器选择指定元素的所有子元素。
  3. 相邻兄弟选择器选择指定元素的所有相邻兄弟元素。兄弟元素必须有相同的父元素，“相邻”意味着“紧随其后”。
  4. 通用兄弟选择器选择指定元素的所有兄弟元素。

伪类选择器

伪类用于定义元素的特殊状态。

元素的“状态”是什么？

  • 它是当用户将光标移到按钮上或移出时按钮的一种状态。
  • 它是链接被点击前后的状态。
  • 它是输入被聚焦时的一种状态。

伪元素选择器

CSS伪元素用于样式化元素的指定部分。

什么是元素的指定部分？

  • 样式元素的第一个字母或第一行。此处举例。
  • 在元素内容之前或之后插入内容。更多细节此处。

属性选择器

有7个属性选择器:

  1. [attribute]选择器用于选择具有指定属性的元素。
  2. [attribute="value"]选择器用于选择具有指定属性和值的元素。
  3. [attribute~="value"]选择器用于选择属性值包含指定单词的元素。
  4. [attribute|="value"]选择器用于从指定值开始选择具有指定属性的元素。值必须是一个完整的字。
  5. [attribute^="value"]选择器用于选择属性值以指定值开始的元素。值不是必须是一个完整的单词。
  6. [attribute$="value"]选择器用于选择属性值以指定值结束的元素。值不是必须是一个完整的单词。
  7. [attribute*="value"]选择器用于选择属性值包含指定值的元素。值不是必须是一个完整的单词。
