信息_模式表表

所需要的只是对INFORMATION_SCHEMA TABLES表的简单查询。我有一个“步行”数据库，其中存储了与步行锻炼有关的数据，如果我想知道哪些表是该模式的一部分，此查询将检索该信息:

SELECT TABLE_NAME
FROM TABLES
WHERE TABLE_SCHEMA = 'walking';
