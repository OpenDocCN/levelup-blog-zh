打造一份强大而有吸引力的入门级网站开发人员简历的6个技巧

原文：https://levelup.gitconnected.com/6-important-elements-to-craft-a-powerful-and-compelling-entry-level-web-developer-resume-52212e8c7d2b?source=collection_archive---------47-----------------------#2021-03-07

获得第一份入门级web开发人员工作的一个重要部分是你的简历。但是到底什么是简历呢？

你的简历本质上是你过去为他人提供的价值的记录。这可以是任何事情，从过去的工作到志愿者经历，甚至是自由职业者。即使你没有该行业的“专业经验”，比如过去的开发工作或实习经历，你仍然可以展示你是如何提供价值的。

你的简历让雇主对你如何帮助解决雇主的问题感兴趣。这篇文章的目的是帮助你制作一份简历，以最好的方式展示你的履历。

以下是我希望有人在我获得第一份网站开发工作时告诉我的6个技巧。下面是对每一个问题的简要总结:

  1. 创建一份定制的目标陈述:为你申请的每个职位创建一份简短的定制目标陈述。
  2. 不要笼统地列出你的教育背景:不要只列出你的专业和你去过的地方。通过仔细阅读工作描述并将其与你的项目联系起来，利用过去的学校项目。
  3. 利用你过去的经历:不要发送你过去的工作和实习经历的相同要点，用工作描述中的关键词定制每份简历。
  4. 提到相关的技能和证书:你可能在申请一份网页开发员的工作，但是，你很可能有很多其他的技能在工作中有用。
  5. 链接到你的作品集网站:你要确保包含一个到你作品集网站的链接。如果雇主已经足够深入地查看了你的作品集网站，他们会很感兴趣，然而，你可以使用作品集网站来真正展示自己。
  6. 分享你的GitHub :即使你可能觉得你的代码不是世界上最好的，整理和清理GitHub是值得的，这样其他人就可以理解你的代码是关于什么的。一旦你整理好了，在简历中加入一个GitHub的链接。

这篇文章的其余部分将会详细介绍每一条建议，但是，要记住的是，你必须愿意接受一些拒绝。只要知道你只需要一个“是”，你就能得到你的第一份网站开发工作。

1.创建自定义目标陈述

拥有一个客观的部分可以帮助你为你申请的每一份工作定制你的简历。为你申请的每一份工作定制一条信息可以帮助你迅速脱颖而出。在目标部分，您还可以提及有助于通过ATS系统的重要关键词。

如果您不确定ATS系统是什么，请点击此处了解它。

为了确保你写的东西是有效的，阅读职位描述，并试图了解他们在寻找什么以及为什么。从那里，你可以去该公司的网站，并进一步了解为什么。了解雇主为什么要找人来填补这个职位可以帮助你更好地定位自己。

例如，假设你正在申请两个前端web开发人员的职位，这两个职位都需要懂React、Javascript和CSS的人。

第一个雇主在工作描述中提到，他们正在寻找能够快速创建React组件的人。随着进一步的研究，你会注意到这家公司是一家初创公司。他们最有可能寻找能够快速构建React组件的人，因为他们想要创建快速原型。

有了这些信息，我们可以开始想出一些东西，包括ATS系统的重要关键字，也是人类可读的。一个小技巧是加粗或突出那些重要的关键词。这些词是否加粗对ATS系统没有影响，但是，当人们浏览你的简历时，加粗的词会很快吸引他们的注意力。因此，我的目标部分可以如下所示:

“利用我在 React ，Javascript和CSS中的经验，帮助ABC公司使用React的最佳实践快速构建功能原型。”

如今，对一家公司进行研究也相对容易。大多数情况下，在谷歌上简单搜索一下公司名称就能获得足够的信息。如果你试图找到一家初创公司的信息，你可以使用Crunchbase.com获得额外的数据，如收入、投资者、他们筹集了多少资金等。

2.非一般性地包括你的教育

如果你是学生，你通常不会有太多的工作经验可谈。雇主知道这一点，但是，他们仍然希望看到你如何帮助解决他们的问题。

然而，许多学生犯的一个大错误是想出了一个极其普通的教育部分。列出你在哪里上学，你的专业是什么，你的平均绩点是多少就可以了。把课程名称或你参加过的俱乐部写进简历是不好的。

例如，如果雇主正在寻找精通React的人，而你已经做了一个在React中建立网站的学校项目，那就在你的教育部分谈论这个项目。

即使是雇主可能在寻找的软技能，你也很有可能找到你在学校所做的显示这种软技能的事情。例如，如果雇主正在寻找一个好的沟通者，而你帮助组织了一个学生俱乐部，你应该提到这一点。谈谈你是如何与多个学生交流并管理一个俱乐部的。

你想通过花一点时间研究来避免你的教育部分过于笼统。

3.利用你过去的经验

你过去工作过的工作或实习是展示你有帮助企业解决问题的记录的好方法。

同样，你要确保你研究了职位描述，并且公司为你过去的工作或实习定制了每份简历的描述。

但如果你过去的工作或实习经历与你申请的工作无关呢？例如，你正试图从非技术背景转行。

即使你过去的经历与你申请的工作无关，你也可以仔细描述你的工作经历，以最符合雇主的要求。

例如，假设你过去是一名教师，正在寻找一份网站开发人员的工作。通过阅读工作描述，你注意到他们正在寻找一个不仅能编码，还能向团队成员提供更新的人。在描述你过去的教学经历时，你可以包括你会向其他老师介绍你学生的进步。

一般来说，在谈论你过去的经历时，你要记住以下规则。他们可以帮助你更好地展示你的专长。

具体点

你对过去如何为雇主提供价值的描述越具体越好。尽可能使用数字。

例如，如果你已经帮助客户提高了网站在你的自由职业项目中的加载速度，你就要试着加入具体的数字。快了多少？仅仅快10%还是快50%？你想具体点。这有助于你更好地推销自己。

谈论解决方案

你在之前的工作中每天做什么对雇主来说并不重要。他们想知道你能为以前的雇主提供什么样的解决方案。尽最大努力避免谈论任何不能展示你是如何帮助公司解决问题的事情。

例如，在实习中，你可能使用了React，然而，不要只是提到你使用了React，而是谈谈你用React做了哪些对雇主有帮助的事情。你有没有想出一套组件来构建一个更大的功能或产品？

将你的工作与你过去如何帮助雇主联系起来，这表明你有能力采取行动帮助企业。你可能只是申请成为一名网站开发人员，但是，你是企业的一部分，你需要了解你的工作如何为企业带来价值。

提及关键词

这个区域是你可以在简历中添加重要关键词的另一个地方。如果你过去从事过某项特定的技术，一定要以某种方式把它包括进来，并且一定要把关键词加粗。

例如，如果你申请一份需要WordPress和PHP开发人员的工作。如果你在自由职业项目中使用过WordPress，你应该确保在你的工作要点中包含这个关键词。一定要把关键词加粗，这样当人们阅读时，加粗的文字可以吸引眼球。

4.提及相关技能和认证

你很可能有很多其他额外的技能对工作有帮助。你要在这一节讲这些额外的技能。证书可以帮助你向雇主证明你确实拥有这项技能。

您通常希望这一部分尽可能简短。没有认证，雇主就没有真正的方法来证明你拥有这项技能。显示你拥有这种技能的另一个方法是在你的作品集网站上放一个你的作品的例子。

例如，如果您熟悉React Native并在过去开发过移动应用程序，您可以在本部分将React Native作为一项技能。然后，在你的作品集网站上，你需要包含一个指向你的手机应用程序的链接和代码。

如果你包括雇主想要的任何重要关键词，确保文本也加粗。这将进一步帮助通过ATS系统。

5.链接到您的作品集网站

确保包含投资组合网站的链接。更好的是，你可以包含一个链接到你的投资组合网站的二维码。只要确保雇主有一个简单的方法到达你的网站。

例如，如果你发送了一份PDF格式的简历，确保链接是可点击的，这样他们就不用复制粘贴到浏览器中了。如果你寄了一份纸质简历，你可以附上一个二维码，雇主可以很快扫描到。

在你的投资组合网站上，你可以向你的雇主推销他们为什么应该选择你。如果雇主花时间看了你的投资组合网站，他们很可能对你感兴趣。

作品集网站的独特之处在于，你可以随心所欲地发挥创造力，不像简历那样必须进行关键词优化。你可以设计任何你认为最能展示你的作品和技能的方式。

您可以包含您构建的网站或应用程序的链接。如果你有自由职业者的客户愿意提供证明，你可以把他们的视频作为社交证明。

在建立你的作品集网站时，你可以选择很多设计，但是，这不是这篇文章的目的。要知道，你可以利用许多心理偏见来帮助指导你如何设计你的投资组合网站。我将在未来写下这一点，并在完成后编辑这篇文章，并附上链接。

6.分享你的GitHub

包含一个Github的链接可以让雇主有机会在联系你之前审核你的代码和工作。

不过这也需要你优化你的Github。如果你有一堆没有真正描述代码内容的随机存储库，雇主将很难找到他们想要的东西。

您希望确保在您的存储库的描述中也提到重要的关键字。这将有助于进一步推销你能胜任这份工作。例如，您可以提到React项目的代码库连接到Firebase数据库。

此外，如果你为开源做出了贡献，你也可以在你的Github个人资料上展示这一点。

结论

说到底，这都是为了展示你如何解决雇主的问题。而是推销你的技能。毕竟你是在用你的时间和专业知识来换取金钱，所以你需要出售你的技能。

你越能推销自己，你就越有可能获得面试机会，离你的第一份网站开发工作更近一步。本文中的要点有助于指导你创建一份令人信服的简历。
