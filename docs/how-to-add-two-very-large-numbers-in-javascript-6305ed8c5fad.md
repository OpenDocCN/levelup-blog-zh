# **手头的问题**

> *创建一个算法，该算法包含一个函数，该函数接受任意两个数字，但采用字符串形式，并将总和作为字符串返回。*
> 
> *该函数将用于在 Javascript 中添加非常大的数字。之所以需要这个程序，是因为在纯 javascript 中不能存储超过 53 位的数字。*
> 
> *为了对较大的数字进行运算，我们把它们转换成字符串，然后对它们进行运算。*
