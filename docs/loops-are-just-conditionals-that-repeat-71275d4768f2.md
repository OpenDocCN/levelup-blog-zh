# goto 和循环有什么关系？

这是我问自己的第一个问题。最初，很明显`while`循环实际上只是一个 if 语句和一个伪装的 goto。但是对于我们喜欢的其他类型的循环来说也是这样吗？

我觉得有可能，所以我决定调查一下。第一步是缩小我的循环列表。在 PHP 中有四个:

1.  for 循环
2.  while 循环
3.  do-while 循环
4.  foreach 循环

我决定尝试只用`goto`和条件句来重建这些。
