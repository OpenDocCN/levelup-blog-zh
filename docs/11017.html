印度出口的软件 比沙特出口的石油还要多

就算你不是足球迷，也希望你听过“石油钱”这个名词。如果你和我一样是曼城或者巴黎圣日耳曼的球迷，我敢肯定你喜欢石油收入。哈哈。

好吧。“石油钱”是什么意思？顾名思义，石油货币是通过石油工业获得的财富。沙特阿拉伯是世界上原油出口最多的国家之一。截至2021年10月，沙特原油出口量为683.3万桶/日。

根据维基百科的统计，石油部门约占沙特预算收入、出口收入的90%和GDP的42%。

现在来印度。印度目前是世界第六大经济体，亚洲第三大经济体，仅次于中国和日本。但是今天的话题不是关于一个国家的经济，而是关于印度不断增长的软件收入。

印度的软件收入

印度储备银行在9月份进行的一项调查表明，2021年3月，软件服务以及印度公司的外国子公司提供的服务的出口额为1483亿美元，超过了沙特阿拉伯2021年的石油销售额1453亿美元

美国已经用它的技术占领了世界。投资者现在像雨点般向科技企业投入资金。每个科技公司都需要软件，对吗？

一些公司有自己的内部技术团队，并由他们的团队制造技术产品。但是许多人尝试经济有效的方法。他们把工作外包出去，而印度攫取了大部分工作。

印度的优势来了。企业可以在印度制造比美国或欧洲更便宜的软件。印度和其他一些南亚国家如孟加拉国、巴基斯坦就是这种情况。

根据商业标准，美国是软件出口的主要目的地，2021年占54.8%；欧洲占30.1%，其中近一半在英国。

根据 GitHub 的数据，在印度大约有580万开发者。

今后

在covid在世界上流行之前，开发人员确实是在家工作的。但从covid时期开始，这种趋势变得如此流行和正常。许多在covid之前没有雇佣远程开发人员的公司现在都欢迎它，并且可能永远不会回去。

许多没有从海外招聘员工的公司可能很快就会这么做。这意味着更多的工作会来到印度和孟加拉国。

印度将会以前所未有的速度发展更多的开发者。不仅印度，大多数南亚国家也会这样做。年轻一代和聪明的头脑将会欢迎它。因为在这些国家，科技工作的薪水比非科技工作高，政府也为他们提供许多额外的福利。

另一个南亚国家孟加拉国最近宣布对所有来自国外的自由职业收入免税和4%的奖励。

对于成长中的开发者来说，普通的软件开发会变得更便宜。而且，越来越多的公司会将他们在南亚或非洲的工作外包出去。人工智能、区块链等新技术将成为科技行业的优质工作。
