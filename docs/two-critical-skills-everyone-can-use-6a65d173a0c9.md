# 每个人都可以使用的两个关键技能

> 原文：<https://levelup.gitconnected.com/two-critical-skills-everyone-can-use-6a65d173a0c9>

你上一次和别人交谈是什么时候？你最近一次在社交媒体上写邮件或发帖是什么时候？你永远也猜不到对这两者都至关重要的两项技能。

![](img/330b579da646763541fa66bad82d8c68.png)

照片由 [Aaron Burden](https://unsplash.com/@aaronburden?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral)

没错。

擅长**口头和书面交流**是当今人们可以拥有的两项最重要的**技能。**

你可能会说“安德鲁，我已经知道如何写作和交流了，这有什么大不了的？我已经知道该怎么做了，没必要再去做了。”

*错了。*

投资变得更好地写作和沟通可以打开你没有梦想或想象的机会之门。你能站在一群投资者面前，让他们投资你的想法吗？你能设计一封电子邮件来获得真正站在这些人面前的机会吗？

真正的技能和挑战是把一个想法或信息以一种其他人能够以类似的方式掌握和理解的方式进行交流。

![](img/1603759ee439c49e8d7db8ae41eda832.png)

照片由 [Kvalifik](https://unsplash.com/@kvalifik?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

这可以通过画一个图表，做一个演示，列出一个清晰的项目清单来完成。以口头和书面形式交换信息的方式几乎是无限的，这样别人就能理解你，并从你提供的信息中获得价值。越来越好地磨练这些技能是你时间的最佳用途之一。

# 磨练这些技能的方法

*   阅读一些关于不同沟通方式的书籍(查看我关于领导力书籍的博客)
*   找一个你想向其学习沟通技巧的导师
*   通过在线写作(书籍、博客、社交媒体)来练习写作技巧
*   参加不同沟通技巧的课程

无论你如何选择变得更好，你未来的自己和其他从你身上学习的人都会感谢你。
