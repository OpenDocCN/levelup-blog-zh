我不知道你是怎么想的，但是向远程工作的转变已经把 Outlook 放在了我工作生活的前沿和中心。它决定了我在什么时候和谁在一起——我发现如果不加以控制，我的日子和星期会在你意识到之前被一个接一个的会议填满。

我想和你分享两个广泛的技巧，一个是关于创建有效的会议，另一个是关于管理你不断收到的邀请。

# 技巧 1:难以捉摸的会议议程

我喜欢面对面工作！你可以看到谁在附近，并在你喜欢的任何时候召集人们进行临时聊天——当你们一起在办公室时，很容易就可以跳进一个分组讨论室并开始实时白板演示。

一旦你在一个遥远的环境中(我认为我们将在一个最好的混合模式中前进)，我们需要慎重考虑我们从人们那里占用的时间。我们都忙得不可开交，所以我们需要意识到其他人的时间，并给他们足够的信息来决定他们是否需要参加你的会议。

起初这可能会引起不和谐，“*你说人们有选择是什么意思？现在是* ***我的*** *会议！*“是的，有些人可能有更重要的事情要做(对不起，这是真的！)，而且这样就可以了！它是双向的。

## 制定议程

议程意味着你已经花了时间思考你试图解决的问题。如果是你的直属团队，而且你已经在 slack 或其他场合谈论过了，可能就没那么重要了。但是，当你邀请那些可能跟不上你当前想法的人时，议程将有助于设置场景。

## 告诉人们你想从会议中得到什么。

这可以是整个议程的一部分，并再次让人们知道你已经真正考虑了我们为什么要开会。

类似于:

*   关于 x 我想做个决定，(决定！)
*   我想讨论一下 X，论证一下为什么 Y 更优越。(知识分享！)
*   决定行动项目，以确保我们不会错过任何与 y 相关的内容。

## 包括预读

如果你在谈论一些复杂的东西，发送适当的链接到维基、文档，甚至是你在餐巾纸上草草写下的东西的图片。让人们异步开始的任何东西！

## 告诉人们你想从他们那里得到什么。

有时我收到一个会议邀请，但我不确定我为什么会在那里。我是他们想随时了解的人吗？我是决策者吗？我是合作者吗？IDK。我已经开始在议程底部列出与会者，并给他们一个角色。

*   迪安·贝克(组织者)，
*   大狗(决策者)，
*   Joe 博客(合作者)等等。

如果你不能为一个与会者找到一个合适的标签，他们为什么参加？

## 我知道你在想什么。

# 提示 2:组织你的会议

当我在 Outlook 中查看我的日历时，如果是糟糕的一天，我只能看到红色的海洋(我的默认颜色)。更有可能的是，我预订了双倍的街区，有时甚至是三倍。是时候对事情进行分类了，这样你就可以区分 ***重要的*** 会议，或许..不太重要的。

## 旗帜

Outlook 让您有机会对会议进行颜色编码。对我来说，用一种颜色看到重复的会议，用另一种颜色看到 1:1 的会议(当然，还是重复！)，等等。现在只有临时会议是默认颜色，我可以很快看到我一周中有多少时间是属于自己的。

![](img/b828397ba404a5afe6d4c4741ab2182d.png)

用颜色区分会议类型

因为我经常被重复预订，所以我可以很快看出一些事情是否跨越了非常重要的事情，比如 1:1 或与我女儿一起游泳的时间，而不是一些可能没有很好定义的特别事情。

我还依靠一个功能，在周五之前预订一周的焦点时间，以确保我可以有一些(通常被预订)自己的时间，你知道，做实际的工作。

## 开始退出。

这是一个很好的练习，看看你有多少重复的会议占用了你的时间。你可以开始问这样的问题:**这里真的需要我吗？**根据提示 1，你知道你开会的目的吗？如果你……没有出现会怎么样？

我非常喜欢举手示意他们要么没有增加价值，要么从会议中获得价值。我认为这是我们都应该做得更好的事情。有多少次你在开会时完全心不在焉？在家工作更是雪上加霜——现在我可以离开摄像头，只是查看邮件和休息，几乎可以做任何我想做的事情。不参加怎么样？发出一条“**谢谢，但我需要下车了，如果有什么我能帮忙的，请告诉我**消息。如果它再次出现，考虑放弃它，并给组织者一个很好的说明。

# 包扎

这些都没有灵丹妙药。我从让 Outlook 成为我的新霸主中学到的一个明确教训是，你必须尊重人们的时间。这包括你自己的时间。

通过为会议设定一个清晰的议程、结果和角色来展示你已经尽了你的职责，以使会议尽可能有效。如果你觉得会议没有增加价值，不要害怕退出。
