打开PowerShell:

  1. 按下“⊞之窗”
  2. 在搜索栏中输入“PowerShell”
  3. 单击“以管理员身份运行”

打开主目录:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cd $HOME\desktop

安装Virtualenv:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
python -m pip install virtualenv
