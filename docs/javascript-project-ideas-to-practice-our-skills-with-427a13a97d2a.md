要想精通 JavaScript，你必须大量练习。要大量练习，大概需要各种各样的 app 思路。

在本文中，我们将看看一些应用程序和小部件的想法，我们可以用它们来练习 JavaScript 编程。

# 计算器应用程序

我们可以创建一个像真正的计算器一样有按钮的计算器应用程序。为了让按钮工作，我们必须操作字符串，这样我们就可以在用户点击按钮时构建算法。

然后，当用户点击 enter 时，我们必须计算结果。如果我们想走简单的路线，我们可以使用 dread `eval`函数，或者我们可以自己解析算术表达式字符串来计算结果。

# 反应本机待办事项应用程序

React Native 是一个基于 React 的移动框架，用于使用 React 和 JavaScript 代码创建移动应用。

这是开始学习移动开发的好方法，因为这是从制作移动 web 应用程序的一个简单过渡。

我们可以使用 React 及其库来创建一个移动 todo 应用程序。

# 工作抓取 Web 应用程序

我们可以制作自己的应用程序，从网站上创建临时工作列表。要用 JavaScript 编写它，我们必须让它在 Node.js 上运行

有像`request`这样的库可以向网页发出 HTTP 请求，然后我们可以用像`cheerio`这样的东西解析结果。

# Twitter 数据挖掘器

与 job scraper 应用程序一样，我们可以通过解析 Twitter 页面并以编程方式请求 URL 来获取导航到页面的数据。

然后，我们可以解析响应，并像使用作业抓取器一样解析它们。

我们还可以使用 Twitter Node SDK 以这种方式获取数据，如果我们想要的数据可用，这比直接从 Twitter 抓取要好。

# 微博

我们可以建立类似 Twitter 的东西，在那里我们可以发布消息，并在下面有一个评论区，供人们回应。

如果我们想改进它，我们也可以让用户上传图片和视频。在接受上传之前，请务必检查格式和大小。

如果我们想做得更好，我们也可以解析标签。

# 拼写检查器

我们可以通过解析输入值中的字符串来构建自己的拼写检查器，然后检查每个单词的拼写错误。

然后，我们必须给他们一些单词的建议，用户可能想改变拼写错误的单词。

# HTTP 服务器

Node.js 附带了`http`模块，我们可以用它来创建自己的 HTTP 服务器。

我们可以学习如何解析请求 cookies、头部和主体，这样我们就可以对它们进行处理。

这是了解更多 HTTP 通信的好方法。

# 手电筒应用

通过让用户改变颜色、亮度等，手电筒应用程序可以变得更加复杂。

然后我们有一个比其他应用更好的手电筒应用。我们可以在浏览器上用普通的 JavaScript 轻松做到这一点。

# 具有用户认证的应用程序

用户身份验证是大多数应用程序都必须做的事情，因为它们持有私人用户数据。

所以做一个有用户认证的 app 是很棒的做法。我们可以创建一个简单的前端，让用户注册一个用户帐户并登录。

然后，我们必须创建一个后端应用程序来保存用户数据，并让人们使用正确的凭据登录。

如果我们想为我们的身份验证应用程序添加一些授权功能，我们还可以添加多种类型的用户。

# 特雷罗克隆

Trello 是一个简单的任务跟踪器。它允许我们将任务框拖放到不同的列来更新它们的状态。

因此，要制作一个 Trello 克隆，我们必须学习如何创建一个具有拖放功能的 UI，并在用户将任务放入新框时更新任务状态。

此外，我们必须创建一个表单，让用户添加和更新表单。

# Yelp 克隆

Yelp 是一个餐馆和活动的评论网站。我们可以通过创建让人们进入餐馆、事件和评论的表单来创建我们自己的 Yelp 克隆。

此外，我们必须添加一个用户表单，让用户注册并添加我们上面提到的那些东西。

# 结论

我们可以从许多想法中创建练习应用程序，来练习用 JavaScript 编程。我们只需要关闭教程并开始练习。

那么我们很快就会成为一名精通 JavaScript 的开发人员。
