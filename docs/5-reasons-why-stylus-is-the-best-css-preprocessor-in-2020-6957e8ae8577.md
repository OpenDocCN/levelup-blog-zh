# #4 手写笔有许多参考类型🍭

## 部分引用🍩

我听说你喜欢筑巢？如果你在嵌套的第 4 层，你想把目标放在第 2 层的父节点上呢？

好吧，手写笔已经覆盖了你！

![](img/e7daf341c89b4eeb93293e60c5e30fa6.png)

部分引用

*   `**^[N]**`瞄准第 n 层的选择器
*   根(此处为`.a`)被认为是在第**0 层**
*   **负值**目标选择器来自**当前父**
*   当前父代(`.c`)可以通过传统的`&`选择器定位
*   **祖父母**可以通过使用`**^[-1]**`比父母高一级来选择
