使用TypeScript、TailwindCSS和Neovim设置Next.js应用程序

步骤1:创建Next.js应用程序

$ yarn create next-app --typescript my-app
$ cd my-app

步骤2:安装TailwindCSS

$ yarn add -D tailwindcss postcss autoprefixer
$ npx tailwindcss init -p
