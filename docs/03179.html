我们可以用免费API构建有用的实践应用

原文：https://levelup.gitconnected.com/useful-practice-apps-we-can-build-with-free-apis-a829e19b1b96?source=collection_archive---------16-----------------------#2020-04-23

在软件开发领域，熟能生巧。所以要尽可能多的找方法练习编程。

有了免费的公共API，我们可以通过创建使用这些API的应用来练习编程。写完实践应用后，我们甚至可以花钱买一些API，写出更有用的应用。

在本文中，我们将研究一些可以使用这些API的实践项目想法。

电话号码验证器

我们可以使用 numverify API来验证电话号码，以确保它们是有效的。

现在，我们不必担心拨打可能没人使用的电话号码。

它可以处理来自232个国家的电话号码，并且可以告诉我们电话号码所在的电话公司和位置。

Git应用程序

大型Git主机都提供了自己的API，让我们可以自动完成需要它们完成的任务，比如拉/推代码、合并代码等。

还有 Bitbucket 、 GitHub 和 Gitlab ，都可以让我们用它们做很多Git操作。

它们都非常安全，因为身份验证是使用OAuth完成的，授权与使用UI操作Git存储库是一样的。

网站/ RSS聚合器

使用 import.io API，我们可以聚集来自各种网站和RSS提要的数据，并从一个来源获得它们。

它支持CORS，因此我们可以直接从我们的前端应用程序访问数据。使用此API需要API密钥。

用import.io聚合的数据在JSON中返回，这样我们就可以马上使用这些数据。

我们可以用它来抓取网站和RSS源，并根据我们的意愿提取数据。作业可以动态运行，也可以通过使用API设置计划来按计划运行。

查找其他网站上使用的技术的应用程序

有没有好奇过我们访问的网站上有哪些技术？我们可以使用让我们验证 API来创建一个应用程序，展示我们选择的网站上使用的技术。

它不需要任何认证，因此我们可以立即使用它们。

API测试应用程序

我们可以使用 Postman API轻松地向API发出我们自己的请求。它让我们通过使用Postman API编写程序来编写API测试脚本。

Postman API可用于发送查询字符串、头，并获得响应正文和头。

字典应用

有了各种API，我们可以用各种字典API创建自己的字典app。

有韦氏词典 API、牛津字典API、 Wordnik API和单词 API。

只要我们限制我们的查询并注册一个API密匙来使用它们，它们都是免费的。

它们提供了单词的定义、发音、单词类型和更多数据。

对于单词API，它还提供了单词的同义词。牛津字典API也有一个词库，单词的翻译，句子API来获取给定单词的例句，等等。

如果我们付费，那么我们可以访问更多的东西，比如每天更多的API请求

文件转换应用程序

Cloudmersive文档和数据转换应用程序让我们编写程序来使用API在各种格式之间转换数据。

它们包括将HTML、Office文件或照片文件等常见格式转换为PDF格式。

我们还可以用它将多个Office文档合并成一个。

API可以转换的其他数据格式包括CSV到JSON和XML到JSON。

Cloudmersive还开发了其他API，如电子邮件验证API、自然语言处理API、OCR APIs、条形码扫描API等。

它可以获取图像，然后用OCR API将它们解析成可读的文本。条形码扫描器API获取条形码的图像，并从条形码返回数据。

结论

有很多API可以用来做各种事情，比如转换文件、条形码扫描、OCR等等。

我们还可以使用API来自动化Git操作，比如从repos推送、获取和提取代码。

最后，我们可以用各种API验证像电话号码这样的数据。
