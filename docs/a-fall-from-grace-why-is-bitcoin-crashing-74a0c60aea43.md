# 失宠:比特币为何崩盘？

> 原文：<https://levelup.gitconnected.com/a-fall-from-grace-why-is-bitcoin-crashing-74a0c60aea43>

![](img/aeed52178cdf886b017b2d8dd46ea9d5.png)

比特币一直在经历过山车，经历了一些潜在的价格大幅波动。不仅像比特币基地这样的大平台看到交易量下降，因为许多投资者感到手头拮据，而且 crypto 也降到了今年的最低水平。

![](img/3f8090d736dcd27ce6274efe9787b148.png)

(图片来源: [CoinDesk](https://www.coindesk.com/markets/2022/07/01/brutal-month-for-bitcoin-as-june-ends-with-biggest-drop-in-11-years/#:~:text=Bitcoin%20(BTC)%20slumped%20over%2037.3,to%20line%20up%20financial%20lifelines.) )

在询问 crypto 为什么会降价时，有一些重要因素需要考虑。随着生活费用的增加，通货膨胀目前处于创纪录的高水平，这也带来了恐惧和信心的丧失。这些担忧是基于整体经济，而不是与加密市场直接相关。

其中，网络安全专家警告称，在价格下跌的过程中，可能会有潜在的[黑客攻击](https://www.forbes.com/sites/jonathanponciano/2022/03/29/second-biggest-crypto-hack-ever-600-million-in-ethereum-stolen-from-nft-gaming-blockchain/)，这似乎是一个有组织的禁令在起作用。随着全球主要市场暴跌，投资者开始担心这对他们的投资组合意味着什么。

但在我们开始指责比特币造成了最近世界各地市场的不稳定之前，我们应该停下来想想它为什么以及如何崩溃。有很多因素可以解释比特币最近从[的荣耀中跌落。](/bitcoins-disastrous-few-months-show-that-sustainability-is-the-world-s-most-sought-after-asset-f43e0ba24e40)

# 比特币为什么会崩盘？

一些人想把比特币的下跌归咎于网络罪犯。这些人会说，大量网络犯罪分子一直在使用比特币来清洗他们非法活动所得的金钱。一些人甚至说这是比特币崩溃的主要原因之一。

这里的论点是，如果你看看网络犯罪分子开始转向数字货币的速度，那么你很容易得出结论，他们更有可能导致比特币的崩溃。

这一观点的问题在于，网络犯罪分子将不得不使用相当一部分资金将其兑换成比特币，鉴于区块链技术在大多数情况下的透明性，这很难做到。如果他们有一大笔钱，那么他们可以很容易地把钱花在其他东西上，比如奢侈品。

这种说法有道理，但它没有解释比特币的价值为何以及如何暴跌。出于同样的原因，以太坊等加密货币正在经历类似的崩溃。也没有证据表明网络罪犯是这些加密货币价格下跌的幕后黑手。

另一个可以解释最近下跌的因素是，比特币对采矿业不再有利可图。这意味着，使用电脑和智能手机开采比特币的人，随着时间的推移，将会遭受损失，尤其是在电力成本高的地区。

这可以解释比特币崩溃的原因是，许多矿工已经开始将他们的采矿设备兑换成美元，这解释了 BTC 价值的下降，但这一理论没有解释为什么其他加密货币，如以太坊，也在经历下行。同样，这些加密货币[和网络罪犯](/5-rules-for-your-personal-and-companys-cybersecurity-4cad3d2b82c2)没有什么特殊因素。

比特币暴跌的真正原因是，许多投资者现在开始意识到这种货币有很大的波动性。在 12 月牛市期间买入比特币和其他加密货币的投资者预计它们会比现在更高，但相反，它们开始崩盘。

许多人害怕在这些动荡的市场中赔钱，并一直试图通过尽快清算资产来使用他们的财务资源来保护自己。

这导致许多人转换或出售他们的资产，这解释了为什么许多加密货币的[价值开始下降。有了众多的加密应用、钱包和高速互联网，销售加密货币变得更加容易。如果你没有钱兑换成比特币，那么它的价值很可能会继续暴跌。](/encouraging-crypto-functionality-why-the-recent-cryptocurrency-crash-may-improve-the-ecosystems-e5260adcf05d)

# 比特币和投资者

如果你是一名一直持有比特币和其他加密货币的投资者，那么随着时间的推移，你很可能会看到大量的收益和损失。这是因为加密货币的价值随着投资者的买卖而上下波动。

这样做的问题是，除非你有钱兑换成比特币，否则你无法保护自己免受潜在的价值损失。如果你试图交换或出售你的资产来换取现金，那么比特币的价格将继续下跌，因为投资者越来越难找到愿意购买的买家。

这只是投资比特币时市场波动带来的诸多问题之一。主要问题在于，比特币和其他加密货币的价值应该与固定数量的资产挂钩，这意味着无论投资者决定购买或出售多少，都应该对价格产生某种影响。

大多数投资者认为，如果没有人操纵市场，导致 T2 失去信心，目前的局面就不会发生。由于人们需要做大量的工作才能从加密货币中赚到[的钱，许多人认为有人利用了这种情况，正在利用他们的资源操纵加密货币的价值。](/on-the-rocks-how-a-freeze-in-crypto-will-affect-investors-1b0c14db0751)

这个论点的问题在于，就我们所见，没有真正的证据或确凿的证据。事实上，要证明有人一直在操纵市场或者他们是幕后黑手是不可能的。

比特币和其他加密货币与股票没有本质区别，因为两者都具有不可思议的潜力。然而，仅仅因为某样东西可能值很多钱并不意味着它就会值很多钱。

比特币暴跌的真正原因是投资者正在对这种货币失去信心，这意味着他们将尽快抛售手中的资产。这意味着，任何考虑买入人民币的投资者都应该先理解人们为什么要抛售他们的资产，然后再决定投资人民币是否是个好主意。
