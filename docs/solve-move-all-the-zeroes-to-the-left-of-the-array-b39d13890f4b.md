# 问题

给定一个随机正数数组，将所有的零移到数组的左边。

## 情况

1.  必须就地完成(空间有限，即无法分配新数组)
2.  右边非零元素的顺序无关紧要

## 例子

```
Input : arr[] = {0, 1, 2, 5, 0, 3, 0, 3};
Output : arr[] = {0, 0, 0, 2, 1, 3, 3, 5};
```
