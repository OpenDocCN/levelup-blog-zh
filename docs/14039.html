CodeIgniter 4: php spark命令

在CodeIgniter 4中，用CLG来搭建你的基础模型非常简单。您可以在项目的根文件夹中运行php spark命令，并生成代码。

如果我想制作一个带有Model后缀的名为Important的模型，这个命令可以做到:

php spark make:model Important --suffix

CodeIgniter 4: -dbgroup标志

如果您希望您的模型有一个特定的数据库连接组而不是$defaultGroup，您可以在创建模型时在CLG命令中为该组命名为-dbgroup标志。

php spark make:model Important2 --dbgroup sakilaGroup --suffix
