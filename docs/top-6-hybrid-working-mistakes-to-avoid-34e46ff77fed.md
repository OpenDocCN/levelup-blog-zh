# 要避免的 6 大混合工作错误

> 原文：<https://levelup.gitconnected.com/top-6-hybrid-working-mistakes-to-avoid-34e46ff77fed>

## #5.没有意识到潜意识的偏见

![](img/b3ff54c7d535a9b34b09ebe784e8ea7b.png)

信用:Canva

在疫情开始的时候，全球的企业几乎在一夜之间就切换到了远程工作模式。对许多人来说，这是保持漂浮的必要条件。

然而，随着我们从危机中走出来，大多数组织都在考虑将远程工作和办公室工作结合起来的混合方法。

虽然混合模式有许多潜在的好处，但它也有自己的一系列挑战。

大多数人仍在琢磨如何让模型有效——如果做得不正确，它会影响个人的职业发展和团队的生产力。

让我们来看看为了在保持灵活性的同时最大限度地提高员工参与度和工作效率，你必须避免的六个最常见的混合工作错误。

# 1.没有定义基本规则

为混合工作模式建立基本规则对于充分利用它至关重要。否则，你可能会降低生产率，削弱公司文化。

要考虑的最重要的事情之一是你将如何与你的团队沟通。您将使用什么平台进行会议和在线聊天？团队会议的频率和形式是什么？是否会有特定的日子或时间要求每个人都在办公室？

在切换到混合模式时，这些问题中的一些需要以不同的方式重新审视。如果在这些问题上没有明确的指导，员工可能会很快变得孤立无援。

此外，围绕工作时间和截止日期设定期望值也很重要。远程工作的员工需要知道他们何时应该在线。否则，他们可能会觉得自己一直“随叫随到”，他们的工作生活平衡将受到影响。

通过花时间回答这些问题，您可以避免许多使混合工作模式不太成功的陷阱。

# 2.没有保持有规律的时间表

在制定混合工作计划时，有两个要点需要考虑。

*   **首先，定义团队希望在办公室工作的日子** —团队中的每个人都应该在同一天来到办公室，以充分利用这一天。
*   **第二，定义团队需要遵循的工作时间**——员工拥有工作弹性并不意味着他们应该在非正常时间工作或长时间休息。应该有相当多的重叠，每个人都有空，不管他们选择什么样的工作场所。

如果你不保持一个有规律的时间表，你就不能有效地合作。当每个人在不同的时间工作时，协调项目或一起进行重要的讨论将是一个挑战。

同时，当每个人都选择自己的时间来办公室时，你就失去了混合工作的好处。你将被迫通过电话继续讨论，让每个团队成员都参与进来。你的会议将会比每个远程工作的人更没有效率。

# 3.不遵守在线会议规则

当一些团队成员在办公室工作，一些远程工作时，您需要在会议期间格外小心。

如果整个团队都在办公室，就没有必要召开电话会议。另一方面，如果至少有一名成员远程工作，您应该通过电话进行所有的团队会议，以确保每个人都在同一页面上。

建立一些电话会议礼仪也是必不可少的。在会议期间，你应该留意通话中的个人。

例如，我见过房间里的人开始相互交谈。他们很快就忘记了会议桥上还有团队成员。房间里的某人问了一个问题，另一个人开始回答，但没有确保电话中的每个人都听到了问题并能跟上讨论。

通过遵循简单的规则，无论团队成员身在何处，您都可以确保会议富有成效。

# 4.没有建立清晰的沟通渠道

通过混合工作，你将开始在办公室里看到人们的身体。你将有机会进行即兴的面对面交谈。

只要讨论不影响整个团队，这就很好。如果他们这样做了，就会在办公室的人和不在办公室的人之间产生沟通的隔阂。

为了避免这种情况，你需要建立清晰的沟通渠道。您必须为团队成员如何制定决策和沟通制定基本规则。

例如，如果有人亲自与你分享一些重要的事情，确保给其他人发一封电子邮件或即时消息。

如果你必须做出决定，一定要通过电话会议来讨论。避免与楼层上的人讨论，稍后与每个人分享。如果你这样做，人们会觉得被冷落，可能不会与团队决策保持一致。

遵循基本规则可以确保每个人都在同一页上，不会感到被冷落。这也将帮助你避免很多困惑和误解。

# 5.没有意识到潜意识的偏见

在混合工作中，你需要小心无意识的偏见。

当你和一个团队成员坐在办公室时，很容易进行一次随意的谈话或分享一个内部笑话。但是当你和同一个人进行电话会议时，就很难建立融洽的关系。

身体上的接近经常会导致无意识的偏见——你倾向于给予身体上在你面前的人更多的重视和信任。

为了避免这种情况，你需要意识到你的偏见，并有意识地努力平等对待每个人。如果团队成员感到被排斥，你也应该鼓励他们说出来。

创造一个包容的环境，让每个人都感到受到重视，这对于混合工作的成功至关重要。

# 6.没有合理地计划日子

有了混合模式，你将不得不应对不同的工作模式。有些日子你会在家工作，有些日子你会在办公室。为了充分利用它，你需要根据一天的工作模式对工作进行优先排序。

如果你在家工作，你应该专注于在那种环境下能有效完成的任务。同样，当你在办公室时，你应该专注于需要你亲自到场的任务。

例如，如果可能的话，试着把团队讨论安排在每个人都在办公室工作的日子。在讨论过程中让人们在同一个物理空间可以帮助他们有效地参与。这也有助于加快决策过程。

同样，为你在家工作的日子计划你可以独自做的活动。不受干扰的工作时间可以帮助你集中精力，更好地完成任务。

# 最后的想法

虽然有一些潜在的陷阱需要警惕，但遵循这些简单的提示可以帮助确保你的混合工作安排是成功的。

通过建立清晰的沟通渠道，意识到无意识的偏见，并合理规划每一天，你可以创造一个包容的、富有成效的工作环境。

请记住，习惯这种新的工作方式需要一些努力，但好处是值得的。
