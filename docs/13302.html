我们知道，当在MySQL WHERE子句条件中过滤文本数据时，字符串被放在单引号中。在这篇博文中，学习如何在MySQL WHERE子句条件中过滤数字数据…

当在WHERE子句条件中过滤基于数字的数据时，不应该将过滤数据放在单引号中。相反，请指定数值本身:

SELECT name, category_id, last_update
FROM category
WHERE category_id = 1;
