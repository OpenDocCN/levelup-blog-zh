反应打字员

React打字员是一个非常容易使用的打字动画库。

我们先来看看:

yarn add react-typist

然后将库导入我们的应用程序:

import Typist from 'react-typist';

我们用Typist包装我们想要添加动画的组件。
我们可以配置avgTypingDelay来设置每种类型之间的平均延迟时间。
