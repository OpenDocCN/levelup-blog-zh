Lazygit — Git管理

一个简单的git命令终端UI，用Go和gocui 库编写。

— 杰西杜菲尔德/拉兹吉特

lazy git提供的炫酷功能:

  • 轻松添加、提交和推送变更
  • 解决合并冲突
  • 轻松查看最近的分支
  • ..

安装Lazygit 

# Using Homebrew
$ brew install jesseduffield/lazygit/lazygit

使用Lazygit

在git存储库(您的项目)内的终端中调用lazygit。

$ lazygit
