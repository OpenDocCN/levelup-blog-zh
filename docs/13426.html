Node.js是什么？

  • Node.js是一个开源的web服务器环境
  • 它可以在浏览器之外运行JavaScript。

在这篇文章中，让我们使用Node.js编写一个web服务器来服务简单的HTML文件。

要求:安装Node.js

推荐LTS版本。

步骤1:创建要服务的index.html

步骤2:使用http模块编写服务器代码

步骤3:运行服务器

$ node server.js

方法2:使用现有的服务器(http-server)

第一步:安装http-server

$ sudo npm install http-server$ cd node_modules/

$ CD http-服务器

步骤2:在目录:node_modules/http-server/bin下创建一个HTML文件

$ cd bin
$ vim index.html

步骤3:运行服务器

$ node http-server . /* run server in current directory*/
