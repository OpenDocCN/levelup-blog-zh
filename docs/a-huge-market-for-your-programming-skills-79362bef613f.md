## 3D 世界非常渴望自动化，你可以实现它。

![](img/c54511c4d153ab4ef946873d8b43aea6.png)

[斯蒂夫·约翰森](https://unsplash.com/@steve_j?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍照

我对 3D 艺术的迷恋由来已久，可以追溯到十年前，那时我放学后会看安德鲁·普莱斯(Andrew Price)，当一个叫乔纳森·威廉姆森(Jonathan Williamson)的家伙发布了一个新视频时，我很兴奋。

当我不忙着在课堂上玩涂鸦跳跃的时候，我会在我的 iPod Touch 上手动查看网址——它们在哪里。

今天的世界看起来不同了，更加古老和寒冷，但 3D 艺术是生活中少数几件仍然带有这种舒适，怀旧魅力的事情之一。这延伸到编程方面，编写脚本和代码，甚至只是连接节点来制作新材料，都是如此有趣。

这让我想到了这篇文章的主题:3D 艺术的世界可能是我所能想到的程序员寻找练习编码技能并同时让自己有用的最大空间。

让我解释一下:

# 太多的事情是不必要的乏味

就像在常规艺术中一样，许多 3D 都归结为乏味的、手动的和重复的步骤——对我们程序员来说，所有这些都是“危险信号”。如果我不得不做同样的事情两次，那么我就开始考虑自动完成这项任务的方法。

当然，艺术需要人们花费数百个小时来完善微小的细节——但这并不意味着达到这一点的技术层面就一定是低效的。我可以向你保证，如果你做了一个带有“制作酷东西”按钮的程序，人们会点击这个按钮，而不是手动操作。

这里有一个简单的例子:在 Blender 中制作漫画风格的轮廓需要一个艰苦的过程，直到几个月前，你基本上是将 3D 对象翻倍，然后稍微缩小。有了油脂铅笔插件，事情变得简单多了，但我遇到的真正问题是:你必须为场景中的每一个物体都做这个过程。一摞书？这是一个一分钟的过程，重复七次，然后你仍然有书架的其余部分。

没有一个头脑清醒的人会喜欢这种过程——而且很容易用一个很少有人知道的小型宏记录器工具来自动化。

这是许多 3D 任务的现状，同时人们制作插件，只需点击一个按钮就能自动生成随机的废弃房屋。

中间地带是如此之大，以至于任何开发人员都可以找到他们胜任的任务，并且还可以直接帮助其他人。如果你花时间推销你的产品，后一部分会给你一种温暖的模糊感觉，或者是冰冷的现金。很多钱被扔来扔去，很多东西被愉快地免费分享。正如我所说，每个人都有自己的事情。

# 框架都在那里，你只需要带上调味汁

Blender 最棒的一点是它是开源的，基于 Python，为它编写自己的插件非常简单。Blender 带有一个“N 面板”,你可以用 N 键打开它，这是大多数人放置和期望他们插件的地方。

现在，请注意，我远不是这方面的专家，但男孩玩起来很有趣，很有力量。有些人开发的工具对我来说太复杂了，甚至在概念阶段都无法理解——然而，另一些人开发工具时，你会问“为什么没有人首先想到这一点？”

更重要的是:事情远不止于此。你也可以写一个自动化其他过程的工具——想想批量水印图像，在线自动分享你的作品。事实上，这些都是我自己编写的脚本，我相信它们可以帮助其他人，所以当我有时间的时候，我会用它们制作更可靠的产品，并将它们列在我的 Gumroad 商店上。

# 一些还没有人制造的工具的想法

为了让你知道我在说什么，这里有一些还没有人做的东西，但是可以节省很多人很多时间:

*   Blender 界面上的“发布到 Instagram”按钮。其他社交媒体平台也一样。它可以简单到打开一个网页，然后把文件的链接放到剪贴板上，也可以复杂到完全自动化。
*   一个记录搅拌机场景中所有按键的工具，然后以更有效的方式重新创建它——有点像人们用来制作延时录像并发布到 YouTube 的 ProCreate record 功能。
*   一个合成器节点，将整个场景而不是单个对象的 3D 转换为 2D 漫画。这将节省漫画艺术家几个小时的时间。

# 这是我为自己打造的一些工具:

需要更多的想法吗？以下是我为自己构建的自动化工具:

*   几个宏对于单调乏味的任务，有人可以构建一个宏包并出售它。
*   Selenium 脚本允许我在各种平台和子编辑上分享我的作品
*   Zapier 自动化适用于所有具有现成集成的主流平台
*   基于 40 行 Python 代码的自动水印和图像压缩/优化工具。
*   根据文本长度动态调整大小的滑稽演讲泡泡生成器。
*   Streamdeck 键/功能可以在新标签页中自动打开我的所有网站，这样我就可以在早上快速查看邮件，不会错过任何内容。
