网页抓取是从网页中提取数据的过程。它可以用来收集关于特定主题的信息或收集用于研究目的的数据。虽然通常用于商业需求，但刮擦也可以用于[高尚的事情](https://www.judiciary.uk/subject/prevention-of-future-deaths/)甚至图形创作。

# 这是第一次吗？

有一些案例。

一个例子是脸书起诉电力风险投资公司，要求赔偿 300 万美元。Power Ventures 是一款允许客户从一个地方管理其社交网络的应用。当然，即使在 2008 年，它也包括脸书。

脸书向 Power Ventures 发送了停止函，但该公司继续收集数据。因此，有人对 Power Ventures 提起诉讼，声称该公司违反了[计算机欺诈和滥用法案](http://www.law.cornell.edu/uscode/text/18/1030)和 [CAN-SPAM 法案](http://www.law.cornell.edu/uscode/text/15/chapter-103)。这项联邦法律禁止发送含有重大误导信息的商业电子邮件。这个案子最终在庭外解决了。然而，这场战斗的后果是，Power Ventures 确实已不复存在。

另一个使用网页抓取工具的杰出斗士是 Linkedin。2013 年，该公司对 hiQ 实验室提起诉讼。LinkedIn 指控 hiQ 未经授权从其网站上抓取数据。hiQ 认为这些数据是公共信息，因此是公平的游戏。该案件于 2019 年以辩护人的胜利而告终。

这些诉讼说明了围绕网络抓取的法律不确定性。虽然一些公司将其视为收集数据的宝贵工具，但其他公司认为这是一种盗窃形式。随着越来越多的公司越来越依赖数据，我们可能会在未来几年看到更多关于网络抓取的诉讼。

# 可以用什么方法合法刮？

网络抓取是从网站提取数据的有用工具。然而，重要的是以不违反任何服务条款或版权法的方式抓取网站。您应该知道一些不同的方法:

*   公开可用的 API:许多网站通过 API 提供他们的数据，可以用来抓取数据，而不违反任何服务条款。
*   屏幕抓取:屏幕抓取是指从公开的网页中提取数据。这通常被认为是合法的，只要被抓取的网页不在付费墙或登录页面之后。
*   通过 robots.txt 和条款条件抓取允许给机器人使用的数据。一般的经验法则应该是，如果谷歌可以抓取它，你也可以。

总的来说，当这些要求发布给你的时候，准备好服从停止的要求。即使这会让你的生意受损。我们仍然停留在一个集中的互联网上，所以尊重你正在抓取的网站的数据所有权是很重要的。

# 结论

毫无疑问，网络搜集私人数据会给你带来麻烦。即使你设法避免了法律迫害，你仍将不得不面对公众舆论。事实是，大多数人不喜欢在他们不知道或不同意的情况下收集他们的个人信息。如果他们发现你一直在搜集他们的数据，他们很可能会生气和不安。在某些情况下，这可能会导致公众羞辱，甚至抵制你的业务。因此，非常小心地收集什么数据以及如何使用这些数据是非常重要的。否则，你会很快陷入困境。
