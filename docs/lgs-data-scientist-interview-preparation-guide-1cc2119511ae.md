# 面试过程

有三轮面试。在变焦通话中，每一轮大约需要 60-90 分钟。

## 初始筛选轮次

面试流程从电话筛选开始，他们试图了解你的技能和你在数据科学领域的经验。

## 第一轮

他们在第一轮中检查图像处理、对象检测和图像分割的深入知识。我被要求解释卷积神经网络的完整流程。

## 第二轮

这一轮的重点是预先训练的 ImageNet 模型，如 VGG，Resnet 和 Inception。您应该熟悉这些模型的完整架构以及它们工作机制之间的差异。

## 第三轮

在最后一轮，他们通常检查实现部分和一些用例部分。整个过程相当快。他们花了十天时间完成了所有的回合。

## 最后一轮管理

一旦你通过了所有的技术检查。你有机会了解你的经理。在这里，他们通常试图了解你过去的经历，同时他们也向你提供关于项目和团队的想法。

# 资源

youtube 上有很多教程，但我发现大多数都遵循相同的原始资源，即 [Coursera](http://coursera.org/) 。

以下是我推荐的课程清单:

*   一个人必须有帮助我们理解任何算法工作的数学知识。[机器学习的数学](https://www.coursera.org/learn/linear-algebra-machine-learning)
*   许多公司现在都在使用 GAN 来完成他们的任务。下面是我目前所知道的 GANs 的厉害历程。[建立更好的生成性对抗网络](https://www.coursera.org/learn/build-better-generative-adversarial-networks-gans)
*   [吴恩达](https://www.coursera.org/specializations/deep-learning)深度学习专精是必须的。他们有一个卷积神经网络的部分，你应该认真对待计算机视觉的角色。

# 建议

*   数据科学不是导入库。一个人必须擅长任何算法的工作机制。
*   你不需要成为一名职业程序员。基本的 python 知识就可以了。
*   是你的自信赢得了面试官的心。

# 需要编程经验吗？

有编程经验就好。但是，根据我的数据科学面试经验，基本的 python 编程知识就足够了。

# 要关注的主题

由于这个角色与计算机视觉有关，因此必须具备以下方面的知识。

*   卷积神经网络(CNN)工作
*   了解物体探测机制
*   OpenCV 是必须的
*   像 VGG，Resnet，盗梦空间预训练模型的深入知识是必须的。
*   图像分割工作机制
*   提高图像质量的不同方法。
*   基本的机器学习算法知识(bagging，boosting，random forest，xgboost 等。…)
*   使用 Keras 构建神经网络的实践操作。
