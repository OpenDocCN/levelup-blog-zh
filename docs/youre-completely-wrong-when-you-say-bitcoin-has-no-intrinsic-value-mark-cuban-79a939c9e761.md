# 当你说比特币没有内在价值时，你完全错了(马克·库班)

> 原文：<https://levelup.gitconnected.com/youre-completely-wrong-when-you-say-bitcoin-has-no-intrinsic-value-mark-cuban-79a939c9e761>

你会明白为什么尽管价格波动，比特币仍然是最稳健的价值储存手段。

![](img/5ca738db706de23dff7054920d5ccc39.png)

来源— [YouTube](https://www.youtube.com/watch?v=yYCs6B_071w)

通过成功的流媒体公司 Broadcast.com，马克·库班成为了白手起家的亿万富翁和比特币爱好者。

1999 年，雅虎以 57 亿美元收购了该公司，成为他们最昂贵的收购。然而，此举在当时并不受欢迎，现在被认为是历史上最糟糕的互联网收购之一。

就在 1999 年股市崩盘之前，库班通过出售他的雅虎股票期权净赚了 10 亿多美元。

达拉斯小牛队(Dallas Mavericks)的现任老板马克·库班一直直言不讳地批评那些认为比特币没有内在价值的人。

在最近的一次采访中，库班表示，那些声称比特币没有内在价值的人没有抓住要点。

他说，比特币不是由黄金或白银支持的实物资产，但有其他方法来确定其价值。

他认为比特币的价值在于它的实用性，以及使用它进行交易的难易程度。比特币的价值在于其去中心化的特性，这意味着政府和金融机构无法控制它。

> 马克古巴人:
> 
> “去中心化使比特币成为促进全球交易的强大工具，因为它不受传统金融系统的费用和限制。”

# 比特币作为价值储存手段无可匹敌。

马克·库班相信比特币的“价值储存”能力，认为它甚至比黄金更好。

他指出，比特币的算法稀缺性和有限供应将随着需求的增加而增加升值。

库班还认为比特币相对于其他区块链具有显著优势，因为它“作为价值储存手段没有竞争”

他认为，随着其他区块链相互竞争并可能失败，人们会将比特币视为更安全的避风港和“更保守的加密游戏”

库班说，越来越多的人会将它视为最佳的“价值储存”选择，并使其成为“加密的避风港”

# 你不能在水下保持沙滩球太久。

马克·库班认为现在是考虑购买比特币的绝佳时机。

尽管有人批评他对加密货币的乐观立场，但库班坚持认为，他只是在分享他对新兴技术的想法，个人可以对自己的钱做出自己的决定。

他承认，很难预测比特币的价格会跌到多低，但他已经开始慢慢购买，但还没有“全押”

库班还提到，他正在关注 FTX 股市崩盘对其他公司的潜在多米诺骨牌效应。

他表示，以这些价格购买比特币可能是明智的，因为其采用率越来越高，人们越来越接受它作为一种支付形式。

库班表示，这是一个迹象，表明人们正在发现使用它的价值，并满足全球经济的真正需求，我们应该认为比特币和加密不像货币，而更像软件。

> 马克·库班:
> 
> “不要把密码当成货币。把它们想象成去中心化的、安全的、网络化的平台，其功能允许开发对个人和企业具有独特效用的应用程序。”

# 网络泡沫与加密货币的现状有相似之处。

马克·库班认为，随着股市受到通胀担忧的打击，加密行业的崩溃正在发生，这与本世纪初的互联网泡沫相呼应。

在 90 年代末的网络时代，投机性投资导致基于互联网的公司估值过高，因为人们可以看到技术的价值，但它还没有准备好。

在 2001 年和 2002 年大崩盘之前的五年里，纳斯达克上涨了 400%。

> 马克·库班:
> 
> “在令人兴奋的应用程序和 NFT 或不可伪造的令牌最初激增之后，加密正在经历互联网经历的平静期，加密正在进入一个“模仿阶段”，在这个阶段，太多公司未能给市场带来新的实用工具。”
> 
> 库班说，从平静中走出来的区块链将是那些使用“智能合同来提高企业生产率和盈利能力”的国家

# 人们仍然在投机而不是创新

最近，马克·库班表示，创新的现状正处于“模仿者”和“白痴”阶段之间的过渡阶段。

然而，他认为仍有许多改变游戏规则的创新要做。

密码行业的现状类似于互联网的早期。他认为，人们做投机者而不是创新者会犯同样的错误。

> 马克·库班:
> 
> “我最喜欢的一句话是…
> 
> 首先，有创新者，然后有模仿者，然后有白痴，对，所以我认为现在我们正处于模仿者和一点点白痴之间的交叉阶段。
> 
> 但仍有许多创新有待完成，这将改变游戏规则”。

# 最后的想法。

马克·库班对比特币的看法也有反对意见，包括其高波动性、有限的使用案例、缺乏监管和环境影响。

一些批评者认为，这些因素使得比特币不像传统货币那样可靠，也不那么有用。

库班不同意，他认为比特币是促进全球交易的强大工具，并认为由于其有限的供应和分散的性质，比特币作为价值储存手段是无与伦比的。

库班将早期的互联网流媒体与比特币进行了比较，并表示人们试图理解这项技术为什么有用，但看不到我们如何或为什么会使用它。

库班认为现在是考虑购买比特币的绝佳时机，尽管价格可能会波动，因为接受度在不断提高。

你需要决定现在是否是投资和尝试的好时机。
