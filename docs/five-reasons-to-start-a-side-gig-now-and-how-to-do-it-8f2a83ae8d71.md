# **什么是兼职:**

这是你主要工作之外的兼职项目。这是一个赚外快的好方法，但是如果你发现了它的潜力，并且决定全职关注它，这也是一个非常有利可图的方法。

兼职可以是写博客、开发网站、自由职业等等。但我们不要忘记，你可以做传统的兼职，比如优步、Lyft 或任何不在线的工作。

不浪费更多的时间，让我给你 5 个理由，为什么你需要认真考虑有一个侧面的骗局。
