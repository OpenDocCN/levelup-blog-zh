让我们创建一个Marimekko图表来可视化我们的数据

Marimekko图表是一种二维堆叠条形图。我们在y轴上将分类数据分割为百分比，在x轴上有一个额外的数字维度。

这个名字来自芬兰纺织服装品牌 Marimekko 。

我将使用Plotly库创建一个Marimekko图表。

说明

我有一个带有设备类型的标签列表。宽度数组用于x轴。在本例中，它们表示针对相应的设备类型销售了多少不同的型号。数据变量是一个字典，分别包含销售目标和实现情况。

labels = ["Computer","Washing Machine","Screen","Vacuum","TV"]
widths = np.array([30,25,10,15,20])data = {
    "Target": [50,80,60,70,30],
    "Sold": [50,20,40,30,70]
}

在 add_trace 方法中， texttemplate 决定了块中的文本。在我们的例子中，面积是百分比和宽度的乘积。

texttemplate="%{y} x %{width} =<br>%{customdata[1]}",

我们可以定义当鼠标悬停在每个数据块上时将显示什么。

hovertemplate="<br>".join([
            "Device: %{customdata[0]}",
            "Total Products: %{width}",
            "Percentage%: %{y}",
            "Area: %{customdata[1]}",
        ])
