# 什么是编程？

我喜欢把软件工程师或软件程序员看作是把业务需求翻译成一台或多台计算机能够理解的指令的翻译者。有许多不同的语言是专门为不同的问题或计算机系统而构建的，如 Java、C#、SQL、JavaScript 等。事实上，有几百种不同的语言[可以用来开发软件！](https://en.wikipedia.org/wiki/List_of_programming_languages)

大多数编程都是通过在您喜欢的文本编辑器或集成开发环境(IDE)中修改文本来完成的。文本编辑器的一个例子是 Notepad++、Visual Studio 代码、Sublime Text 或 Atom。IDE 在文本编辑器上有额外的功能，如在其中运行代码，设置断点以在某些代码行停止代码执行，以及语法自动完成。IDE 的一个例子是 Visual Studio、Eclipse、IntelliJ 或 RubyMine。

一旦你写好了代码，你需要把它转换成计算机能理解的指令。这个过程在某些语言中被称为编译，你将文本表示转换成计算机或虚拟机可以执行的字节码或二进制码。把二进制代码(1010110)想象成计算机可以通过 CPU(中央处理器)发送的一串开或关的电信号。

最后一步是将那些二进制或字节码指令部署或放置到将运行您的应用程序或程序的一个或多个系统上。对于 web 开发，这通常是 AWS 或其他云环境上的 web 服务器。对于移动开发，这是上传到应用商店，然后让您的最终用户下载应用到他们的手机上运行。

就是这样！写一个文本文件，编译成二进制指令然后上传到服务器。我在这里忽略了一些细节，但通常这是任何软件工程师为一个应用程序或系统所经历的工作流程。

# 节目的类别/流派

如果您想更深入地了解编程世界，可以更深入地了解细节，并为您提供从哪里开始的更多细节。有许多不同类别或类型的节目

**手机开发**

![](img/7bb74f20d633249f34004f7bf098a9d4.png)

照片由 [Bagus Hernawan](https://unsplash.com/@bhaguz?utm_source=medium&utm_medium=referral) 在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄

移动开发旨在为 iPhone 或 Android 手机开发应用程序，然后部署到各自平台的应用程序商店。对于 Android 开发，您将使用 Java 或 Kotlin 进行编程，并利用 Android Studio 作为您的 IDE。一旦你建立了你的应用程序，你可以通过插入你的手机并在物理设备上运行应用程序或者使用模拟器(手机的软件版本)来测试它。在确认一切工作正常(在许多不同的手机上看起来都不错)后，你将把你的代码打包成一个 zip 文件，并提交到 App store 进行审查。一旦它在几天后被谷歌批准，那么它将被列出并可供任何有你支持的手机的人下载。

对于 iOS 开发来说，过程非常相似，你开发你的代码，用手机或模拟器测试，然后提交给苹果公司审查。您将使用 XCode 作为您的 IDE，Objective C 或 Swift 作为您的开发语言。要成为苹果的开发者，你需要支付 99 美元/年，而安卓开发者需要支付 25 美元/年。

**网页开发**

![](img/9bf3f7ee29d389f11c2249ca46548924.png)

丹尼尔·科尔派在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

典型的 web 开发使用后端服务器来运行 API，前端服务器向客户端提供 JavaScript，API 连接到数据库来存储用户数据。有许多框架可以用来快速构建 web 应用程序，其中包括 React、Angular 2 和 VueJS 作为使用 JavaScript 或 TypeScript 的前端框架。后端框架包括。Net Core 使用 C#，Spring Boot 使用 Java 或 Kotlin，Ktor 使用 Kotlin，Express 使用 Node，Ruby on Rails。典型的数据库包括 Postgres、MariaDB、MySQL 和 Microsoft SQL Server。

还有其他技术可以利用，特别是当应用程序与移动应用程序集成或者需要扩展到越来越多的用户时。有时开发人员只关注前端或后端技术，被称为“前端”或“后端”开发人员。跨前端和后端系统构建的开发人员被称为“全栈”开发人员。

# 如何开始编程

*   缩小你想做的编程类型。你想开发移动应用吗？你更愿意建一个网站吗？你有兴趣创造一个游戏吗？做一些研究，直到你能弄明白，甚至尝试用这些语言进行一些基本的编程
*   选择一种语言。一旦你缩小了你想从哪种类型的编程开始，选择一种语言，并开始获得专业知识和经验。
*   在 Youtube 上观看教程，阅读博客文章，搜索 StackOverflow，学习几门 Udemy 课程或阅读书籍。无论你学得多好，都要从那里开始。
*   如果你想转行，可以考虑参加一个编码训练营。我见过许多人在不到一年的时间里，从不太懂编程到找到一份年薪 7.5 万英镑的软件开发工作。
