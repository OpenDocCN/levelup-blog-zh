浏览器和 Node.js 应用程序都使用 NPM 包。NPM 是最大的 JavaScript 库。

在本文中，我们将看看 NPM 上流行的 JavaScript 包。

# [颜色](https://www.npmjs.com/package/colors)

我们可以使用这个库毫不费力地用颜色显示文本。

我们可以用它来设置各种颜色的颜色。背景颜色也可以改变。

有了这个，我们可以在明亮和正常的背景色之间进行选择。

它还可以使文本加粗、倾斜，并应用其他文本样式。

# [dotenv](https://www.npmjs.com/package/dotenv)

Dotenv 是一个有用的库，让我们从`.env`文件中读取环境变量。

它对于直接从操作系统中读取环境变量也很有用。

因此，它非常适合后端应用程序。

我们可以控制如何读取`.env`文件，包括我们是否希望在我们的应用程序中包含空白或`#`。

# [jquery](https://www.npmjs.com/package/jquery)

曾经流行的用于 DOM 操作的 jQuery 库也以节点包的形式提供。

它提供了很多 DOM 操作功能，比如操作类名、动态改变样式等等。

此外，它还有许多实用函数，比如用于发出 HTTP 请求的`ajax`函数等等。

# [打字稿](https://www.npmjs.com/package/typescript)

TypeScript 包允许我们将 TypeScript 支持添加到我们的项目中。

使用 TypeScript 有很多好处。它为 JavaScript 增加了许多功能，比如数据类型注释、数据类型检查，同时保留了 JavaScript 类型系统的灵活性。

因此，我们可以用它创建许多数据类型，如联合类型、交集类型、可空类型、文字类型、泛型类型等等。

# [最小化](https://www.npmjs.com/package/minimist)

`minimist` 库让我们将命令行参数解析成一种我们可以轻松使用的格式。

我们可以调用软件包自带的函数，然后它返回一个 JavaScript 对象，我们可以在程序中使用它。

# [aws-sdk](https://www.npmjs.com/package/aws-sdk)

这是用于节点应用的官方 AWS SDK。它让我们可以在我们的节点应用中使用他们的服务。

AWS 有许多服务，包括文件存储、云托管、图像识别等等。

# [@类型/节点](https://www.npmjs.com/package/@types/node)

这个包允许我们为内置节点模块添加 TypeScript 类型注释。

通过为标准节点模块提供自动完成功能，我们可以利用这一点来帮助我们进行开发。

# [永远](https://www.npmjs.com/package/semver)

semver 是节点应用程序的语义版本解析。

我们可以用它把语义版本名解析成字符串。

此外，我们可以使用它们来比较版本，并获得给定版本字符串的最低版本。

# [redux](https://www.npmjs.com/package/redux)

Redux 是一个状态管理库。它通常与 React-Redux 一起为 React 应用程序创建状态管理解决方案。

我们需要集中的状态管理，因为没有比这更简单的方法来在复杂的前端应用程序的组件之间共享数据。

# [eslint](https://www.npmjs.com/package/eslint)

ESLint 是 JavaScript 的缩写。它检查 JavaScript 项目的语法错误和不良编码实践。

默认情况下，它有许多内置规则，我们可以添加插件来扩展它提供的规则集。

它支持最新的 JavaScript 语法，并且知道不推荐的和不好的语法，所以在开发任何 JavaScript 项目时，它也非常方便。

# [温斯顿](https://www.npmjs.com/package/winston)

Winston 是一个日志记录器，它让我们在 Node.js 应用程序中记录数据。

它可以用于简单的日志记录，它还有许多格式化功能，可以按照我们想要的方式格式化我们的日志文件。

我们可以过滤数据，并将日志记录级别更改为我们想要的级别。

# [节点获取](https://www.npmjs.com/package/node-fetch)

这个库将 Fetch API 引入 node . js。Fetch API 是浏览器中的标准 HTTP 客户端。

它取代了我们之前使用的`XMLHttpRequest`构造函数。

Fetch API 是基于承诺的，支持更多种类的请求，并且可以自动处理响应，不像旧的`XMLHttpRequest`构造函数。

有了这个包，我们可以在节点应用程序中使用它。

# [css-loader](https://www.npmjs.com/package/css-loader)

css-loader 是一个模块，用于将 css 导入功能添加到 Webpack 中。

有了它，我们可以将 JavaScript 代码文件中的 CSS 文件作为字符串导入，并以我们喜欢的方式使用它们。

# 结论

ESLint 是最流行的 linter 之一，用于检查 Node.js 的语法和代码样式问题。

Winston 是 Node.js 应用程序的常用日志程序。

我们还有在命令行中显示颜色的库、命令行参数解析器等等。
