# 取代

Python 解释器附带了一个交互式 shell，我们可以在其中键入代码，并在屏幕上获得返回的结果。

这被称为 REPL，是读取-评估-打印循环的缩写。顾名思义，我们可以在一个无限循环中输入 Python 代码并从中获取输出。

当我们输入一些并按回车键，我们将得到输出结果。然后，我们可以键入更多的命令，并从中获得输出，直到时间结束。

要使用 REPL，[安装 python](https://www.python.org/downloads/) ，然后在终端中键入单词`python`。这就是开始执行 Python 代码所需要做的全部工作。

# 公式

在 Python 中，表达式是任何返回单个值的指令。它们是值与运算符的组合。例如:

```
1 + 2
```

是一种表达方式。

当我们把它输入 REPL 时，我们应该得到 3。

# 错误

如果我们输入翻译不理解的东西，REPL 会显示错误。

如果我们有语法错误或者一些逻辑错误，比如试图调用`None`上的方法，我们就会得到这个。

# Python 运算符

Python 有许多操作符，让我们可以按照自己喜欢的方式计算结果。

下面是数学运算符的列表:

*   `**` —指数运算符(如`2 ** 3`返回 8)
*   `%` —余数运算符(如`20 % 8`返回 4)
*   `//` —整数除法/地板商运算符(例如`20 // 8`返回 2)
*   `/` —除法运算符(例如`20 / 8`返回 2.5)
*   `*` —乘法运算符(例如`2 * 3`返回 6)
*   `-` —减法运算符(如`5 — 2`返回 3)
*   `+` —加法运算符(如`2 + 1`返回 3)

上面的列表按优先级从高到低排列运算符。

Python math 中的运算顺序类似于真正的数学。

像往常一样，如果需要的话，我们可以使用括号来覆盖通常的优先级规则。

例如，我们可以用括号来计算:

```
(5 - 1) * ((3 + 1) / (3 - 1))
```

然后我们得到 8，因为它是同一个`4 * (4 / 2)`,也就是同一个`4 * 2`,也就是 8。

# 整数、浮点和字符串数据类型

Python 中常见的数据类型包括整数、浮点数和字符串。

整数是整数，如-1 和 2。

浮点数是十进制数，如-1.3 和 3.1。

字符串是用引号括起来的文本，如`'foo'`和`'bar'`。它们也可以像`''`一样是空的。它们可以是单引号或双引号，但是我们按照惯例使用单引号。

当我们看到:

```
SyntaxError: EOL while scanning string literal
```

这意味着我们忘记了字符串中的引号。

# 字符串串联和复制

我们可以用`+`操作符将字符串连接在一起。串联字符串就是将字符串连接在一起。

例如，我们可以编写以下代码来连接两个字符串，如下所示:

```
'foo' + 'bar'
```

上面的表达式应该返回`'foobar'`。

我们只能将字符串与其他字符串连接起来。如果我们试图将`+`操作符与一个字符串和一个数字操作数一起使用，我们会得到一个错误。

例如，如果我们写:

```
'foo' + 5
```

我们将得到以下错误:

```
TypeError: can only concatenate str (not "int") to str
```

我们可以使用`*`操作符来返回一个字符串，该字符串具有重复给定次数的给定字符串。

例如，我们可以写:

```
'foo' * 5
```

重复字符串`'foo'` 5 次。

第二个操作数必须是整数。如果我们将第二个操作数改为 0 或负整数，我们将得到一个空字符串。

# 将值存储在变量中

变量是我们可以用来存储一个值的东西。

我们将表达式返回的任何内容存储在变量中。

我们使用赋值操作符，用`=`表示，将右边操作数返回的值设置为左边的变量。

例如，我们可以写:

```
x = 1
```

将值 1 存储在变量`x`中。

然后当我们键入`x`时，我们应该会看到屏幕上显示 1。

## 变量名

变量名只能是一个不带空格的单词。它只能使用字母、数字和下划线字符，并且不能以数字开头。

有效的变量名包括如下名称:

*   `accountBalance`
*   `account_balance`
*   `account_1`
*   `_2`
*   `foo`
*   `ACCOUNT_BALANCE`

其他不像那些的都是无效的。

变量是区分大小写的，所以`foo`和`FOO`不一样。

# 结论

Python 有一个简单的方法来用它的 REPL 测试程序代码，我们可以输入 Python 代码并看到返回的结果。

表达式是可以组合成单个值的代码片段。

常见的 Python 数据类型包括字符串、整数和浮点数。

用 Python 做数学和现实生活中做数学是一样的。这些操作以同样的方式工作。

我们可以通过用`+`操作符连接字符串并用`*`操作符重复它们来操作字符串。

变量是实体，我们可以在其中分配表达式的返回值并存储它们。

变量名区分大小写，只能包含数字、字母和下划线。
