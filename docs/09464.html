一个CSS选择器可以包含多个简单的选择器。在简单的选择器之间，我们可以添加一个关系，这叫做组合子。

CSS中有四种不同的组合子:

  • 后代选择器( 空格 )
  • 相邻同胞选择器(+)
  • 子选择器(>)
  • 通用同级选择器(~)
