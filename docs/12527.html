面向Web开发人员的15+ VS代码扩展🤯

对Visual Studio代码的一些很棒的扩展感兴趣，这些扩展将使您的web开发生活变得更容易？在本文中，我将讨论VS代码的15+扩展。

所以，让我们开始吧！

1。 JavaScript (ES6)代码片段
它提供了JavaScript、Vue、React和HTML代码片段。它是web开发的必备扩展。

2。Live Server
Live Server扩展提供了您的web应用程序在编辑器中的实时预览。对于前端开发人员来说，这是一个方便而有用的工具。

3。
这个扩展执行javascript、CSS和HTML代码的格式化。

4。 更好的评论
这个扩展帮助你创建更人性化、易读的评论。

5。 路径智能感知
这个扩展通过自动补全文件名使开发时间更快。你在语句中输入文件名，它会搜索并给你建议。

6。Git lens
我们几乎每天都在使用Git。Gitlens是增强git功能的visual studio代码插件。

7。 代码时间
这个扩展跟踪你的开发时间，并为你提供有用的统计数据，比如你今天有多少个小时的代码
跟踪和查看进度非常有用(任何人都可以使用这个扩展)

8。 ESLint
它是JavaScript的林挺实用程序。它检查你的代码中的常见错误，并让你在编辑器中知道。这就像一个虚拟的对等体在你写代码的时候验证你的代码。

9。Polacode
像这样的代码片段？我们可以用这个扩展来创建一个漂亮的代码宝丽来并分享它。

10。 CSS窥视
这直接从HTML文件中窥视CSS ID和class，并转到各自的CSS定义。

11。 项目经理
处理多个项目并在它们之间切换是Visual Studio代码中常见的
现象。项目经理使得在VS代码中管理多个项目变得轻而易举。

12。 Rest客户端
REST客户端允许你直接在Visual Studio代码中发送HTTP请求并查看响应。使用这个扩展，我们可以测试API并直接在VS代码中查看它们的响应。

13。 自动关闭标签
该扩展自动添加HTML和XML的关闭标签。

14。 括号对上色器2
有了这个VS代码扩展，你可以对匹配对进行颜色编码，以帮助缓解那个痛点，并致力于使代码本身工作——而不是编辑器。

15。 将JSON粘贴为代码
从JSON、JSON Schema、TypeScript交互生成类型和(反)序列化代码。将JSON/JSON Schema/TypeScript粘贴为代码！
