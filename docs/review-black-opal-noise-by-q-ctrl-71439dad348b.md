# 我以为我知道那是什么…

一旦你开始了量子计算的冒险，用不了多久你就会被引入噪声的概念:你的结果理想地应该是这个和那个，那么所有这些其他的数字是什么？嗯，那是噪音造成的。

在高层次上，噪声是一个容易理解的概念。你甚至可以说这是“环境问题”,就此打住。环境并不完美，这会导致错误。现在，回到你定期安排的线性代数…
