# 1.机器学习 A-Z:数据科学中的实践 Python 和 R

基里尔·叶列缅科和哈德琳·德·庞特维斯的《完全指南》是为那些以前有过编码知识的人准备的。它为**机器学习**领域的顶级工作前景提供了所需的所有准备。

课程侧重于算法的实际应用，而不是科学术语。了解如何使用软件进行模拟、计算机视觉、挖掘数据库、检查和练习系统等等。

课程结束后，你将对数据分析和人工学习工作者使用的方法和策略感到舒适。

*   **线上平台:** Udemy
*   **价格:** $12.99(贴这篇文章的时候。)
*   **等级:**初级/中级
*   **评论:** 4.5/5 (140，605 名评论者)
*   **链接:** [课程网站](https://www.udemy.com/course/machinelearning/)

# 2.吴恩达的机器学习

这是 Coursera 上最受欢迎的机器学习培训课程之一。在本课程中，您将了解最有效的机器学习技术，并获得实践它们并让它们为自己服务的机会。

这描述了机器学习的构造，主要通过深度学习的使用。对于那些想比标准机器学习技术更进一步的人来说，这是一门出色的课程。

*   **在线平台:** Coursera
*   **价格:** $39-$89 每月(这是 Coursera 的一般订阅计划)
*   **级别:**中级
*   **审阅:** 4.9/5 (155，140 名审阅者)
*   **链接:** [课程网站](https://www.coursera.org/learn/machine-learning)

# 3.面向编码人员的机器学习入门

这是最好的趋势之一，提供人工智能领域的课程，旨在教育人们关于机器学习和如何进入这个行业。本课程基于旧金山大学为数据科学硕士项目录制的课程。

所有的知识都是从一开始就设计好的，并在这样做的时候强调理解。对于初学者和经验丰富的初学者来说，有很多不同的选择。

*   **线上平台:**快艾
*   价格:免费
*   **等级:**初级/中级
*   审查:不适用
*   **链接:** [课程网站](https://course18.fast.ai/ml.html)

# 4.通过复数视觉理解机器学习

整个课程提供了一个简短的主题指南，这意味着只有一个简单的信息技术背景。

当你一直在寻找一种方法来解决这个至关重要的问题时，这就是你的课程。Pluralsight 拥有超过 5000 门关于机器学习所有最新主题的非常可靠的课程。

*   **在线平台:** Pluralsight
*   **价格:** $29 每月(这是 Pluralsight 的一般订阅计划)
*   **等级:**初学者
*   **评论:** 4.5/5 (1965 年评论者)
*   **链接:** [课程网站](https://www.pluralsight.com/courses/understanding-machine-learning)

# 5.成为一名机器学习工程师

这个 Udacity 纳米学位课程将帮助你发展前瞻性数据统计和数据分析所需的专业知识。使用机器学习棱镜发现数据研究的开始和结束。了解如何隔离和定义有价值的属性，这些属性可以用来以尽可能好的方式描述数据。

通过这一切，你还将经历几个更相关的机器学习案例研究和真实世界的例子。这个课程是与 AWS(亚马逊网络服务)和 Kaggle 合作创建的。

*   **线上平台:** Udacity
*   **价格:** $399 每月(Udacity 纳米学位项目费用)
*   **等级:**中级/高级
*   **审查:** 4.6/5 (750 名审查者)
*   **链接:** [课程网站](https://www.udacity.com/course/machine-learning-engineer-nanodegree--nd009t)
