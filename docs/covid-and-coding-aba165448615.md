# *COVID 期间的训练营*

*在疫情进行了几个月后，我决定是时候将我的职业生涯转向软件工程了。我知道我不想再花几年时间去获得一个计算机科学学位，所以我开始探索其他选择，并登陆了编码训练营。我研究了我所在地区(休斯顿)所有可用的编码训练营，最终选择了我认为最好的一个，熨斗学校。当我申请，办理入学手续，并被录取时，我非常兴奋。*

*在我申请的时候，学校计划在我们项目开始一周后重新开放。我很高兴能亲自做这个节目，但就在我们开始前几周，一股新冠肺炎热潮席卷了休斯顿地区。不久后，熨斗学校决定延长校园关闭日期，现在实际上已经过了我的项目毕业日期，所以我在熨斗学校的全部经历都将在网上进行！我崩溃了，感觉我将会错过学习“你需要亲身体验”的东西。*

*我彻底考虑过将我的项目推迟到以后开始，在校园重新开放后开始，但对于 COVID 来说，没有人知道那会是什么时候。所以，我决定继续这个在线项目。那是在 zoom 上听课、小组讨论、结对编程项目、测试等的漫长日子。老实说，尽管完全在线，还是决定这么做是我做过的最好的决定。*

*我了解到，有了合适的工具，你可以在网上学习和工作，即使不比面对面的学习和工作更好，也一样好。除了课后的聚会，我不觉得我错过了太多，相反，我真的了解到我更喜欢远程工作！我能够每天抽出一个半小时的通勤时间，用在编码上，甚至更多，使用在线工具进行合作和完成配对作业很容易，当然，你可以在休息时间做一些家务。最重要的是，我学到了即使人们不在你面前，你也可以和他们建立友情和友谊。我可以说，不仅决定去熨斗学校是我做过的最好的决定之一，而且决定去的时候也是我做过的最好的决定之一。否则我不会遇到很多很棒的人。*

# *COVID 期间的职务搜索*

*让我们面对现实吧。找工作总是需要坚持和一致性，但把疫情扔进这个组合中，现在需要一些额外的创造力。当疫情第一次开始时，我们看到招聘人数全面下降，因为公司不知道会发生什么，但现在我们开始看到软件工程领域的招聘人数回升。我相信随着世界变得越来越科技化和远程驱动，这将继续增长，这将需要更多的软件开发。等式的另一部分是有多少科技公司承诺允许员工永久在家工作。这为开发者打开了更多的大门，因为他们不仅仅局限于物理上位于他们区域的公司。*

*就实际的搜索过程而言，除了亲自面试之外，几乎没有什么不同。我们仍然在网上申请，我们可以在网上建立联系，我们现在甚至可以通过视频会议软件在线面试。像任何事情一样，这有它的优点和缺点。网上采访可以让你适应更多的采访，因为他们没有通勤或旅行时间，但有时也很难通过相机推销自己。我相信，公司将继续想出创造性的方法来使这个过程更好，但仍然是在线的，但现在必须这样做。随着我在疫情继续找工作，我会发布更多关于这个的信息。*

# *在 COVID 期间担任软件工程师*

*在 COVID 开始之前，我对了解一些 100%远程技术公司产生了兴趣，这些公司没有任何建筑。我查的最多的三个是*扎皮尔*、 *Gitlab* 和*啊哈！*。我想更多地了解这些公司是如何运作的，总的来说，他们的工作方式实际上有效吗？*

*通过我的研究，我了解到这种远程工作的方法效率极高！它降低了管理成本，减少了员工的通勤时间，并为公司开辟了更广阔的人才库，因为他们可以从任何地方雇用任何人。当疫情开始时，许多科技公司发现自己不得不过渡到这种工作方式，如 Zapier、Gitlab 和 Aha！他们的运作方式几乎没有变化。我相信这永远改变了软件工程领域。大多数科技公司现在都知道，你可以创造伟大的产品，远程管理团队。但是这些公司的员工会怎么想呢？*

*员工们发现自己开始了全新的工作生活，不管是好是坏。尽管不再需要穿职业装，也不再需要通勤去办公室，但我们需要的、有些人甚至赖以生存的人际交往却在减少。尽管我们可以在休息时间做一些简单的家务，但现在我们没有时间和同事、朋友聊天了。*

*这种新的工作方式给每个人带来不同的感受。作为一个曾经鄙视远程编码训练营的人，讽刺的是，我现在正积极地在这些完全远程的公司中寻找工作。我相信这是软件开发的未来，我相信从长远来看，这将允许更好的工作生活平衡。*
