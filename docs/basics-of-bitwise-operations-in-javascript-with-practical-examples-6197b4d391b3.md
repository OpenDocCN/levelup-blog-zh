在计算中，一位(二进制数字的缩写)是计算机可以表示的最小数据单位。它是一个二进制值，只能是 0 或 1。计算机使用位来存储和处理信息，因为它们可以用一系列的 0 和 1 来表示任何类型的数据。

例如，一个位可以表示一个简单的真/假值(1 表示真，0 表示假)，也可以用来表示一个更复杂的数据，如字符或数字。通过使用位的组合，计算机可以表示和操作多种数据类型，包括文本、图像、音频和视频。

一组 8 位称为一个字节。选择一个字节的位数(8)是因为它允许计算机使用相对较少的位数来表示较大范围的值。例如，一个字节可以代表 256 个不同值中的任何一个(2⁸ = 256)，这足以对所有 ASCII 字符(包括字母、数字和符号)进行编码。

在现代计算机中，数据通常以比单个比特或字节更大的单位来处理和存储。例如，一个字通常是一组 2、4 或 8 字节(16、32 或 64 位)，双字通常是一个字的两倍长(32、64 或 128 位)。然而，所有这些更大的单位仍然基于比特的基本单位，并且所有数据最终被表示为一系列的 0 和 1。

按位操作是一种操作类型，它处理存储在计算机内存中的值中的各个位。它们通常用于对数据执行低级操作，如测试、设置或移动单个位的值。以下是 JavaScript 中有用且实用的按位运算的几个例子:

## &(按位与)

该运算符将第一个操作数的每个位与第二个操作数的相应位进行比较。如果两位都为 1，则相应的结果位被设置为 1。否则，结果位被设置为 0。此运算符可用于通过检查最低有效位来确定一个数是奇数还是偶数。例如:

```
const isEven = (n) => (n & 1) === 0;
console.log(isEven(2)); // true
console.log(isEven(3)); // false
```

## |(按位或)

该运算符将第一个操作数的每个位与第二个操作数的相应位进行比较。如果任一位为 1，相应的结果位将被置 1。否则，结果位被设置为 0。此运算符可用于将数字中的特定位设置为 1。例如:

```
const setBit = (n, bit) => n | (1 << bit);
console.log(setBit(0, 2)); // 4
```

## ^(按位异或)

该运算符将第一个操作数的每个位与第二个操作数的相应位进行比较。如果这些位不同，则相应的结果位被设置为 1。否则，结果位被设置为 0。此运算符可用于切换数字中的特定位。例如:

```
const toggleBit = (n, bit) => n ^ (1 << bit);
console.log(toggleBit(5, 0)); // 4
console.log(toggleBit(4, 0)); // 5
```

## ~(按位非)

该运算符反转操作数的所有位。该运算符可用于对数字求反。例如:

```
const negate = (n) => ~n + 1;
console.log(negate(5)); // -6
```

## << (left shift)

This operator shifts the bits of the operand to the left by the specified number of places. This operator can be used to multiply a number by a power of 2\. For example:

```
const multiplyByTwo = (n) => n << 1;
console.log(multiplyByTwo(5)); // 10
```

## >>(右移)

该运算符将操作数的位向右移动指定的位数。该运算符可用于将一个数除以 2 的幂。例如:

```
const divideByTwo = (n) => n >> 1;
console.log(divideByTwo(10)); // 5
```
