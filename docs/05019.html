学习JavaScript:教学的顺序理论

原文：https://levelup.gitconnected.com/learning-javascript-a-sequential-theory-of-instruction-6763009bbe34?source=collection_archive---------23-----------------------#2020-07-29

本文是学习如何用JavaScript编程的系列文章的开始。JavaScript是一种优秀的首次语言，因为它非常有用。JavaScript在大多数人使用的两个主要环境中运行web浏览器和桌面。JavaScript也是一种非常实用的学习语言，因为有很多应用程序和工作需要或者可以使用JavaScript进行编程。出于所有这些原因，以及更多的原因，如果你刚刚开始学习编程，JavaScript是一种优秀的第一语言。

如何学习编程

有成百上千的网站提供学习如何编写计算机程序的教程。许多这类教程都是以经典的“你好，世界！”例如，向学习者展示如何获得短语“你好，世界！”显示在屏幕上，从这里开始，教程直接向学习者展示如何编写代码来做算术，然后是if语句、循环和函数。学习者很快就淹没在代码编写中，可能不理解他们正在编写的代码是怎么回事。

我将要写的系列文章采用了一种不同的方法。下面是我计划对每个主要JavaScript特性遵循的步骤。

步骤1:学习通过变量跟踪读取代码

一个新的编程学生应该学习的第一件事是如何阅读程序并理解它们在做什么。学习代码阅读技巧最好的方法之一就是学会通过一个程序追踪变量的值，这就是所谓的变量追踪。

通过记下程序中使用的每个变量以及该变量在程序的每个步骤中存储的值来跟踪变量。下面是一个简单的JavaScript示例，首先是一个短程序，后面是一个变量trace:

let salary = 2000;
let deductions = .25;
let net_salary =  salary – (salary * deductions);
salary = 3000;
deductions = .30;
net_salary = salary – (salary * deductions);

下面是变量trace:

工资— 2000
扣款— .25
净_工资— 1500
工资— 3000
扣款— .30
净_工资— 2250

当学习者在阅读程序时能够成功地更新变量值时，他们就掌握了成功阅读代码的能力。

当学习者浏览不同的JavaScript结构(if、for、while等)时。)，变量追踪会变得更难，但对学习者来说更有价值。

步骤2:学习编写代码

在学习者成功地学习了如何读取代码和执行变量跟踪之后，下一步是学习使用正确的语法编写代码。这一步包括给学习者一些特定的编程问题，比如单词问题，他们必须将这些问题翻译成可运行的JavaScript代码。

在学习如何编写解决算术问题的代码时，文字问题的一个例子是:

声明一个名为profit的变量，并将其设置为0。声明一个名为revenue的变量，并将其设置为10000。声明一个名为costs的变量，并将其设置为8250。设置profit等于revenue减去costs。打印变量profit的值。

为了让学习者成功地将这个问题转化为一个程序，他们必须理解程序的顺序性质，以及如何形成声明变量和执行算术的正确语法。在执行了几次成功的变量跟踪后，学习者将知道如何去做。

下面是生成的程序:

let profit = 0;
let revenue = 10000;
let costs = 8250;
profit = revenue – costs;
print(profit);

和变量追踪一样，每一个新的构造都给学习者提出了越来越难编写的程序。然而，如果学习的过程中掌握了变量追踪，那么这些新的构造将会通过努力和坚持而被掌握。

步骤3:阅读和理解模板

模板是解决问题的方法或算法。有一套通用的模板，开始程序员必须学会成为这方面的专家。我的文章将通过使用模板作为例子来教授所有的JavaScript构造。

为了演示，这里有一个交换两个变量值的模板:

声明两个变量并赋值。声明一个临时变量，并将第一个变量的值赋给它。将第二个变量的值赋给第一个变量。将临时变量的值赋给第二个变量。

通过向学习者展示代码并让他们跟踪在实现模板的程序中找到的变量来教授模板。下面是变量交换的代码:

let var1 = 23;
let var2 = 12;
let temp = var1;
var1 = var2;
var2 = temp;

变量trace看起来像这样:

var 1–23
var 2–12
temp—23
var 1–12
var 2–23

您也可以像这样进行跟踪:

var 1–23，12
var 2–12，23
温度— 23

步骤4:将模板应用于独特的问题

当学习者用不同类型的变量或不同版本的结构在程序上执行了几次变量跟踪时，学习者就准备好将模板应用于一个独特的问题。这里有一个例子:

莎拉有两个手电筒，一个绿色的，一个蓝色的。绿色手电筒的电池很强，但开口很窄。蓝色手电筒的电池较弱，但开口较宽，更适合大型场所。莎拉需要一个手电筒在她的大后院走动。编写一个程序，将电池从绿色手电筒移动到蓝色手电筒。

程序如下:

let greenFlashlight = "strong";
let blueFlashlight = "weak";
let table = greenFlashlight;
let greenFlashligh = blueFlashlight;
let greenFlashlight = table;

在学习者解决了几个应用模板问题(比如这个)后，他们将准备好继续下一个JavaScript构造。

为什么顺序教学效果更好

华盛顿大学的一群计算机科学教育的学生和教授将这种顺序理论应用于一个初学编程的班级，并用更传统的编程指导来教授另一个班级。

与传统班级的学生相比，使用顺序教学方法的学生在测试中得分更高，写的评估也更好。CSE小组的理论是，因为第一节课是以一种更结构化的方式介绍编程技术的，所以与使用更传统的、分散的方法相比，他们能够更好地处理编程内容。
