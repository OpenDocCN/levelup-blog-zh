对许多人来说，在自己热爱的行业中担任某种职务是梦想成真。

我的目标一直是能够利用我的数学和编码技能来帮助团队和球员实现他们运动的最终目标。

我真的从未想过我会有机会这样做。但不知何故，它发生了，我得到了一份数据科学家的工作，为加拿大奥运团队工作。

我的旅程并不疯狂，但可能会激励和帮助你，所以这里是我如何被聘为奥运会数据科学家的故事。

# 我如何开始从事数据科学

我的数据科学之旅始于大学。我去了麦克马斯特大学，主修数学和统计学。

我擅长数字，喜欢寻找模式。我知道我想用我的技能做点什么，但不太确定那是什么。

直到我遇到了这门课，这门课讲述了一个叫做机器学习的有趣概念。当时我不知道什么是机器学习，但课程描述听起来非常类似于我喜欢在数学课上做的事情。所以我决定参加这个课程，看看它是否适合我。

原来我爱机器学习！我喜欢在数据中寻找模式，并利用这些模式进行预测。

学完那门课程后，我知道数据科学是我的领域。我完全投入进去了。

我开始购买一堆关于数据科学的在线课程。事实上，今天我已经完成了 30 多门课程，包括:

*   结构化查询语言
*   （舞台上由人扮的）静态画面
*   深度学习
*   人工智能
*   熊猫
*   计算机视觉
*   自然语言处理
*   计时器系列

这些课程教会了我比学校更多的东西。我可以在自己的时间里完成所有的事情，这真的很棒，因为我可以在任何时候学习任何我想学的东西。

在参加了所有这些课程并积累了技能后，我决定是时候开始做我自己的个人项目了。

# 创建我的投资组合

我想通过创建自己的模型和项目向自己证明我真正理解了我所学的一切。

我当时不知道的是，这些兼职项目将构成我投资组合的一大部分，向潜在雇主展示我不仅在数据科学方面知识渊博，而且有能力应用我的技能。

但问题是，我只想围绕我感兴趣的东西做数据科学项目。我唯一感兴趣的是运动。

因此，我决定围绕不同的运动创建一系列数据科学项目，看看我能否创建预测可能发生什么的模型。

## 我的运动项目示例

我做的一个项目叫“ ***预测 NHL 进球*** ”。在这个项目中，我收集了所有 NHL 比赛的数据集，并试图预测每场比赛会有多少进球。

我使用了不同类型的回归模型，如多元线性回归和随机森林，以查看哪个模型具有最佳的预测准确性。

我实际上能够创建一个相当准确的模型，我的结果发表在 Kaggle 上，这是一个数据科学竞赛和项目的网站。

我做的另一个项目叫做***重新定义篮球运动员的位置***。在这个项目中，我收集了所有 NBA 比赛的数据集，并试图预测哪个球员在场上打哪个位置。

我通过使用不同类型的机器学习算法来做到这一点，如 k 均值聚类、支持向量机和深度学习。

我真的很喜欢做这些项目！

它们很有趣，围绕着我真正感兴趣的东西。但主要是因为我一直在做这些项目，我在建立一个展示我技能的投资组合，这将帮助我找到我梦想中的工作。

# 求职时间

在我有了一个强大的投资组合后，是时候开始找工作了。

我开始申请多伦多附近的数据科学工作。但问题是，我没有得到任何回应。

我申请了 50 多份工作，只收到两份面试请求。那些是我根本不想要的职位。

我开始变得有点沮丧，因为感觉我已经付出了所有的努力，但我仍然不够好。

但是后来有一天，不知从哪里，我在网上看到一个职位写着 ***“加拿大团队的数据科学家”*** 。

起初，我认为这太好了，不像是真的。但后来我意识到这很适合我。这是一份数据科学的工作，与体育有关。

我申请了这份工作，几周后我收到一封电子邮件，说我被选中参加面试。

我简直不敢相信！

但是现在困难的部分来了。在面试中表现出色。

# 一生一次的采访

我对面试非常紧张。这是我梦想工作的面试。

面试一开始很简单。他们问了我一些关于统计学和不同模型知识的问题，比如线性回归和聚类。

但是后来他们开始问我一些关于我的项目的更具体的问题。

他们想知道一切。我是如何收集数据的，我是如何清理数据的，我使用了什么模型以及为什么，结果是什么，如果我再做一次这个项目，我会有什么不同的做法。

伙计，我给了他们一切！

我详细讲述了我所有的体育数据科学项目。面试官对我印象深刻，我知道这么多，并且能够将我的技能应用到所有这些不同的运动中。

采访持续了大约 45 分钟，最后 15 分钟甚至感觉不像是采访。只有我和桌子对面的人在真诚地谈论体育，以及我们如何利用模特来获得优势。

面试结束后，我非常自信自己做得很好。

几天后，他们打电话给我，给了我这份工作！

我太激动了！这是有史以来最好的感觉。

我所有的努力终于有了回报，我有了一份自己真正喜欢的工作。

# 看看我的工作

现在我打赌你对我作为奥运会数据科学家的工作非常感兴趣。所以让我来分解一下我的日常生活。

通常我的一天从查看最新的体育新闻开始。我想了解所有最新的发展，这样我就可以继续致力于提高我们分析能力的项目。

然后我通常会花几个小时研究新型号或改进旧型号。

我创建的一些主要模型根据运动员的平均表现排名、比赛中的奖牌百分比以及前 5 名比赛的平均分数等变量来预测运动员在他们的运动中的奖牌百分比。

我还创建了一些模型，显示了一名运动员参与奖牌争夺所需的预测年数。

我发现有趣的是，我正在创建的模型与我在空闲时间做的没有什么不同。我还只是在创造模型，在给定特定体育变量的情况下，预测体育运动中会发生什么。

这让我的工作变得非常容易和简单。

# 有兴趣成为体育领域的数据科学家吗

如果你像我一样，在我刚刚开始数据科学之旅时，你想像我一样进入体育行业。那么这就是我的建议:

1.  开始学习教授数据科学基本技能和概念的课程。参加学习 Python、机器学习、数据可视化的课程，以及统计学的基础课程。
2.  通过创建自己的运动相关项目，将你将学到的技能付诸行动。想一想你想做的项目是什么运动，想一想你能从这项运动中获得什么有趣的见解。
3.  从这些项目中创建一个投资组合。要么使用 GitHub 作为展示您的项目的资源库，要么创建您自己的网站，在那里可以找到这些项目的见解。
4.  开始与体育数据专家建立联系。你可以在 LinkedIn 上找到他们。给他们发一条信息，要求进行 10 分钟的快速聊天，或者在你进入体育行业的过程中，他们可以给你任何有价值的建议。这是一件我本应该做得更多，但却没有做并后悔的事。
5.  显然，如果你看到任何与体育数据职位相关的工作，就申请它。即使这不是你想要的确切工作，最困难的部分是进入体育行业，之后你可以过渡到你想要做的实际工作。
