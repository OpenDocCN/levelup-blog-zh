通过实现编程模板学习Python

在这篇文章中，我将介绍一系列文章，这些文章将呈现一种不同的方式来教授初级Python编程。我的方法是将语言作为一种实现一组编程模板的方法来教授，这些模板描述了程序可以执行的特定功能，而不是特定结构的语法。

大多数计算机编程教科书都是围绕语法和编程结构组织的。一本教科书将从如何形成变量开始，转到形成语句，通常是通过引入算术，然后转到输入和输出，然后是分支，然后是循环，然后是函数，最后是一两个关于数据结构的章节。提出问题，但主要是作为展示手边的编程结构的一种手段。

通过编程模板引入构造，学习者明白了为什么需要一个特定的构造来实现那个模板，它被设计来完成一些工作。学习者被教授计算机编程应该被教授的方式，作为解决问题的手段，而不仅仅是语法形式的集合。

定义的编程模板

Michael Clancy和Marcia Linn在他们的教科书Designing Pascal Solutions:A Case Study Approach中，将编程模板定义为“可重用的编程知识抽象”编程模板是描述如何完成常见编程任务的编程知识模式。

编程模板的集合

以下是我打算用来教授初级Python编程的编程模板。

  • 提示，然后读取:使用这个模板将数据放入变量。
  • 输入、处理、输出:这个模板描述了大多数计算机程序。
  • 输入一个，处理一个:使用一个循环从用户那里得到一组输入。
  • 做指定次数的事情:使用for循环重复一组语句。
  • 从备选方案中选择:使用if语句进行选择或基于比较做出决定。
  • 输入并处理，直到完成:使用while循环处理数据，直到遇到sentinel值。
  • 累加值直到完成:使用一个循环来输入一些数据，将值累加到一个变量中，比如求一个平均分数。
  • 填充列表:使用一个循环将一组值放入列表中。
  • 处理列表中的每个元素:使用for循环访问列表中的每个元素。
  • 搜索列表:使用循环顺序访问列表元素，直到找到指定值或到达列表末尾。
  • 插入列表:在已有的值列表中插入一个新值，比如插入一个数字，保持列表的升序。
  • 复制列表:将一个列表中的值复制到另一个列表中。
  • 复制列表时插入:将排序后的数据列表复制到另一个列表中，在正确的位置插入新值以保持排序顺序。
  • 处理一个文件的元素，直到完成:打开并访问一个文件的所有元素，例如找到文件中存储的一组值的平均值。
  • 插入文件:类似于插入列表，但这次数据存储在文件中。
  • 划分和处理元素:对一组排序后的数据进行二分搜索法。

使用编程模板教学

这是一个我将如何使用编程模板教Python的例子。对于这个例子，我将演示如何使用输入、处理、输出模板。

第一步是提出一个问题让学习者去解决。这里有一个问题示例:

从用户那里获得一组三个测试分数。将测试分数相加，然后将总和除以测试分数来计算平均值。向用户显示平均值。

第二步是将编程模板呈现为伪代码。对于输入、处理、输出，伪代码可以是:

输入所有需要的值
处理所有值
输出处理的结果

下一步是向学习者展示生成的程序。下面是这个程序的一个可能的Python版本:

# Input
numGrades = 3
grade1 = int(input("Enter the first grade: "))
grade2 = int(input("Enter the second grade: "))
grade3 = int(input("Enter the third grade: "))
# Process
total = grade1 + grade2 + grade3
average = total / numGrades
# Output
print("The average of the grades is", average)

注释有助于学习者了解这三个阶段，并且是学习者完成任何练习所必需的。

学习编程模板是很重要的，因为它向学习者展示了他们将会正确处理的几乎每一个计算机程序的问题解决方案的结构。如果他们不知道从哪里开始解决问题，他们只需记住模板的步骤，就可以立即获得解决问题的输入。
