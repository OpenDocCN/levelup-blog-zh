编程训练营之所以吸引人，是因为它们提供了一个机会，让你可以离开目前的工作，去做一些更能激发智力、更受欢迎的事情，坦白地说，是一些薪水更高的事情。(尽管要小心*向你兜售*梦想的训练营，因为那里有很多选择，我建议你做一下研究。)你所要做的就是投入 14 周的时间，不知不觉中，你就会从零开始成为全栈开发人员或软件工程师。当然，这 14 周很有挑战性，充满了许多 12 小时的工作日，但是如果你像大多数新兵训练营的成员一样，你对代码的热情会帮助你度过漫长的训练营。

然而，那种认为只需要一个新兵训练营就能获得那些应得的“发展资金”的想法在很多情况下都是错误的。那些成功找到工作的人超越了新兵训练营——招聘经理也看到了这一点。幸运的是，我参加了一个编码训练营，不仅讨论了下面的要点，还在项目期间和之后重申了这些要点。这些可能一开始感觉不舒服或不容易，但是相信我，一旦你开始找工作，它们就值得你去做。

# **下面是我希望在从编码训练营毕业前知道的 5 件事:**

## 1.知道如何编程不会让你找到工作，但人际关系网会

对于我们中的许多人来说，陷入了在 LinkedIn 或你喜欢的工作论坛上一份又一份地发送简历的数字游戏中，很少得到雇主的回复会令人沮丧。知道如何编码是不够的，事实上，我敢说，这只是获得职位的一半。如果你被困在这个可怕的“简历黑洞”中，人际关系网将极大地提高你在数字游戏中的运气。

根据 [Payscale](https://www.payscale.com/career-news/2017/04/many-jobs-found-networking) ，“一些专家说[**70%的**](http://money.usnews.com/money/blogs/outside-voices-careers/2014/09/17/dont-believe-these-8-job-search-myths) 人最终能有现在的职位都要感谢人脉。还有人说更像是 [**百分之八十**](http://www.recruitingblogs.com/profiles/blogs/80-of-today-s-jobs-are-landed-through-networking) 甚至 [**百分之八十五**](https://www.linkedin.com/pulse/new-survey-reveals-85-all-jobs-filled-via-networking-lou-adler) 。”雇主更有可能雇用一个能证明其成功的人，而不是一份简历后面的一张随机面孔。事实上，它通常可以为雇主节省雇佣招聘人员或支付工作列表的费用。所以，上 LinkedIn，联系你的朋友和家人，开始社交吧！

## 2.你应该每天(或者尽可能多地)编码

是的，人际关系网是成功的一半，但是一旦你获得了面试机会，你就要了解自己的能力。更重要的是，招聘经理希望看到你已经在编码了；第一个也是最容易看到你是否这样做的地方是你的 GitHub 档案。确保你的 GitHub 贡献用绿色方块填充。当你说你对编码充满热情时，用实际的代码表现出来。证据就在那些绿色的方块里。

第二，要么用，要么丢。编码是一项复杂的技能，在新兵训练营期间，你必须训练你的大脑以一种新的方式思考，这种方式对许多人来说并不自然。如果你不是在编码或练习，你会发现越来越难重新投入进去。编码迫使你去解决问题，随着你的学习，以前的挑战和挣扎变成了你的第二天性。回想一下您旅程的开始，在那里理解像 for 循环这样基本的东西是令人困惑的，但现在与 React.js 中的 hooks 这样的东西相比，似乎是如此基本。您不想失去这一进步。记住，新兵训练营只是你代码之旅的开始。

## 3.是的，你是在和拥有计算机科学学位的人竞争，但是不要让这给你带来冒名顶替综合症

这个行业很棒，因为不需要大学学历。这太棒了，因为它让许多人有机会获得一份比投资一个学位更容易获得的高薪工作。但是，你还是要和有 CS 学位的人“竞争”。

然而，值得注意的是，拥有计算机科学学位并不能神奇地让一个人更有资格。这取决于个人、他们的技能水平(由他们的投资组合和以前的工作证明)，以及他们为准备工作投入的工作量。看到一个有 CS 学位的人成为竞争对手可能会令人生畏，但不要低估你在新兵训练营度过的几周的价值，不要让冒名顶替综合症成为障碍。你被教授了一门特殊的课程来为你的工作做准备。你已经有了基础，所以你只需要诚实地面对自己:我真的没有为工作做好准备吗？如果是这样，复习和练习一些基础知识。

然而，这并不是天真。许多公司仍然只雇佣计算机科学毕业生，但你必须提醒自己，你有资格在一家重视新兵训练营招聘的公司找到一个职位。

## 4.此外，是的，你不需要大学学位，但面试也不是在公园散步

坦率地说:几乎没有人喜欢在专家组面前接受白板挑战。不能使用谷歌，也不能在代码编辑器中输入代码，这很伤脑筋。仅仅因为这个领域不需要大学学位，并不意味着你不具备从事这项工作所必需的技术知识。这是一个技术行业，雇主希望新员工能够完成工作，并具备必要的技术基础。你的简历让你获得了成功，但你需要证明你的技术经验，无论是通过评估还是面试过程中的算法挑战。

同样，参加新兵训练营是不够的，你必须学会如何面试，并确保你能以技术的方式谈论开发和工程。你需要在面试过程中投入时间，从实践面试和失败的面试中学习。

## 5.基本面是你的朋友。不要一开始就被那些要求 X，Y，Z 技术的应用所困扰。

一旦你毕业了，你会很想学习每个人都在谈论的下一个最好的技术或者你一直在工作列表上看到的语言。也许你想学 Python 或者学 TypeScript。

在科技之谷很容易迷路。然而，你想要成为市场中的一个“楔子”,想要在你的新兵训练营中学到的东西上磨练，而不是过度概括。特别是，对于那些学习了 JavaScript(JS)的人来说，首先，把你的 HTML/CSS 和 Vanilla JS 基础知识整理好，然后在你的腰带下得到一个框架。如果你知道 React，如果你想学习新的东西，也许可以开始专注于学习 React 钩子。一旦你打下了基础，就要关注你的学习能力。如果你还没有阅读文档，那就开始阅读吧。学习新技术？用基本的 CRUD 操作建立一个待办事项列表。先从小处着手。

# 结论

你不会奇迹般地找到工作，找工作是一个过程。完成一个新兵训练营并不能保证你马上就能在这个领域找到工作。将你的投资组合、简历、求职信、GitHub 和 LinkedIn 整理好以备申请需要时间和精力。在你得到面试机会之前，你可能会寄出很多申请。

是的，这个过程很辛苦，也很累人，但是如果你坚持交际、编码、巩固你的基础知识，并努力工作，你最终会找到一份工作。旅程是艰难的，但是如果你能通过新兵训练营，你就能通过求职！在任何行业获得第一份工作都不容易，但这个过程是值得的。每个人都在快速进入这个领域是有原因的！
