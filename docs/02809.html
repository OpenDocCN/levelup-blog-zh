我们可以使用免费的公共API来创建实践项目

原文：https://levelup.gitconnected.com/free-public-apis-that-we-can-use-to-create-practice-projects-7cf4dcab9551?source=collection_archive---------14-----------------------#2020-04-05

在软件开发领域，熟能生巧。所以要尽可能多的找方法练习编程。

有了免费的公共API，我们可以通过创建使用这些API的应用来练习编程。

在本文中，我们将研究一些可以使用这些API的实践项目想法。

动物照片应用

我们可以用一些API轻松创建动物照片应用程序。他们为我们提供了可爱动物图片的URL，比如狗和猫，我们可以用它们来创建应用程序。

一些API包括 Cat API ，它需要一个API密钥才能使用。

狗API 是完全开放的，为我们提供狗及其品种的URL数据。

它们也是 RandomCat 、 RandomDog 和random foxAPI，分别提供猫、狗和狐狸的动物照片。

始别。网上为我们提供柴犬狗、猫或鸟的随机图片。

动漫App

我们可以用 AnimeNewsNetwork API创建自己的应用程序来显示动画数据。它为我们提供XML格式的动漫新闻。

这实际上很好，因为它让我们有机会使用库将XML转换成JSON，以便在我们的代码中使用它们，从而在这个过程中学习更多关于JSON和数据的知识。

此外，还有 AniList API，它为我们提供了一个需要OAuth认证的GraphQL。

我们都可以对这两者进行一些练习，以便为使用OAuth或GraphQL的任何工作做好准备，OAuth或graph QL是分别用于身份验证和构建API的越来越常见的技术。

GraphQL很有用，因为它让我们可以控制我们希望在响应中返回哪种数据。REST APIs很难实现这种控制。

如果你是吉卜力工作室动画的粉丝，比如《龙猫》系列，还有一个吉卜力工作室API，它可以在一个API中为我们提供所有的数据。

博物馆应用程序

我们可以构建自己的应用程序来显示来自博物馆的文本和图像。有一个国立博物馆 API，让我们可以通过一个API访问他们的艺术收藏数据。

它需要一个API密钥来访问，所以我们可以毫不费力地使用它。

还有哈佛艺术博物馆 API，让我们可以访问他们的艺术收藏。

如果我们想要访问图标，我们可以使用 Iconfinder API通过注册一个API键来访问它们的图标。

图书应用程序

构建一本书很棒，因为我们可以练习显示书中的数据。我们可以使用 Google Books API来访问他们的大量藏书。

它还使用OAuth，这样我们就可以练习使用OAuth来访问API。

此外，我们可以从开放库 API获取图书数据，该API包含XML或JSON格式的数据。我们可以更改用于访问数据的查询字符串的格式，这样我们就可以练习构建查询字符串并使用它来访问数据。

Trello应用

Trello是简单易用的任务跟踪器。我们可以通过创建power-ups为其添加我们自己的功能，power-ups是使用 Trello API 根据我们的喜好定制其功能的插件。

通过从我们自己的Trello帐户的仪表板设置功能，Trello API允许我们做任何我们允许的事情。

谷歌分析应用

谷歌分析从我们的网站收集数据，因此我们可以跟踪即将到来的流量，并从中获得一些见解。

为了使这个过程自动化，我们可以使用 Google Analytics API来访问显示在我们自己的程序中的相同数据。

然后，我们可以使用API编写自己的程序，根据自己的喜好定制google analytics的功能。

假日应用

日历索引 API为我们获取世界各地的假期和工作日。

数据可以通过API键访问，因此我们可以轻松地获得数据并在自己的程序中使用它。

结论

有了这些API，我们可以用它做很多项目来练习。如果它给我们提供了有用的功能，我们甚至可以自己使用它们。

仅仅因为它们是实践应用，并不意味着如果它们有用，我们就不能使用它们。
