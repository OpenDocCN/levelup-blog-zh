成为后端开发人员的路线图

原文：https://levelup.gitconnected.com/roadmap-to-becoming-a-back-end-developer-f082b24e1a30?source=collection_archive---------11-----------------------#2020-05-14

学习成为一名后端开发人员需要一段时间。

在本文中，我们将探讨成为后端开发人员的途径。我们会专注于我们必须学习的东西。

饭桶

Git之所以成为最流行的版本控制系统，是因为它能够拥有本地和远程存储库。它还可以让人们通过分叉来制作远程存储库的副本。

拉请求也是让人们在合并代码之前检查代码的好方法。

如果我们学会了如何拉和推代码，发出拉请求，合并代码，那么我们就有了一个好的开始。

要了解更多关于Git的信息，我们可以去Git完整指南了解更多。

SSH和命令行基础

SSH代表安全外壳。这意味着我们必须登录到服务器，并知道如何通过运行命令来进行操作。

它们只是一些像文件操作命令，改变.bashrc等命令。这是我们必须知道的。

常见的操作包括操作文件、创建链接、导航文件系统、向命令行添加选项以及编辑文本文件。

HTTP(S)

后端应用程序经常必须与其他API对话，所以我们必须从它们发出HTTP请求。

我们应该知道如何发出HTTP请求、设置请求头、解析响应并使用它们。

此外，它们接受来自客户端的请求，因此我们必须学习如何解析查询字符串、检查HTTP动词，以及解析请求体，以便从客户端的请求中获取数据并得到我们想要的东西。

数据结构和算法

前端开发就是解决问题。数据结构和算法让我们用计算机程序解决问题。

程序由用算法操纵数据结构的代码组成。所以我们必须学习这两者来创建我们自己的程序。

我们必须了解各种算法的速度，这样我们才能创建速度足够快的程序供人们使用。

常见的算法包括排序、搜索、贪婪算法和优化。

Geeks for Geeks 是学习算法的绝佳资源。

字符编码

我们必须学习字符编码，这样我们才能在屏幕上正确显示数据。

最常见的是Unicode，所以我们必须了解字符集中有什么。另一种常见的编码是ASCII。

远程Git主机

我们必须学习如何使用远程Git主机，如Github、Bitbucket或GitLab，这样我们就可以将代码推送到远程存储库。

此外，我们必须知道如何合并代码和分叉库。我们可能还需要管理权限，创建和远程存储库。

数据库ˌ资料库

我们必须学习如何操作数据库以及如何将数据保存到数据库中。

关系数据库是最受欢迎的，因为大多数数据可以很容易地用关系建模。

因此，SQL是操作关系数据库最流行的语言，因为它被设计用来操作关系数据库数据。

NoSQL数据库在数据分析和缓存方面也有一些有限的用途，所以如果我们想要分析数据或使用Redis进行缓存，我们也必须使用它们。

节点. js

Node.js是在服务器上运行JavaScript应用程序的运行时环境。有一些流行的后端框架，如Express，就是为它而设计的。

我们可以用它们来开发自己的后端应用程序，然后使用Sequelize这样的ORM来操作数据库。

对于一个完整的后端解决方案，还有Nest.js，它将所有东西都整合到一个框架中。

计算机编程语言

Python在web开发中也很受欢迎。Django和Flask是Python后端开发的流行框架。

Django是一个全功能框架，具有基本的路由、认证和ORM功能。

Flask是一个具有基本路由功能的基本框架，我们必须自己添加其余部分。

Java 语言(一种计算机语言，尤用于创建网站)

Java和Spring在企业应用中很受欢迎。Spring是一个做所有事情的框架。

它让我们能够创建一个具有全面框架的尖端、高性能后端。

PHP和MySQL

PHP和MySQL仍然很受欢迎。PHP语言已经有了很大的改进，像Laravel这样的流行框架使得PHP开发变得简洁明了。

MySQL仍然是与PHP应用一起使用的最受欢迎的数据库，因为它们经常与WAMP和XAMPP这样的服务器配对在一起。

结论

后端开发有自己的路。成为一名熟练的后端开发人员需要学习很多东西。

这将是未来几年的路线图，因此随着技术的成熟，现在是成为后端开发人员的最佳时机。
