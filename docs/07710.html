开放终端:

  1. 按“命令⌘ +空格键”
  2. 输入“终端”
  3. 按“回车键”

检查默认版本:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
python --version

检查可执行文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
which python

安装自制软件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

安装Pyenv:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
brew install pyenv

安装Pyenv依赖项:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
brew install openssl readline sqlite3 xz zlib

查看Python版本:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
pyenv install --list

安装Python:

  1. 从以下这些说明中找到版本
  2. 复制提供的命令
  3. 将命令粘贴到终端
  4. 按“回车键”
  5. 重复

设定电脑的默认版本:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”

打开主目录:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
cd ~

创建Zsh配置文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
echo "" >> .zshrc

打开Zsh配置文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
open -a textedit ~/.zshrc

编辑Zsh配置文件:

  1. 从这些指令下面复制代码
  2. 将代码粘贴到“文本编辑”中
  3. 单击“文件”菜单
  4. 点击“保存”
# Pyenv initialization
if command -v pyenv 1>/dev/null 2>&1; then
  eval "$(pyenv init -)"
fi

重启Zsh:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”
exec zsh

创建临时目录:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”

打开临时目录:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”

设置目录的默认版本:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到终端
  3. 按“回车键”

检查默认版本:
