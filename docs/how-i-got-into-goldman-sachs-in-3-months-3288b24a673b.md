# 旅行

在高盛之前，我在一家名为 Innovaccer 的初创公司工作。我在考虑换公司，2022 年 1 月，我开始准备。我并没有特别针对高盛，我从未想过我会有机会在这样一家伟大的公司工作。

# 应用

我正在浏览 Linkedin，突然看到高盛有一个职位空缺。我读了 JD，它是我一直在寻找的东西。我知道肯定有很多人申请了，为了在他们中脱颖而出，我需要一份更好的简历和推荐信。我把简历做得更好，让其中一个员工给我推荐。

> 在获得推荐之前，我已经向 20+个人寻求推荐。因此，如果你处于没有人推荐你的情况下，你只要坚持下去，最终会有人回复你的信息并推荐你。

# 等待时间…

申请后，我完全忘记了这件事，两个星期后，我突然接到一个电话，我没有回答(我有一个习惯，不接未知号码的电话)。但过了一段时间后，我意识到我也回了电话。
***是高盛的招聘人员*** 她拿走了我的基本信息，告诉我我将收到一个黑客等级测试链接，并告诉我需要在一周内完成。

# 黑客等级测试

当天就考了。有两个问题，一个是简单的，另一个是中等难度的。我能够在时限内解决这个问题。提交后，我非常兴奋，因为我知道我现在肯定会接到面试电话。

![](img/25c71b30a11f3a8e9fee6b06914c1bfd.png)

# 面谈

我接到一个电话，安排我的采访，我被告知将有 4 轮，每一轮将编码一轮。根据面试官的反馈，我在一天安排了两次面试，在另一天安排了另外两次面试。

# 第一轮和第二轮

每一轮有两个受访者，他们都问了我一个编码问题。第一轮和第二轮总共有 4 个编码问题。问题是简单和中等水平。我能够在没有任何提示的情况下提前解决所有的问题。受访者对我的解决方案非常满意，我也是。他们问了我一些关于 DBMS、OS 和 C++的基本问题。

附注:我不能透露我被问到的确切问题。

## 反馈

前两轮结束后，不到一个小时我就接到了招聘人员的电话。她告诉我，我们已经收到了积极的反馈，我们可以继续进行下两轮。我告诉她我有空，她安排了面试。

# 第三轮

这一轮又有两个受访者，我被问了两个编码问题，这一次他们连续问了我两个问题，更加严厉地盘问了我。他们还问了我许多数据结构和 C++的内部知识，但是因为我对所有的问题都有所准备，所以我能够回答所有的问题。我再次在没有任何提示和时间限制的情况下解决了两个编码问题。受访者似乎很高兴，并对我说，保持在同一个电话，因为下一个采访者将加入同一个电话。

# 第四轮，即管理轮

至少对我来说，管理轮是最可怕的一轮，因为我已经被 4 家公司拒绝了，所以这次我想确保我不遗余力地通过管理轮。经理要求我介绍他，并开始一个接一个地问我问题。他主要关注我的项目、我所做的工作和系统设计。我能够正确地回答大多数问题。他看起来对我的表现很满意。面试一结束，我就给招聘人员打电话，告诉他们面试已经结束了，还需要多长时间才能得到结果。

# 结果

同一天下午 4:00，我接到招聘人员的电话，她告诉我一个我一生中最好的消息，我被录用了。

# 准备

我知道我必须尽快做出改变。这就是我如何在不到 3 个月的时间里准备面试的。

# 编码

我开始在这个著名的网站 Leetcode 和 Interview bit 上练习编码问题。
🔗https://leetcode.com/

🔗【https://www.interviewbit.com/ 

我列出了一些非常重要的话题，并开始着手解决。主题是👇
👉树木
👉图表
👉链接列表
👉散列法
👉地图
👉堆栈和队列

我排除了动态编程，因为与动态编程相关的问题主要是面试中问的标准问题。我每天解决 5 个问题，周末解决更多。这帮助我表明我解决了几乎所有的面试问题。
