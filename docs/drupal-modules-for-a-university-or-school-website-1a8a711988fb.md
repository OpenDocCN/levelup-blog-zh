## Drupal 模块的综合列表，可用于为学校或大学的网站添加丰富的功能

![](img/d64a3eb4597fc6f9916c228662e1f7f3.png)

[由 ADCI 解决](https://www.adcisolutions.com/?utm_source=medium&utm_medium=referral&utm_campaign=medium-story-modules&utm_content=medium-story-modules-1)

大学网站需要丰富的功能。通常，它们由包含不同类型内容的多个页面组成。例如，它可以是一些学生名单、课程、书籍、大学活动时间表、学校新闻或长篇文章。我们将向您展示一份 **Drupal 模块**的列表，您可以将其添加到您大学的网站上，并获得管理用户数据的高级机会。

# [奥皮尼奥](https://www.drupal.org/project/opigno)

该模块是一个核心模块，其本身功能有限，但它提供了灵活的构建模块和 API 来创建一个定制的电子学习系统。此外，Opigno 有多个附加模块，您可以根据自己的期望和目标选择最合适的模块。这意味着公司可以构建自己的简单解决方案，从而缩短创建时间。

如果您想测试和评估 Opigno 模块，您可以使用他们提供基本配置的配置文件。你不需要对 Drupal 7 系统有广泛的工作知识来检查它。

# [客房预订](https://www.drupal.org/project/room_reservations)

房间预订模块是专门为**大学的网站**开发的，这些网站需要管理对自习室的访问和使用。

您可以根据您的要求进行设置，并在短时间内将预订系统整合在一起。

# [参考书目&引文](https://www.drupal.org/project/bibcite)

书目和引文允许您维护、生成和导入书目数据。它还有相关模块，扩展了 BibCite 的可能性:

[书目&引用—迁移](https://www.drupal.org/project/bibcite_migrate) —从书目(biblio)模块迁移您的书目数据。

[书目&引用—替代度量](https://www.drupal.org/project/bibcite_altmetric) —将替代度量徽章添加到 BibCite 引用实体。它允许你在各种社交网络或不同网站上显示你的内容的提及情况。

[谷歌学术](https://www.drupal.org/project/bibcite_altmetric) —提供元标签的数量，以帮助在谷歌学术对学术内容/文章进行
索引。

# [用户进度 API](https://www.drupal.org/project/user_progress)

用户进度 API 模块由宾夕法尼亚州立大学赞助。它被开发用于绘制学生在整个系统中的进度。

此外，这个项目提供了视图集成。这意味着您可以浏览所有收集的数据或将这些信息导出为您想要的任何格式。

# [电子学习](https://www.drupal.org/project/elearning)

电子学习是最适合学生参加在线互动课程的模块之一。电子学习模块的功能之一是跟踪所有用户的结果和答案。你将能够更好地了解学生的需求，并根据这些结果，你可以更快地提出个人建议或给他们一个详细的反馈。

# [证书](https://www.drupal.org/project/certificate)

在证书模块的帮助下，您可以创建和授予 PDF 证书。它有一个用于管理自定义证书模板和映射的界面。此外，您可以选择一个特定的标准，本模块将根据这些标准授予学生证书。

# [课程](https://www.drupal.org/project/course)

此模块允许在您的网站上创建在线课程。您可以添加任何类型的内容和任何数量的已评分或未评分的课程对象。有机会将本模块与上一模块连接起来，在课程结束时创建一个标记化的 PDF 证书。

# [测验](https://www.drupal.org/project/quiz)

这个模块提供了通过 Drupal 创作和管理测验的工具。例如，您可以使用此模块为学生创建一个测试。所有的点将被存储在一个数据库中，你可以随时查看它们。你可以使用许多不同的问题类型。例如，多项选择、一个或长或短的答案、量表等。

# [H5P —创建和共享丰富的内容和应用](https://www.drupal.org/project/h5p)

如果你的网站上有大量的内容，你一定要考虑这个模块。使用 H5P，您将能够创建交互式视频、问题集、演示文稿、抽认卡。此外，您可以为学生生成任务或测试，添加视频以供观看，上传信息丰富的演示等。

# 结论

Drupal CMS 对于大学网站来说是非常棒的。它非常灵活，适合处理大数据和大量内容。
