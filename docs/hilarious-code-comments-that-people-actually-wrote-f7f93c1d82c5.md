## 会让你 LOL 的代码评论

![](img/244c7d5cd799e0521fdfaaf1bfe443e3.png)

布洛克·韦格纳在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的照片

代码注释是描述计算机程序正在做什么的人类可读的描述。一个良好记录的代码和正确工作的代码一样重要。

现在，我们知道代码注释有多重要，另一方面，代码注释也很有趣。

在本文中，我们将浏览一些有趣的代码注释，这些注释实际上是人们在代码中写的。
