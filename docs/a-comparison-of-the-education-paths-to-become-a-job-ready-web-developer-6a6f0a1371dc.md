web 开发就业市场继续以健康的速度增长，许多公司都在开发应用程序——他们需要添加新功能或维护他们的产品。这意味着许多人正在考虑通过任何可能的方式成为网络开发者的可能性或积极尝试。在本文中，我们将探索这些选项，看看哪一个适合您。

有几个选择来学习成为一名网站开发人员的技能。要看你对学习适应的节奏，不需要指导能学到多少，自己愿意投入多少钱。

考虑到这一点，有几个选项可供选择。可能的选择是自学 web 开发，去训练营，去社区大学，或者获得学位。

# 自我学习

自学不花钱，但也是最难的方法。如果你有 web 开发的天赋，这可能是一个不错的选择。网上有很多免费资源，包括 [Mozilla 开发者网络、](https://developer.mozilla.org/en-US/) freeCodeCamp、Medium 博客文章和 YouTube 视频。付费选项包括书籍、Coursera 课程、Udemy、在线学位和类似网站。

如果你想从免费选项中学习，那么 YouTube 上的结构化视频课程是一个很好的起点。他们从教授琐碎的 Hello World 程序到更高级的技术开始。你可以跟着做，然后编写你自己的程序，直到你熟练为止。然而，这意味着你必须在没有指导的情况下学习，没有人会检查你的代码，除非你请一个有经验的开发人员来做。这是免费的，但是你必须自己做所有的事情。

对于非常基础的东西，你也可以去 Codecademy 这样的网站学习大多数流行语言的语法，但内容不足以让任何人做好工作准备。

付费选项包括书籍、Coursera 课程和在线学位等。你可以从书中学到的许多概念在各个地方都可以免费获得，比如 freeCodeCamp，所以很难证明为大多数书付费是正确的。另外，实体书不能用电脑搜索，所以很难找到你想要的。它们很快就会过时，因为网络技术一直在变化。

如果你旁听课程，Coursera 是免费的，但是如果你想要一个证书，coursera 是收费的。课程有视频和文本，使学习更容易。然而，这仍然是自学，所以大多数时间你只能靠自己。Coursera 也提供在线学位，因此这些学位更有声望。

也有许多在线学位。他们比这些选项花费更多，但也更有声望，这可能会根据谁阅读你的简历而加分。不同层次不同程度之间含量不同。但是，一般来说，软件开发的学士学位和硕士学位都有类似的内容。

常春藤联盟和宾夕法尼亚大学等名校在 Coursera 上提供软件开发学位。当然，学费将高达 25，000 美元。一个很棒的免费选项包括来自 [Traversy Media](https://www.youtube.com/user/TechGuyWeb/videos) 的视频，它涵盖了许多 web 开发主题。

# 训练营

编码训练营是私立学校，提供学习 web 开发和其他软件主题的短期课程。他们的质量参差不齐，学费也很高，所以你需要仔细阅读每个训练营的评论，然后再选择一个。

训练营确实提供了更短的完成时间。然而，它大部分时间都忽略了基本的计算机科学原理，因此在面试和工作中取得成功的关键知识可能会丢失。这是需要注意的一件事。

有面对面的和在线的新兵训练营。面对面提供更多的互动和帮助，但在线更灵活。这可能取决于你的 web 开发能力和基本原则。

训练营包括像[应用学院](https://www.appacademy.io/immersive/course?location=san-francisco)这样的地方，它提供 15 周的 Ruby on Rails 训练营，价格为 12000 美元。

# 社区学院

社区大学提供类似训练营的课程，但是可能需要更长的时间。他们提供实用课程，价格比训练营低，因为学费由政府补贴。这意味着你可以花更少的钱获得训练营的好处。还有面对面的全职、兼职或在线选择。

这是一个可行的选择，因为它更实惠，所以风险更低。这也意味着你完成后债务会更少。

不列颠哥伦比亚理工学院的[网络开发文凭](https://www.bcit.ca/study/programs/7540dipma#courses)可能是个不错的选择，因为它有很多课程涵盖软件开发和网络编程的基础。然而，对于国际学生来说，价格有点高，每学期超过 12，000 加元的学费，尽管国内的学费要便宜得多。

# 度

私立和公立大学有许多软件开发学位。价格区间很广，从几千块钱一年到几万块钱一年。他们有理论和实践模块，所以对学习很有帮助。有面对面的和在线的学位。像我们上面探讨的在线学位，提供的支持和互动较少。这对能够自学的人来说很好。

面对面的学位提供了更多互动和获得实时帮助的机会。这对那些可以全日制上学或者愿意慢慢完成学业的人来说是有好处的。

学位也比其他方法更有声望，这可能对一些雇主很重要。然而，在大多数情况下，熟练程度更重要，所以花更多的钱去一个更有声望的学校可能不值得。

本科和硕士都有，可以教你一般的软件开发。如果你没有学位，学士学位是不错的。硕士学位对那些已经获得某个专业学士学位，并想转行从事软件开发职业的人来说很好。

低成本学士学位的一个例子是东佛罗里达州立学院的这个学位，学费为每年 6118 美元。它包括软件开发的所有基本内容，如面向对象编程、web 应用程序编程、数据库等等。

这个来自中乔治亚州立大学的硕士学位有一个软件开发项目，该项目包括数据结构、面向对象编程、软件测试、移动应用和其他课程。对于那些想要快速学习并且已经获得学士学位的人来说，这是一个可行的选择。它也很实惠，一年的学费为 3825 美元。它也比学士课程短得多。

# 结论

学习 web 开发有很多选择。最便宜的选择是从 YouTube 上免费学习或者旁听 Coursera 的课程。一旦你发现你喜欢它，那么社区大学是一个很好的选择，因为你可以以比训练营更低的价格轻松地参加更短的项目或个人课程。它们也是由政府运营的，所以它们可以获得政府补贴来降低当地学生的学费。如果你准备投身于网络开发事业，学位是获得声望和严格学习的绝佳选择。它教授基础知识，也有实践课程。价格差异很大，所以选择你能负担得起的，不要背负太多债务。
