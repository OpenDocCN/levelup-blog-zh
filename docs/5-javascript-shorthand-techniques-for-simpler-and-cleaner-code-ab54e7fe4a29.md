这里有一些我已经融入到日常编码生活中的速记技巧。

# 1.三元运算符

如果我们满足一个条件，让我们的代码遵循一个特定的路径，如果存在其他条件，让我们的代码遵循另一个路径，这是任何开发人员最常用的逻辑。三元运算符使逻辑看起来简单，适合一行。

> 我尽量远离嵌套的三元逻辑，因为它们会使代码难以阅读。

# 2.速记 if 语句

如果我们满足一个条件，你的逻辑要求你做一些事情，你可以跳过花括号，在同一行写你的代码。

# 3.分配默认值

如果预期值为 falsy，我们可以使用 OR 运算符将默认值赋给变量。

# 4.使用&&运算符代替 IF

如果我们满足某个条件，我们可以使用&&操作符，使代码更简单，而不是使用 IF 语句。

# 5.传播算子

我们可以使用 Spread ( `…`)操作符使我们的代码看起来更整洁。
