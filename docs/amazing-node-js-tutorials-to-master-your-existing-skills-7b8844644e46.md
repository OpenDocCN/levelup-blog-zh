浏览器 JavaScript 和 Node.js 都使用 V8 运行时引擎执行代码。V8 翻译 JS 代码，转换成更快的机器码。机器码是计算机无需解释就能理解和运行的低级代码。让我们看看使用 Node.js 的理由[。](https://www.hiddenbrains.co.uk/blog/why-use-node-js-enterprises-solutions.html)

# 为什么是 Node.js？

使用 Node.js 进行开发可以提供更快的开发时间，需要更少的服务器，并且具有无与伦比的可伸缩性。

LinkedIn 是大公司使用 Node.js 的主要例子。它显著提高了他们的开发速度。此外，对于网飞，它将应用程序的加载时间缩短了大约 70%。Medium、New York Times、PayPal 和许多其他公司也利用 Node.js 提供的潜力来改进他们的产品。

这不仅对公司有利，Node.js 开发人员的平均薪酬是每年 92500 美元(T2)。掌握 Node.js 是一项高回报的投资。这里有一些当今最好的教程来提升你的技能。
