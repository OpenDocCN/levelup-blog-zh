4.使用材质添加过渡-用户界面

我们使用的过渡是Material-UI的 Slide API 。我们必须创建两个新的状态变量:slideIn和slideDirection，分别对应Slide的in和direction道具。

in指示如果true组件何时出现，如果false组件何时退出。

direction指示组件从进入的位置。这意味着在不改变direction的情况下，组件从相同的方向进入和退出。这不是我们想要的旋转木马的效果！幻灯片必须从相反的方向进入和退出，以给出在页面上向左或向右移动的外观。因此，我们需要在设置in之间将direction从false更改为true。

我们还需要使用setTimeout以便在下一张幻灯片进入之前，前一张幻灯片有时间退出。
