# 第一——不是 Responsive⁣

你知道手机用户占所有互联网用户的多少吗？嗯，超过 53%。很大，不是吗？全世界超过一半的互联网用户，想象一下！确保你的网站可以在所有设备上运行，这将使你的网站可靠、可信，并让移动用户了解你。

要检查网站响应性:

1.  在**的**谷歌 Chrome 标签中打开**你想要**测试**的网站**。
2.  右键点击**网站**的**登陆页面**，打开**菜单**。
3.  **打开**菜单后，点击`inspect`。
4.  然后点击**`Toggle device toolbar`如下图所示:**
