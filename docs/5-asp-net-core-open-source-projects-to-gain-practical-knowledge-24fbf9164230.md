## 在实践中学习。

![](img/6102d0ae2f315f8b297e8a04d3891bfd.png)

[钳工](https://unsplash.com/@benchaccounting?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍照

从事真实的商业项目有助于软件开发人员通过获得实践知识而成为专业人员。开发人员除了阅读许多关于设计模式的书籍之外，还需要了解如何使用这些模式来解决现实世界中的各种问题。

即使是开发人员从事的最大的项目，通常也不会实现所有现有的方法、模式和软件工程技术。为了尽可能多的学习，一个开发人员需要同时从事多个项目，或者经常改变项目。然而，第一种和第二种选择对我和许多其他软件开发人员都不起作用。

真正的开源项目可以帮助我们！简单地将它们从存储库签出到本地机器，并分析它们的架构、设计和实现细节，就可以扩展开发人员的实践知识。新知识可以应用到另一个项目中，与他人分享，只是让开发人员更聪明。

除了书籍、文章和在线课程之外，我已经为教育目的个人使用了很长时间的不同复杂性的 5 个令人敬畏的开源 ASP.NET 核心项目的列表。

# **1。**无商业

洋葱架构的最流行的开源跨平台电子商务系统。该项目有文档帮助您快速理解其架构、主要设计决策、系统需求、安装步骤和其他方面。

**部分技术:**

*   雷迪斯
*   LINQ 至 DB
*   ASP。网络核心 5.0
*   NUnit、Moq、FluentAssertions

**链接:**[https://github.com/nopSolutions/nopCommerce](https://github.com/nopSolutions/nopCommerce)

# 2.虚拟商业

这是一个微服务架构的电商项目。GitHub 及其网站上提供了架构概述和大量文档。

**部分技术:**

*   雷迪斯
*   码头工人
*   信号员
*   绞刑
*   ASP。网络核心 3.1
*   OpenApi，Swagger
*   实体框架核心
*   XUnit，Moq，自动固定

**链接:**[https://github.com/VirtoCommerce/vc-platform](https://github.com/VirtoCommerce/vc-platform)

# 3.简单商业

另一个模块化的 ASP.NET 核心电子商务系统，易于在本地运行，开始探索它。

*   Serilog

*   虚张声势
*   身份服务器 4
*   ASP。NET MVC 核心 5.0
*   实体框架核心 5.0

# 4.格兰节点

这是一个具有现代技术的高性能电子商务系统，具有良好的代码质量。

**部分技术:**

*   奥达塔
*   码头工人
*   AWS SDK
*   MongoDB
*   自动映射器
*   ASP。网络核心 3.1

**链接:**[https://github.com/grandnode/grandnode](https://github.com/grandnode/grandnode)

# 5.eShopOnWeb

这是一个示例应用程序，不是真正的应用程序。对于那些发现以前的项目太复杂的软件开发人员来说，这个项目是一个好的开始。

eShopOnWeb 示例是一个由 Microsoft 支持的单一部署 Web 应用程序。

**部分技术:**

*   码头工人
*   媒体人
*   JWT 代币
*   ASP。网络核心 5.0
*   Blazor WebAssembly
*   实体框架核心 5.0

**链接:**[https://github.com/dotnet-architecture/eShopOnWeb](https://github.com/dotnet-architecture/eShopOnWeb)
