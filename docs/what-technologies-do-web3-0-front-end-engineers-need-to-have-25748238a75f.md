最近声音越来越大，越来越疯狂。对于我们的前端，需要具备什么技术？

先说一下`Web3.0`是怎么推导出来的

# 1.互联网

先说什么是 web。1989 年，CERN(欧洲粒子物理研究所)的蒂姆·伯纳斯·李领导的一个技术小组提交了一个新的互联网协议和一个使用该协议的文档系统，命名为万维网，简称`WWW` (World Wide Web)，就是我们现在所知道的“互联网”。其目的是使世界各地的科学家能够使用互联网交流他们的工作文件。它使用的技术主要是`HTML, URI, URL, HTTP`等。，可以静态显示网页内容。这就是我们

# 2.Web1.0

一般来说，Web 1.0 指的是 20 世纪 90 年代和 21 世纪初的互联网。说白了就是博客、留言板，以及 AOL、CompuServe 等早期门户网站组成的互联网。在 Web 1.0 上，静态网页被动读取，网页构建协议使用`HTTP, FTP`，等等。在 Web1.0 的情况下，网页的内容是只读的、静态的，类似于杂志，只能阅读，不能修改，不能交互。在 Web 1.0 下，用户只是信息(网页)的消费者，无法与之互动。Web1.0 也是拨号上网，平均宽带 50k。

一般来说，Web 1.0 是只读的、去中心化的。

# 3.Web2.0

而 Web2.0 大概是 2005 年左右。Web2.0 的一般含义是，用户可以在网页上创建和发布自己的内容，主动参与互联网，而不再是简单的被动阅读网页。整个网页的最终资金和控制权仍然被网页的所有者占据。脸书、推特和 YouTube 等社交媒体也是这个时代的产物。当然，用户生成内容以这种方式扩散也造成了自然垄断，这也导致了几个问题:

*   用户数据是集中的
*   用户数据不可移植
*   用户数据被出售

总而言之，Web2.0 是阅读和写作的共存

# 4.Web3.0

Web 3.0 是一个去中心化的 Web，将权力和数据集中在用户而不是一个公司手中。在网络上分发数据包含分散式区块链技术。Web 3.0 这个术语已经存在很多年了，只是在去年才流行起来。有了 Web3，网络是去中心化的，所以没有权力控制它，构建在网络之上的去中心化应用(app)是开放的。分散式网络的开放性意味着任何一方都不能控制数据或限制访问。任何人都可以建立和连接不同的 dapps，而无需中心公司的许可。

Web 3.0 的主要功能包括:

1.  **语义网**——它是 Web 3.0 的关键，让机器更容易处理数据。
2.  **AI**——AI 是影响 Web 3.0 技术普及的主要关键因素之一。它通过海量的 web 数据使机器变得更加智能，以满足用户的需求。
3.  **3D 图形**——web 3.0 因其 3D 技术已经超越了传统互联网，它提供了一个比 2D 更真实的 3D 网络世界。
4.  **无处不在** -同时或无处不在的概念，移动设备的兴起有所增加，很多人可以更容易地随时随地访问互联网。
5.  **开放性和互操作性**是指应用程序编程接口、数据格式、协议的开放性，以及设备和平台之间的互操作性。
6.  **全球数据存储库**是跨程序和网络访问信息的能力。

简而言之:Web 3.0 是读取、写入和拥有 Web 的一切。

# 5.介绍

对于我们这些前端开发人员来说，应该掌握哪些技术或者了解哪些知识？简单来说，web3 开发人员创建了分散的全栈应用程序，这些应用程序依赖于区块链并与之交互。下面简单说一下。

我们先来了解一下什么是专业术语:

*   **Web3** :它是以太坊区块链和你的智能合约之间的连接。
*   以太坊:一个去中心化的开源区块链，允许用户通过创建智能合约与网络互动。它的原生加密货币是以太。以太坊是市值仅次于比特币的第二大加密货币。它是由维塔利克·布特林于 2013 年创建的。
*   智能合约:它们是存储在区块链上的计算机程序，在满足预定条件时运行。智能合同是用可靠性语言编写的。
*   **去中心化**:数据状态不是由中心实体、平台平台或个人收集
*   **区块链**:区块链网络是一种点对点连接，信息在多个设备之间共享，这使得黑客几乎不可能入侵。它是一种记录信息的系统，很难或不可能改变存储在网络上的信息。
*   **Solidity** :一种用于编写智能合同的面向对象编程语言。它用于在各种区块链平台上实现智能合约，最著名的是以太坊。Solidity 的语法类似于 javascript。要理解扎实，最好有 javascript 等编程语言的背景。直接跳到稳定是个坏主意
*   **Dapp** :代表去中心化 app。它们是在一个分散的网络或区块链上运行后端代码(主要用 Solidity 编写的智能合同)的应用程序。Dapps 可以使用 react、Vue 或 Angular 等前端框架构建。
*   **比特币比特币**:世界上第一种广泛使用的加密货币。
*   **Crypto** :又称加密货币，Cryptocurrency，是一种去中心化的数字货币。
*   **NFT** :不可替代的令牌，一种所有权记录在链上的数字资产。
*   **道**:去中心化自治组织，去中心化自治组织。
*   **元宇宙**:一个通过技术手段创造的虚拟世界的概念。
*   **DeFi 分权金融**:分权金融体系。
*   **Token** :可以理解为加密货币、NFT 等数字资产的统称。
*   **GameFi** : Game + DeFi，一款中文连锁游戏，游戏中的金融系统可以通过加密货币和 NFT 映射到现实中。

介绍完以上，我们再来说说区块链对于开发者来说也是如何分类的。主要是核心区块链开发(核心区块链工程师负责区块链系统的架构和安全协议)和区块链软件开发(这些区块链开发者使用核心区块链开发者提供的设计架构创建 Dapps)。

# 6.开始

让我们言归正传。对于我们的前端开发，我们希望开发驻留在区块链并与之交互的分散式应用程序，我们必须使用 web3.js 和 Ethers.js 库。

## 6.1 Web3.js

web3.js 是一个 JavaScript API 库。要让 DApp 在以太坊上运行，我们可以使用 web3.js 库提供的 web3 对象。web3.js 通过 RPC 调用与本地节点通信，它可以与任何公开 RPC 层的以太坊节点一起使用。web3 包含 eth 对象- web3.eth(用于与以太坊区块链交互)和 shh 对象- web3.shh(用于与 Whisper 交互)

## 6.2 添加网站 3

`web3`引入您的项目与我们现有的参考方法相同

*   **Npm** : npm 安装 web3
*   **凉亭**:凉亭安装网 3
*   **Metor** :流星加以太坊:web3
*   **香草** : dist。/web3.min.js

## 6.3 使用

然后您需要创建一个 web3 的实例并设置一个提供者。为了确保不覆盖现有的提供者，比如在使用 Mist 时，它是内置的，您需要检查 web3 实例是否已经存在

```
if (!web3) {
  web3 = new Web3(web3.currentProvider);
} else {
  web3 = new Web3(new Web3.providers.HttpProvider("http://localhost:8545"));
}
```

因为这个 API 被设计为与本地 RPC 节点交互，所以默认情况下所有函数都使用同步 HTTP 请求。如果你想做一个异步请求。大多数函数允许在参数列表后传递可选的回调函数，以支持异步

```
web3.eth.getBlock(48, function(error, result){
  if(!error)
      console.log(result)
  else
      console.error(error);
})
```

## 6.4 乙醚

为以太坊区块链及其生态系统提供了一个小而全的 JavaScript API 库。最初用于 ethers.io，现已扩展为一个更通用的库。功能类似于 web3.js。

特点如下:

*   将私钥保存在客户端，安全可靠
*   JSON wallet 文件的导入和导出(Geth、Parity 和 crowd、sale)
*   从任何契约 ABI 创建 JavaScript 元类对象，包括 ABIv2 和可读 ABI
*   支持通过 JSON-RPC、INFURA、Etherscan 或 MetaMask 连接到以太坊节点。
*   该库非常小(~88kb 压缩；284kb 未压缩)

## 6.5 工具

当然，除了以上所述，我们还将使用许多工具来改进我们的开发

*   Truffle :提供一个开发环境，使用以太坊虚拟机来编译和测试智能合约，在项目中用作构建依赖
*   Remix IDE :编写和使用智能合同的完美环境，通过它我们可以直接从浏览器创建、修改和执行智能合同。更像是一个编辑
*   **meta mask**:Chrome 扩展，允许你从浏览器连接以太坊区块链网络
*   **Ganache** :提供一个本地区块链环境来测试您的智能合约

# 7.建造

我们希望建立一个完整的堆栈`Dapp`如果你想给你的项目添加 UI，react.js、vue.js 或 angular.js 是很好的 javascript 前端框架，因为它们可以很容易地使用 ethers.js 或 web3.js 网络集成与区块链集成。多个平台允许你创建一个完整的 Dapp，而无需编写代码，例如 [Bunz](https://link.juejin.cn?target=https%3A%2F%2Fwww.bunzz.dev%2F) 、 [Dapp builder](https://link.juejin.cn?target=https%3A%2F%2Fdappbuilder.org%2F) 、 [Atra io](https://link.juejin.cn?target=https%3A%2F%2Fatra.io%2F) 、 [Bubble io](https://link.juejin.cn?target=https%3A%2F%2Fbubble.io%2Fblog%2Fdapps-nft-plugin-nexus-bubble%2F)
