完成我的第一个30天创造性编码挑战后，一个艺术家编码者的反思

几个月前，我参加了一个为期30天的编码挑战-# code 11月，你知道，整个11月都在编码！

"11月对开发人员来说是一次挑战，以提高他们的创造力和技能。目标是在11月的每一天构建一段有创意的代码。”

如果你很了解我，你会知道这种事情不是我的事情！一是因为它涉及到在一个月内持续做同样的事情，二是因为它涉及到代码。

我在工作中被介绍给Codevember对布莱恩·罗默和戴夫·里德大声喊出来。如果没有他们两个，我真的无法完成这个挑战！他们分享资源，发起了一个聊天小组，甚至有一个奖品的承诺！我们会把我们每天的创作贴在小组上，我相信，在这个过程中，我们会互相激励和启发。

你可以在这里查看我11月份的作品。在这篇文章中，我反思了我对code 11月挑战赛的准备、过程和收获。

第1部分:找到动机

我从了解什么是真正的创意编码和 生成艺术 开始？作为一名艺术家和程序员，那天我真的被我的发现惊呆了。我心中的艺术家被我看到的一些创作震撼了，我的程序员大脑无法相信那些创作本质上是逻辑、数学和概率的结果。

我想这就是为什么我如此有动力去成功完成这个挑战；我意识到我可以通过代码创造艺术。这真的让我兴奋！！！

第2部分:选择语言

有一段时间，我一直想学习处理，这似乎是探索新平台的绝佳机会。我浏览了一些比较不同语言创造性编码的博客，尤其是对初学者而言。这让我来到了 p5.js 图书馆。

我决定使用P5，因为它拥有熟悉JavaScript的所有处理能力。它附带了一个在线编辑器，非常适合保存和分享我的作品。

第三部分:做心理准备

现在，我尝试了几个月的挑战，但只持续了一周；所以我下定决心要完成这项工作。这包括找出我自己的弱点，以及让自己保持参与的方法。这促使我为这个项目设定一些目标以及一些约束。此外，我事先计划好了整个月的后勤工作，比如除了工作之外还有多少时间可以做这件事，工作中的任何重要截止日期，其他个人事件等等。我知道我会在感恩节周末休假，所以我必须做好心理准备，比预期晚一些完成那些天的工作，也许不会在11月的最后一天结束挑战。

第4部分:设定我的目标

我逐渐意识到，设定目标，尤其是为一个兼职项目设定目标，有助于你专注于手头的任务。因此，对于2019年11月，我从3个简单的开始:

  • 了解创造性编码，发现不同的艺术家。
  • 精通p5.js库。
  • 每天创作数字艺术:)

第5部分:添加一些约束

在这个项目中，比目标本身更能帮助我的是我为自己设定的规则。

  • 在512x512像素的画布上创作一些东西。
  • 艺术品不需要动画。
  • 代码应该包含一些随机性。

虽然这些约束看起来不太重要，不值得一提，但考虑到我的工作风格、编程经验和工作量，它们实际上帮助我保持了正轨并保持了一致性。

第6部分:绘制细节

我没想到，但这个过程中最有趣的部分对我来说变成了素描。在开始的时候，我这样做或者是为了得到正确的数学结果，或者仅仅是为了想象最终的版本。几天后，它成为编码过程中重要的一步。我特别喜欢解读每日提示，并在上面加入我自己的艺术元素。在纸上画出这个想法帮助我集思广益，组织我的代码，了解我的技术局限性。

第7部分:艺术编码

面对这个挑战，我发现，我真的很喜欢编码部分。当然，有汗水和泪水！但总的来说，我学到了很多新的、很酷的东西。

这是一张草图，我只是想把一些圆圈画在一起。偶然看到一个的圆形装箱算法教程，决定试试。当我继续跟踪时，我在某个点上迷失了，无法复制结果。代码是用纯JS写的，我不能完全理解它的逻辑。然后我搜索了一些直接用p5 写的，这样会更好翻译。我可以执行它，但是我的结果没有之前的准确。同样，每张草图都代表了一种不同的技术或创意挑战。无论如何，我从色彩中获得了很多乐趣——为不同的草图和主题挑选调色板。

最后一堆想法…

  • 找到合适的教程需要一些时间。总的来说，我认为我花的时间比我预期的要多——这不是一件坏事，只是要确保你的时间表允许。
  • 我提前做了一些准备，甚至在这个月开始之前，可能是阅读博客，寻找教程，寻找灵感和计划一些草图。(我在下面添加了这些链接的集合。)
  • 如果你错过了一天、两天或一周，不要大惊小怪。回去吧，继续剩下的挑战。重要的是你每天都要尝试新的东西。
  • 我发现了一个由创意程序员组成的令人敬畏的在线社区，有些人像我一样刚起步，有些人背后有大量的工作。作为一名艺术家，发现算法艺术的媒介是一种真正的快乐。
  • 我最突出的学习就是我知道的不多。事实上，即使在一个月的编码之后，我发现我只是触及了表面。我想，没关系。它只是意味着，我还有很多东西要学。

总的来说，我认为我可以成功完成这个挑战，因为我真的很兴奋。我参加是因为我认为这将有助于拓展我的创造力和编码技能(事实也确实如此)。此外，设定目标和了解自己的进步帮助我远离了对自己在一个月结束时会取得什么的不切实际的期望。
