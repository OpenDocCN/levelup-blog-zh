什么是开源？

开放源码或开放源码软件是在互联网上公开提供的代码，世界上任何人都可以访问。任何人都可以在Github平台上发布他们的代码，每个人都可以访问它。

许多人认为开源意味着自由代码或软件。当我说它可以被任何人访问时，并不一定意味着可以使用免费软件。

源代码是在许可下发布的，允许用户查看、修改和发布他们认为合适的软件。有许多许可证都附有代码，告诉你如何使用这些代码。这是适用的，因此您不能将其用于生产。

不管你是一个想学习的开发初学者，你都可以将这些代码用于你的个人项目和学习目的。

开源的好处

  1. 世界各地的人都可以为您的代码做出贡献
  2. 向世界展示你的项目和技能
  3. 减少了大量工作和精力
  4. 创建一个你可以和他人一起成长的社区
  5. 在项目上合作并维护它。

什么是Git和GitHub？

如果我们在谷歌上查看git的定义，它会说:

“Git是一个分布式版本控制系统，用于在软件开发过程中跟踪源代码的变化”

但是，你所说的版本控制系统是什么意思呢？

有没有发生过这样的情况:您更改了代码，由于一些错误，您需要回滚到代码的前一个版本，或者希望在更改代码之前有一个代码的备份，这就是版本控制的意义所在

版本控制系统是一种帮助软件团队管理源代码随时间变化的软件工具。他们在一种特殊的数据库中记录每一个变化。

Git只是一个帮助开发者实现版本控制的工具。Github和Git不一样，它是一个用来托管使用Git的项目的服务。

如何为开源做贡献？

提交、推送、拉取和合并

现在你已经学习了Git和GitHub，你需要做的就是在Github上找到一个项目并探索。如果您发现一个bug，您可以通过简单地分叉项目并创建一个pull请求来纠正它。阅读特定项目的文档，并查看要贡献的预先存在的问题。

找到你的第一个问题并提交一个pull请求可能会很困难，有许多像 issuehub.io 这样的网站可以帮助你找到与你的技能相关的问题，你可以贡献给GitHub，或者只是在GitHub上搜索。

拉取请求可以是任何事情，比如编辑项目文档、重新设计网站、或者发现bug和问题等等。提交第一个拉动式请求的步骤包括:

  1. 登录GitHub并找到项目资源库
  2. 派生项目存储库
  3. 将fork克隆到您的本地存储库中
  4. 向本地存储库添加必要的更改
  5. 创建一个新的分支，提交并将更改推送到您的分支
  6. 创建拉式请求

黑客啤酒节

Hacktoberfest 是一场由数字海洋、英特尔和Dev主办的庆典 于10月1日至10月31日向所有人开放。这是进入开源世界最简单快捷的方式。

规则很简单，你需要做的就是为任何开源项目做贡献，并打开至少4个拉请求，然后赢得一件很棒的t恤和贴纸。拉取请求可以是从编辑文档到修复项目中的问题的任何内容。

而是支持开源。如果你以前从未为开源做出过贡献，现在是开始的最佳时机

现在你已经学习了基础知识，知道如何为一个项目做贡献，是时候学习如何通过你的开源贡献赚钱了，最好的方法是通过GSOC

谷歌代码之夏

谷歌代码之夏 是一个全球性项目，致力于让更多的学生开发者参与开源软件开发。该计划为期3个月，学生在大学假期期间与一个开源组织合作进行一个编程项目。

一旦被接受，谷歌会为开发者的贡献支付一大笔津贴。这个过程仅限于大学生

  1. 注册GSoC
  2. 搜索项目并创建建议
  3. 将您的申请发送到您希望贡献的组织
  4. 为这个项目贡献三个月

成功的谷歌代码之夏申请者将获得3000美元到6600美元的津贴。

开源如何给你带来工作机会？

如果你被GSoC选中，那么你已经证明了自己是一名优秀的软件工程师，这是一个可以在简历上展示的巨大成就，许多公司都准备雇佣在GSoC工作过的开发人员。

GSoC只是一种方式，Github也可以是一个组合，招聘人员可以在其中看到你的项目、技能和工作。您可以在GitHub上发布您的个人项目，并为开源项目做出贡献，这可以展示您的技能。

这不仅创造了工作机会，而且为开源项目做贡献表明你已经为工作做好了准备。

这是大量的信息，但从现在开始会变得更好，你可以用Git和GitHub做更多的事情，比如创建一个组织、托管一个站点、DevOps等。它是开发人员中最受欢迎的开发工具。
