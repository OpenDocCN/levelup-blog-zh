## 一份我最终得到的工作的软件求职信

![](img/c78452b19d20237a7a985b83ec4d8b6b.png)

由[凯特·马凯特](https://unsplash.com/@katemacate?utm_source=medium&utm_medium=referral)在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

> *我的软件求职信几乎总是遵循类似的模式。我以一个友好的开场白开始，提到这个角色，并讲述我相关的个人品质。*
> 
> *记住:* ***我为角色*** *量身定做了求职信的每一部分。公司提到他们想要一个既能胜任新项目又能加入现有项目的人，所以我提出来了。即使我对一个需求没有经验，我至少会表现出对它的兴趣。*

*你好* ***公司*** *。*

我叫特雷弗，我对你们网站上广告的初级软件设计师职位感兴趣。我好奇、热情、有动力、学习能力强，会花额外的时间来完成我想达到的目标。我可以在现有的项目上工作，也可以从零开始建立一个项目，我不会向我周围有更多经验的人学习。

> 然后，我继续解释从技术的角度来看我是如何合格的，并通过提及我使用过的类似技术或我将如何变得熟练来确保解决我缺乏或不精通的技能。这个角色把随叫随到列为工作职责，所以我也承认这一点，并在结尾加入了一点幽默。

我有 Java 的背景，因为这是我大学三年的主要学习内容之一。我过去用 bash 做过一些脚本编写，虽然已经有一段时间了，但我相信我很快就能让自己回到令人满意的水平。至于 SQL，它是我最喜欢使用的语言之一。在我为期七周的项目结束实习期间，由于这是我感兴趣的事情，我的主要精力主要集中在重构和改进我所在公司以前编写的后端和数据库代码上。我用过 Ubuntu、fedora 和一点点 Kali，在基于 Unix 的环境中感觉很舒服。我认为自己是一个高度灵活的人，理解软件项目的规格变化很快，在网络中，快速响应有时是必须的，而且是一天中的所有时间。黑客并不总是尊重我们的晚餐时间。

> 然后我会简单地谈谈我的兴趣和爱好。这不是必须的，但是我相信一个公司想要雇佣一个比“我为代码而活”更有个性的人。

工作之余，我喜欢阅读关于新技术的文章、阅读消遣、练习乐器、提高烹饪技能、练习编程技能和其他与计算机科学相关的技能。

> *我对这家公司了解多少？在这里，如果我没有听说过这家公司，我会做一些调查，告诉他们为什么我喜欢他们正在做的事情，为什么我想成为其中的一员。*

*我以前听说过* ***公司*** *，因为它是主要的网络公司之一，而网络安全是我的兴趣所在，我希望将来能深入研究。我很想成为* ***公司*** *团队的一员。*

> 非常正式的收尾。我感谢公司花时间考虑我，并表示我可以尽快与您进行讨论。

感谢您的考虑，并在您方便的时候随时通过下面的电子邮件或电话联系我:

***my email****@ Gmail . com
555–555–5555*

*最诚挚的问候，
特雷多伊*

这就是我求职信的精髓。应该简洁，不要太长也不要太啰嗦。一封好的求职信的目标是让你未来的潜在雇主相信三件事:

*   你很专业，对这个角色和公司感兴趣
*   你非常适合这个角色和公司
*   你是一个友好的人，他们会愿意与之共事

> 最后一点:我**强烈建议**你阅读自己写的任何求职信，向朋友和家人寻求反馈，并保留一封你满意的求职信作为未来信件的模板。
