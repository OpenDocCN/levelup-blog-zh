LeetCode 75级别1 —是子序列

原文：https://levelup.gitconnected.com/leetcode-75-level-1-is-subsequence-fe5183c02ce9?source=collection_archive---------25-----------------------#2022-07-03

问题

给定两个字符串s和t，如果 s 是 的子序列t，返回，否则返回false。

一个字符串的子序列是一个新的字符串，它是通过删除一些(可以是没有)字符而不打乱其余字符的相对位置而从原始字符串形成的。(即"ace"是"abcde"的子序列，而"aec"不是)。

例子

  • 例1: 
Input: s = "abc", t = "ahbgdc"
Output: true
  • 例2: 
Input: s = "axc", t = "ahbgdc"
Output: false

约束

  • 0 <= s.length <= 100
  • 0 <= t.length <= 104
  • s和t仅由小写英文字母组成。

解决办法

class Solution:
    def isSubsequence(self, s: str, t: str) -> bool:
        if not s:
            return True
        if not t:
            return False
        
        sleft, sright = 0, len(s)
        tleft, tright = 0, len(t)
        
        while sleft < sright and tleft < tright:
            if s[sleft] == t[tleft]:
                sleft += 1
            tleft += 1
        
        return sleft == len(s)

时间和空间复杂性

  • 时间复杂度 : O(t)。给定t是字符串t中元素的数量。
  • 空间复杂度 : O(1)。没有使用辅助存储空间。
