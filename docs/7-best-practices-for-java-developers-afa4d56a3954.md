# 4.避免内存泄漏

内存泄漏通常会导致软件性能下降。由于 Java 自动管理内存，开发人员没有多少控制权。但是仍然有一些标准的实践可以用来保护您的应用程序免受内存泄漏的影响。

*   当查询完成时，总是释放数据库连接。
*   尽可能多地使用`Finally`块。
*   发布存储在静态表中的实例。
