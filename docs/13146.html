在本教程中，我们将使用SwiftUI和AVFoundation创建一个简单的手电筒应用程序。我们走吧！

演示预览

  1. 创建一个名为toggleTorch()的函数。它依靠AVFoundation来工作。device.hasTorch 可以防止没有手电筒功能的设备崩溃。

2.然后，创建一个视图和按钮来触发该功能。

可以先设置一个状态。例如:

@State private var torch = false- OR -
@AppStorage("torch") var torch = false

添加带动作的火炬按钮。

Button(action: {
  torch.toggle()
  toggleTorch(on: torch)
}) {
 .
 .
 .
}

3.(可选)美化视图。
