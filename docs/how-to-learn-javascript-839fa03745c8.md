## 学习 JavaScript 的资源汇编列表

![](img/d111dfbb2a9ad0eb054eee62e76ce806.png)

由 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上的[窗口](https://unsplash.com/@windows?utm_source=medium&utm_medium=referral)拍摄

在今天这个时代，编码变得比以往任何时候都更受欢迎。人们辞掉工作，学习编码，获得前端开发岗位作为第二职业的情况并不少见。

前端开发是一个如此有吸引力的职业，原因有很多:需求高，薪水高，任何人都可以在互联网上免费(或花少量的钱)学习如何编码。

也许学习前端开发最重要的方面是拥有扎实的 JavaScript 技能。大多数前端开发工作面试都涉及繁重的 JavaScript 编码以及对底层概念的深刻理解。

但是学习 JavaScript 可能会令人望而生畏，因为有太多的资源可供选择。因此，我整理了一份帮助我学习 JavaScript 的最佳地点列表。

如果你是前端开发的新手，请查阅[前端开发人员手册](https://frontendmasters.com/books/front-end-handbook/2018/)。这是对前端开发的一个很好的介绍，对相关技术进行了高层次的解释。

# **在线课程**

学习 JavaScript 的最好方法是实践，因此在线课程提供了一个很好的学习环境，要求你完成活动和项目。

许多网站都有免费的课程选项，但通常只需 40 美元或更少，你就可以升级到专业级别，获得更多内容。

一些网站，如 Lynda，甚至提供与你的 LinkedIn 个人资料的整合，这样你就可以在简历中展示你的课程成就。

这是在技术招聘人员面前脱颖而出的好方法。

# **书籍**

就我个人而言，我喜欢阅读技术书籍。你可以在网上免费找到大部分你需要的书。对于初学者，我建议从下面的 JavaScript 和 jQuery 书籍开始，尽管我不建议过于依赖 jQuery，因为它有点过时，而且大多数雇主认为这是一个障碍。

[你不知道 JS](https://github.com/getify/You-Dont-Know-JS)

[JavaScript Allongee](https://leanpub.com/javascriptallongesix/read)

JavaScript jQuery(售价 22 美元)

[雄辩的 JavaScript](https://eloquentjavascript.net/)

# **网站&博客**

Mozilla Developer Network 是查找最棘手的 JavaScript 问题答案的最佳资源之一。我强烈建议通读他们的文档。CSS Tricks 也是一个很棒的网站，可以让你了解最新的 JavaScript。

[MDN](https://developer.mozilla.org/en-US/)

[CSS-招数](https://css-tricks.com/)

# **工具**

下面是一些快速测试 JavaScript(和相关框架)的好工具。它们不需要额外的设置费用，并且是学习课程时很好的工具。

[CodePen.io](http://codepen.io/)

[CodeSandbox](https://codesandbox.io/)

[小提琴](https://jsfiddle.net/)

# **框架&库**

我强烈建议在了解一个框架或库之前，学习 JavaScript (vanilla JS)的所有细节。开发人员经常一头扎进 React 或 Vue 的学习中，却不明白到底发生了什么，所以这些应该被视为 JavaScript 之旅的第二步。

[反应](https://reactjs.org/)

[Vue](https://vuejs.org/)

[有角度的](https://angular.io/)

[余烬](https://www.emberjs.com/)

# **前端开发面试帮助**

一旦你准备好参加面试，下面的网站可以帮助你练习编码技能。这些主要是基于算法的，所以请确保您熟悉常用的排序和搜索算法。

[代码字节](https://coderbyte.com/)

[黑客排名](https://www.hackerrank.com/)

# **会议&聚会**

最后，我强烈建议尽可能经常参加聚会和会议。然而，会议可能会很贵，所以聚会是在你的城市里结识其他 JS 爱好者的好方法。

[JS Conf](https://2018.jsconf.eu/)

[流利](https://conferences.oreilly.com/fluent/fl-ca)

[ForwardJS](https://forwardjs.com/)

[Meetup.com](https://www.meetup.com/)

在开始成为 JS 开发人员的旅程时，要记住的最重要的事情是不要气馁，不要和别人比较。成为专家需要多年的实践和真实世界的经验(在任何领域都是如此)。如果你被严重卡住了，去[堆栈溢出](https://stackoverflow.com/)寻求帮助。

你会有很多没有工作邀请的技术面试，这没关系。但是通过从里到外学习 JavaScript，你将处于一个非常好的位置去获得一份非常棒的前端开发工作。
