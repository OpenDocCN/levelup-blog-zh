# 使用 Vue.js 构建应用程序的优势

> 原文：<https://levelup.gitconnected.com/advantages-of-building-applications-with-vue-js-b2a23e4199c8>

Vue.js 是一个 JavaScript 库，用于使用 MVVM(模型-视图-视图模型)架构模式构建 web 界面。Vue.js 完全是从零开始设计的，可以逐渐被采用。因此，如果你熟悉 HTML、CSS 和 JavaScript，那么 Vue.js 非常容易上手，
在许多情况下，Vue.js 取代了 Angular、Ember 和 React——这是一件好事，并使它与众不同。除此之外，Vue 与其他现代工具和支持库相结合，呈现出完美的单页面应用程序。

![](img/37b731c5d8216de38835ffd314f7e1ea.png)

**1。非常小的尺寸**
JavaScript 框架的成功取决于它的尺寸。尺寸越小，用的就越多。Vue.js 最大的优势之一就是体积小。这个框架的大小是 18-21KB，用户下载和使用它不需要时间。它击败了所有笨重的框架，如 React.js、Angular.js 和 Ember.js。

**2。易于理解和开发应用**
简单是 Vue 基因的一部分。这个框架受欢迎的原因之一是它非常容易理解。由于 Vue.js 结构简单，用户可以很容易地将它添加到自己的 web 项目中。通过这个框架可以开发小型和大型模板。

**3。Vue.js 以其灵活性而闻名，它提供了快速书写和直接从浏览器运行的完美平衡。您可以使用 ES6、JSX、组件、路由、捆绑等构建复杂的应用程序。这使得使用 Vue.js 开发跨平台应用成为首选。它为您提供了对各种环境的访问。**

**4。良好的详细文档记录**
它还提供了详细文档记录的优势，这使得开发人员编写和执行第一个应用程序更加简单。你只需要对开发网页或应用程序的 HTML 或 JavaScript 有基本的了解。这是因为文档特性已经足够全面了。

**5。解决 AngularJS 和 ReactJS 的缺点 Vue.js 的创始人尤雨溪介绍了这个流行的 JavaScript 框架。他创建 Vue.js 的目的是解决两个最流行的 JavaScript 框架——angular js 和 react js——的缺点。
因此，他通过提取 AngularJS 部分开发了一个轻量级解决方案，而没有添加任何其他概念。他希望结合两个框架的最佳特性，这就是他如何开发出新的 Vue.js 的原因。**

Vue.js 采取了基于组件的方法，组件层次结构的单向数据流，虚拟渲染能力，高性能，道具，以及在 web 应用程序中应用程序的状态管理所扮演的角色等。

![](img/323b48a32f37dbd9a3eb2885280e866a.png)

> 结论
> Vue 是一个非常强大的工具，可以在网络上构建丰富的体验，而且框架非常成熟，由社区很好地维护。
> 它在开发者中非常受欢迎，从一项调查来看，2017 年 89%的开发者表示会再次使用该框架。
> Vue 凭借内置的简单性和更多的好处远远超过了他的竞争对手:
> -一个温和的学习曲线，
> -优秀的文档，
> -对开发者来说非常快的编码速度。
