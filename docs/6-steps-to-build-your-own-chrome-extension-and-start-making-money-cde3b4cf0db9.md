# 步骤 1:创建目录结构

在开始编写扩展代码之前，我们需要一个文件夹来创建扩展，这个文件夹应该有一个特定的结构，这样扩展才能工作。

我们主要需要三样东西:

*   清单文件
*   一个图标
*   HTML 文件

# 步骤 2:创建清单文件

清单文件就像是 Chrome 理解和正确加载扩展所需的配置文件。
