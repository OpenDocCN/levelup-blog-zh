在Windows 10上安装CUDA驱动程序和工具包

附有说明和截图的简明指南

扩展指南使用术语和命令的定义来帮助您了解正在发生的事情。

加入NVIDIA开发者计划:

  1. 访问官网。
  2. 点击“立即加入”
  3. 输入电子邮件地址
  4. 点击“下一步”
  5. 点击“创建账户”
  6. 输入用户信息
  7. 点击“创建账户”
  8. 验证电子邮件地址
  9. 点击“提交”
  10. 输入用户信息
  11. 点击“提交”

下载Visual Studio 2019:

  1. 访问官网
  2. 点击Visual Studio 2019旁边的“社区”
  3. 单击“下载Visual Studio”

安装Visual Studio 2019:

  1. 打开下载的文件
  2. 点击“继续”
  3. 检查“用C++进行桌面开发”
  4. 点击“安装”

下载CUDA驱动程序:

  1. 访问官网
  2. 输入图形卡信息
  3. 选择“工作室驱动程序”
  4. 点击“搜索”
  5. 点击“下载”
  6. 点击“下载”

安装CUDA驱动程序:

  1. 打开下载的文件
  2. 单击“确定”
  3. 选择“NVIDIA图形驱动程序”
  4. 点击“同意并继续”
  5. 点击“下一步”
  6. 点击“关闭”

下载CUDA工具包:

  1. 访问官网:[10.0][10.1][10.2][11.0][11.1][11.2]
  2. 按“回车”
  3. 点击“窗口”
  4. 单击“x86_64”
  5. 点击“10”
  6. 单击“Exe(本地)”
  7. 点击“下载”

安装CUDA工具包:

  1. 打开下载的文件
  2. 单击“确定”
  3. 点击“同意并继续”
  4. 单击“自定义(高级)”
  5. 点击“下一步”
  6. 取消选中“NVIDIA GeForce体验组件”
  7. 取消选中“驱动程序组件”
  8. 取消选中“其他组件”
  9. 点击“下一步”
  10. 点击“下一步”
  11. 点击“关闭”

下载cuDNN库:

  1. 访问官方网站
  2. 单击“我同意cuDNN软件许可协议的条款”
  3. 点击“下载cud nn v 8 . 0 . 5(2020年11月9日)，针对CUDA 11.1”
  4. 单击“适用于Windows (x86)的cuDNN库”

打开PowerShell:

  1. 按下“⊞之窗”
  2. 在搜索栏中输入“PowerShell”
  3. 单击“以管理员身份运行”

打开下载目录:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cd $HOME\downloads

解压缩cuDNN库:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
expand-archive cudnn-11.1-windows-x64-v8.0.5.39.zip -destinationpath cudnn-11.1

复制DLL文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cp cudnn-11.1\cuda\bin\cudnn*.dll "c:\program files\nvidia gpu computing toolkit\cuda\v11.1\bin"

复制H文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cp cudnn-11.1\cuda\include\cudnn*.h "c:\program files\nvidia gpu computing toolkit\cuda\v11.1\include"

复制库文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cp cudnn-11.1\cuda\lib\x64\cudnn*.lib "c:\program files\nvidia gpu computing toolkit\cuda\v11.1\lib\x64"

下载TensorRT:

  1. 访问官方网站
  2. 点击“TensorRT 7”
  3. 完成调查
  4. 勾选“我同意NVIDIA TensorRT许可协议的条款”
  5. 点击“TensorRT 7.2.2”
  6. 点击“用于Windows10和CUDA 11.1和11.2 ZIP包的TensorRT 7.2.2”

解压缩TensorRT:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
expand-archive tensorrt-7.2.2.3.windows10.x86_64.cuda-11.1.cudnn8.0.zip -destinationpath tensorrt

复制TensorRT库:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cp tensorrt\tensorrt-7.2.2.3\lib\*.dll "c:\program files\nvidia gpu computing toolkit\cuda\v11.1\bin"

安装图形外科医生:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
python -m pip install tensorrt\tensorrt-7.2.2.3\graphsurgeon\graphsurgeon-0.4.5-py2.py3-none-any.whl

安装通用框架格式:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
python -m pip install tensorrt\tensorrt-7.2.2.3\uff\uff-0.6.9-py2.py3-none-any.whl

安装ONNX GraphSurgeon:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
python -m pip install tensorrt\tensorrt-7.2.2.3\onnx_graphsurgeon\onnx_graphsurgeon-0.2.6-py2.py3-none-any.whl

打开Visual Studio文件:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
start-process “c:\programdata\nvidia corporation\cuda samples\v11.1\4_finance\blackscholes\BlackScholes_vs2019.sln”

编辑链接器输入属性:

  1. 单击“项目”菜单
  2. 单击“属性”
  3. 双击“链接器”
  4. 点击“输入”
  5. 单击“其他依赖项”
  6. 单击向下箭头
  7. 点击“编辑”

添加cuDNN库:

  1. 从下面这些说明中复制库名
  2. 将库名粘贴到Visual Studio 2019中
  3. 单击“确定”
  4. 单击“确定”
cudnn.lib

构建示例:

  1. 单击“构建”菜单
  2. 单击“构建解决方案”

运行示例:

  1. 从下面这些指令中复制命令
  2. 将命令粘贴到PowerShell中
  3. 按“回车”
cmd /k "c:\programdata\nvidia corporation\cuda samples\v11.1\bin\win64\debug\blackscholes.exe"
