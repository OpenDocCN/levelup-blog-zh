谷歌是一个伟大的搜索引擎，作为开发人员搜索解决方案。

在本文中，我们将看看作为开发者如何更好地在 Google 中搜索东西。

# 使用选项卡

我们可以针对不同的目的使用不同的选项卡。例如，我们可以在图像选项卡中搜索虚假图像，并使用网络选项卡来搜索问题的解决方案。

# 使用引号

我们可以用引号将所有内容括起来，以确保整个短语包含在所有搜索结果中。

# 使用连字符排除单词

我们可以使用连字符来排除单词，这样我们就可以删除搜索中不需要的单词。

例如，我们可以搜索`Angular -1.5`，这样我们就可以搜索角度 1.5 的结果。

# 使用冒号搜索特定站点

我们可以使用冒号和它后面的网站域名来搜索我们要找的东西。所以我们可以键入类似于`Angular site:stackoverflow.com`的内容，在堆栈溢出中搜索关于 Angular 的问题。

# 查找链接到其他网页的网页

`link`关键字让我们找到链接到另一个页面的东西。

例如，我们可以键入`React link:stackoverflow.com`来查找所有带有链接到堆栈溢出的单词“React”的页面。

# 使用通配符

星号`*`字符是通配符。我们可以把它们放在任何地方使用。

例如，我们可以编写`Angular * React`来搜索任何介于“Angular”和“React”之间的短语。

# 寻找与其他网站相似的网站

我们可以使用`related`关键字来搜索与其他网站相关的网站。

所以我们可以编写`related:stackoverflow.com`来搜索与堆栈溢出相关的站点，回答问题。

# 谷歌搜索可以做数学

我们可以通过输入可以计算的表达式来进行算术运算。

所以像`1 + 2`这样的算术表达式可以输入，我们得到 3。

# 一次搜索多个单词

OR 运算符允许我们在搜索框中搜索多个短语。例如，我们可以如下使用它:

```
Angular OR React
```

然后我们搜索`Angular`和`React`。

# 搜索一系列数字

`..`操作符可以让我们搜索一个数字范围。例如，我们可以使用`1..5`来搜索数字范围从 1 到 5 的结果。

此外，我们可以通过编写`World Cup ..2006`来搜索世界杯结果，从而搜索单个数字

# 附近的位置

我们可以搜索离我们很近的东西，而不需要明确指定我们的位置。

所以我们可以搜索`developer confereneces nearby`得到附近所有的开发者大会。

# 搜索包含所有文本的结果

我们可以使用`allintext`操作符搜索结果中的所有文本。

所以类似于`allintext:developer confereneces`的东西会搜索结果中所有跟在后面的单词。

# 搜索标题中包含所有单词的结果

标题中的单词都可以一起搜索。我们可以使用`allintitle`操作符来进行搜索。

因此`allintitle:developer confereneces`将搜索标题中同时包含“开发人员”和“会议”的项目。

# 快捷指令

谷歌里有很多快捷方式。

我们可以搜索“天气*邮政编码*”来搜索给定邮政编码的天气。

操作符为我们提供了单词的字典定义。

“时间*位置*”为我们提供了时间，因此它对于我们不知道其他人的时间的远程会议来说很有用。

# 自动更正拼写

谷歌会自动更正拼写错误，这样我们就不用担心打字时的拼写错误了。

我们可以尽可能快地打字，谷歌会照顾 rets。

# 查找特定类型的文件

这有利于搜索特定类型的文件。例如，我们可以通过使用如下的`filetype`操作符来搜索伪造的 PDF 文件:

```
dummy filetype:pdf
```

然后，我们可以搜索用于测试的虚拟 PDF 文件。

# 结论

我们可以利用谷歌的这些技巧，让我们在搜索东西时变得更加轻松。

开发人员必须一直在网上搜索东西，所以这些技巧会有很大帮助。

有一些操作符可以搜索任何内容，还有很多没有在这里介绍。
