# C/C++

C 编程语言是几乎所有低级软件组件的基础语言。C 的抽象更接近硬件，C 为程序员提供了一个极简语法(有 32 个保留关键字)。与其他现代流行的编程语言相比，c 的语法更接近于汇编语言。因此，C 编译器可以高效地将 C 源代码翻译成机器语言，并生成轻量级且速度极快的二进制可执行文件。

C++是 C 的扩展——所以你可以用 C++的特性来弥补 C 缺少的现代语言特性(例如:类、名称空间等)。).学习 C/C++对任何程序员都有很多好处。C/C++促使你编写优化的代码，因为 C/C++不提供自动垃圾收集。c 提高了你解决问题的能力和基本的计算机科学技能，因为它没有提供预先构建的数据结构和功能齐全的标准库。举个例子，如果你在 C 语言中需要一个栈，你必须自己构建。

同样，学习 C/C++对提高计算机科学知识和技能有很大帮助。下面这个故事解释了程序员为什么要先学习 C 语言。

# **巨蟒**

Python 是一种开发人员友好的、简单的、动态类型的解释编程语言。许多 web 开发人员、数据科学家、机器学习工程师和系统管理员在日常编程任务中使用 Python。Kivy 项目也支持 Python 进入移动开发者社区。有时候，写一个 Python 源代码比写一个伪代码实现一个特定的算法要快。

Python 提供了许多处理数据结构的内置函数。另外，Python 是处理数据记录最简单的编程语言之一。学习 Python 对所有程序员都有好处。Bash 非常适合自动化。如果您的自动化脚本想要处理数据，Bash 并不合适——因为它是一种命令语言。另一方面，Python 可以用最少的语法处理数据，并像 Bash 一样用最少的语法调用其他进程。所以如果你学了 Python，就可以写出干净的、功能齐全的自动化脚本。

此外，Python 非常适合在线编码挑战和快速解决问题的技术面试。例如，看看在 Python 中删除列表的重复项有多容易，甚至不需要使用任何导入语句。

```
def remove_duplicates(L):
    return list(set(L))
```

# 去

Go 是一种静态类型的通用编程语言，使用类似 C 的语法设计。它具有许多其他现代编程语言所具有的特性，比如垃圾收集、内存安全和并发支持。Go 编程语言具有开发人员友好的语法，但与其他流行语言相比，它的性能仍然令人印象深刻。Go 语言的设计给我们所有程序员上了很多有价值的课。

Go 只有 25 个关键字，但它提供了构建任何软件系统所需的所有功能。大多数编程语言都添加了大量的语言特性，让开发人员的工作变得更加轻松。但事实上，新的语言特性很快就超越了你的代码库。此外，许多有经验的程序员通常不关心该语言最新的语法技巧和提示。

Go language 的设计教会了我们如何用最小和完整的解决方案解决工程问题。试想， [25 个](https://golang.org/ref/spec#Keywords)关键词足以设计出一个全功能的编程语言。另一方面，现在 C++有 [97 个](https://en.cppreference.com/w/cpp/keyword)关键词，Rust 有 [54 个](https://doc.rust-lang.org/reference/keywords.html)关键词——可能开发团队会添加更多。Go 支持面向对象编程，甚至不需要将`class`定义为关键字。当你学习围棋概念时，你总是倾向于用简单有效的方法解决问题。Go 语言的生态系统是学习更好的软件设计模式的好方法。
