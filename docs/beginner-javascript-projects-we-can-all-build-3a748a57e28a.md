在软件开发领域，熟能生巧。所以要尽可能多的找方法练习编程。

在这篇文章中，我们将看看一些提高我们编码技能的实践项目想法。

# 时钟应用程序

我们可以用 JavaScript 构建自己的时钟，在浏览器中显示我们计算机上的时间。

我们所要做的就是使用`setInterval`定时器函数并将`Date`对象格式化为日期字符串。

# 小费计算器

小费计算器让我们从小计中计算小费。我们可以发挥想象力，将不同国家的小费规则编程到我们的应用程序中，或者让用户输入自己的小费率。

# 地图应用程序

像谷歌地图和必应这样的地图提供商通过 API 提供数据，让我们的应用程序使用他们的数据。

这包括地图数据，包括地图本身。我们可以为位置添加标签，并添加要在地图中显示的位置数据。

这让我们可以制作很多东西，比如餐厅定位器、活动定位器等等。

# 比赛

我们可以用 JavaScript 构建游戏。我们所要做的就是使用像 [Phaser](https://phaser.io/) 这样的框架来构建游戏。

我们必须了解游戏循环，这是一个运行到程序结束并更新屏幕上图形的循环。

此外，我们还必须知道如何将游戏资源加载到我们的游戏程序中，制作它们的动画，然后做一些像碰撞检测这样的事情来检测我们希望在游戏中出现的任何碰撞和移动。

# 手电筒应用

我们可以构建一个手电筒应用程序，当我们点击一个按钮来改变它的颜色和用它做其他事情时，它会改变颜色。

# 报价应用程序

一个在屏幕上显示随机报价的应用程序是一个容易上手的初学者项目。

我们可以显示来自我们选择的 API 的报价，如[FavQs.com](https://favqs.com/api)API，并将它们显示在我们的屏幕上。

# 计数器应用程序

我们可以构建一个应用程序，当我们点击按钮来做这些动作时，它会增加或减少计数。

它让我们练习在点击按钮时操作 DOM 来更新计数器。

# 购物车应用程序

这是一个应用程序，让我们从屏幕上显示的销售杂货项目列表中添加和删除杂货商购物车项目。

我们可以从待售商品列表中添加和删除商品，并将它们放入购物车。

完成后，我们可以编辑购物车中的商品。一旦我们编辑完购物车中的商品，我们就可以得到总数并结帐。

购物结束后，我们还可以通过电子邮件发送收据，这是一项额外的功能。此外，我们可以做一个管理前端来改变项目的库存。

# 课程应用

我们可以制作一个类似 Coursera 和 Khan Academy 的应用程序，人们可以在其中查看书籍和观看视频教程。

为此，我们需要一个 UI 来显示课程及其内容，还需要一个管理 UI 来管理课程内容。

# 抽认卡应用

抽认卡是我们用来记笔记、学习和记忆东西的卡片。

我们可以让用户创建抽认卡，让他们通过添加新的卡片和把东西放在那里来学习。

然后我们为他们添加一些功能来编辑和删除卡片。

# 预算应用程序

这个应用程序将允许用户添加他们的支出和收入数据，然后根据他们设定的预算检查他们花了多少钱。

它让用户清楚他们是否还有钱，或者他们是否达到了他们的储蓄目标。

我们可以让他们设定目标，并根据目标进行衡量。

# 笑话 App

笑话应用程序显示硬编码到我们的应用程序中或随机来自 API 的笑话。

我们为查看者添加一个按钮来显示一个新笑话。

# 随机人 App

无论出于何种目的，生成虚假数据总是一件痛苦的事情。因此，我们可以制作一个应用程序来生成和显示一些人的虚假数据。

此外，我们可以让用户批量创建它们，这样，如果他们想创建大量的虚假数据，他们就可以这样做。

假数据可以导出为 CSV、JSON 或 SQL。

# 结论

我们可以创建大量有用的应用程序用于实践。它们不一定要有用，只要有趣，帮助我们学习编程就行。
