**什么是异步编程？**

异步编程使您的程序能够启动一个可能长期运行的任务，然后不必等到该任务完成，而是能够在任务运行时继续响应其他事件。一旦任务完成，你的程序就会得到结果。这将使应用程序能够在后台完成任务的同时继续工作。
