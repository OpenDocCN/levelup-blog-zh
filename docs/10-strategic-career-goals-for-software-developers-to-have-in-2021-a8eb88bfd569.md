# 什么是战略性职业目标？

战略性职业目标是那些将推动你在一年、两年甚至五年内朝着你想要的职业方向前进的目标。它们有一个目的，反映了你的专业视野，对你的职业成功有重大影响。

# 结论

![](img/4fd479b30a55ca0cfa169ac37d276ba5.png)

杰斯·贝利在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

总而言之，软件开发人员在 2021 年应该瞄准的战略性职业目标是:

1.  成为某个领域、技术或语言的专家
2.  在规模或复杂性为 X 的系统上工作
3.  找到一个特定的职位
4.  在一家著名的科技公司工作
5.  开始一场侧推
6.  销售你写的第一个软件
7.  指导某人，看他们随着时间的推移变得更好
8.  帮助团队中的一名新人加入并开始工作
9.  在聚会、会议或活动中，就你感兴趣的技术话题发表演讲
10.  写一篇有用的博客或一篇被大量分享的文章

引用《高效能人士的 7 个习惯》一书的作者斯蒂芬·柯维的话，“关键不在于优先安排你日程表上的事情，而在于安排你的优先事项。”如果你需要更多的指导，看看这个[数字规划器](https://gum.co/techcareerplanner)，这是我个人设计的，给雄心勃勃的软件开发人员清晰、一致和勇气，让他们在你的职业生涯和生活中前进并达到新的高度。
