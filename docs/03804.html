每个开发人员都应该知道的顶级VSCode键盘快捷键

原文：https://levelup.gitconnected.com/top-vscode-keyboard-shortcuts-every-developer-should-know-9e2118bbc2cb?source=collection_archive---------13-----------------------#2020-05-27

这里列出了一些基本的VSCode键盘快捷键，每天至少可以为您节省几分钟。

  • 上下移动线条-T0】
  • 跳转到匹配括号-T1】
  • 替换下一个类似的词——T2】
  • 向上/向下复制行-T3】
  • 代码折叠- command + option + [
    代码展开-     command + option + ]
  • 选择下一场比赛--command + d
  • 取消选择下一个匹配- command + u
  • 打开和关闭侧栏-T8】
  • 添加光标上方/下方-T9】
  • 添加多个光标-T10】
  • 删除行 - shift + command + k或command + delete
  • 在finder/explorer中打开当前文件 - command + k然后r
  • 禅模式- command k 然后 z
  • 注释代码- command + /
  • 将光标添加到所有选中的行 - shift + option + drag mouse
  • 分屏- command + \
    切换分屏水平和垂直视图 - command + option + o
