# 大学 vs 训练营 vs 自学:哪一个最适合从事科技行业？

## 对这些受欢迎的教育选择进行诚实的比较，来自一个做过这三件事的人。

![](img/432f28b80d8e13cd80a58ffd0e8d4573.png)

直到最近，在包括科技在内的大多数领域，上大学并获得学位是获得高薪工作的唯一途径。但随着该行业的快速发展，对软件开发人员、工程师、数据科学家和程序员的需求超过了大学的供应能力。

这为编写训练营和在线课程奠定了基础。更便宜、更快捷、更务实的技术工作途径。但是这些真的是学位的可行替代品吗？或者从另一方面来说，有了这些新鲜的选择，学位是不是浪费时间？

**各有什么利弊，每个选项对谁合适？**

我亲身经历了这三种情况，所以我觉得自己是独一无二的，可以针对这个问题提供一些见解。这将是对我在线课程、编码训练营和大学经历的反思，以及我从每一个经历中学到的东西。

# 自学成才

我第一次接触编程是通过在线课程。我注册了 Udemy 初学者 C#课程，并在几周内完成了它。它没有任何先验知识，教授面向对象编程的基础知识，中级和高级后续课程由同一个 Udemy 讲师提供。我还在开普敦大学学习了在线网页设计课程。

## 赞成的意见

1.  迄今为止最实惠的选择。
2.  有大量的课程和网站可供选择，在那里你可以学习任何你想学的语言/话题。
3.  按照自己的节奏学习的能力。
4.  可以远程学习。

## 骗局

1.  几乎没有什么“全局”指导。
2.  找一个社区相对来说比较难。
3.  你不会学到和同学一起学习时学到的软技能。
4.  大量的独立工作和奉献精神是必需的。

## 裁决

我很喜欢这门课，并且觉得我确实对编程的基础有了基本而扎实的理解。然而，这还不足以让你被录用。你需要一个个人项目组合，你需要参加更多的课程来发展技能。

例如，如果你想找一份全栈 web 开发人员的工作，那么从 HTML 和 CSS 课程开始，然后是 Javascript 课程，接着学习 React 和 Django 之类的框架是一个好主意。

基本上，个别课程可能很棒，但你需要以一种有意义的方式将它们拼凑在一起，并与你的目标相关，这需要你在没有指导的情况下找到答案。

理论上，你可以在网上自学你在计算机科学学位中学到的一切(很可能是免费的)，但是除非你愿意，否则这个*将是一个非常困难的任务，这一点我怎么强调都不为过。*

# *训练营*

*训练营提供了一个比在线课程更有条理的学习环境，同时也比大学更短、更便宜。长度从大约 3 个月到 2 年不等，在线或校园，有各种路径和进度可供选择，这种学习方法是专为让人们快速进入技术职业而设计的。尤其是那些希望转行赚钱的职场人士。*

*在我为期 3 个月的训练营中，我们学习了 Python 编程、Java 面向对象编程、一些数据科学基础知识以及职业导向技能，如软件开发流程、敏捷开发等。我们在校园里有一个讲师和助教。我们的任务提交被评分，我们得到了反馈。我们还完成了一些项目来增加我们的投资组合，其中一些我至今还保存在 GitHub 上。*

## *赞成的意见*

1.  *现代速成课程。*
2.  *高度注重实践技能。*
3.  *学习和参与的同龄人，以及教学支持。*
4.  *完成后的一些职业前景。*

## *骗局*

1.  *缺乏理论基础。*
2.  *有些可能非常昂贵。*
3.  *通常倾向于 web 开发或全栈，这可能不是你想走的路。*
4.  *非常夸张的职业成果，可能有点“欺诈”*

## *裁决*

*我真的很喜欢训练营，在我完成了 3 个月的课程后，我确实得到了一份工作，所以我可以告诉你**是的，这是可能的**。但是它能和大学毕业生的工作类型和薪水相比吗？那是一个低薪的初级职位。但是，考虑到我以前没有经验，还是赢了。*

*想想看:像谷歌、亚马逊、微软和苹果这样的梦想公司每年都有成千上万来自世界上最好的学校的计算机科学毕业生与他们拼命争夺工作。有了这样的人才储备，他们还需要接受一个 6 个月的训练营毕业生吗？*

*是的，你可以找到一份工作。但事实是，你仍然可能不具备大学毕业生所具备的技能。这绝不是一条规则，但你必须竭尽全力让自己与众不同，除非你是一名职业生涯中期的职业人士，希望进入一个新的方向，提升技能或改变领域，在这种情况下，训练营可能是完美的。*

# 大学

我在大学一年级，我很喜欢它。我很荣幸有处于该领域前沿的教授。例如，我第一学期的函数式编程课程是由流行的函数式语言的主要创造者之一 Haskell 教授的。在大学里，你周围都是这种人，在这种环境中学习的好处怎么说都不为过。

然而，我的部分课程确实显得多余和过于理论化，但这是学习计算机科学绝对一阶基础的唯一方法，而这是训练营不会教你的。此外，我已经为即将到来的夏天准备了一些可能的实习机会**(所有这些都只接受大学生的申请)。**

## 赞成的意见

1.  信誉和声望，其影响是相当大的。
2.  深厚的理论知识。
3.  向领域和网络专家学习的机会。
4.  你可能会被聪明、勤奋和有竞争力的同事所包围。

## 骗局

1.  可能是出了名的昂贵(尤其是在美国，对于世界各地的国际学生来说，如果你住在欧洲，就不会这么贵)。
2.  一些课程有些过时，大学适应得很慢。
3.  显然，并非所有的大学生来平等。学校的名字对你简历的影响很大程度上取决于你学校的声誉和排名。
4.  制度和机构并不适合每个人，作为学生，你必须面对它。

## 裁决

大学是传统的，贵，(安全？)选项。现在你会听到许多年轻人抱怨他们的学位毫无价值，甚至毕业后也找不到工作，但计算机科学和类似的学位却不是这样。

事实是，这些都是令人垂涎的、竞争激烈的学位课程，大型科技公司通常会在顶尖学校周围设立办公室，只是为了抢夺它们的毕业生。然而，这并不意味着它总是正确的选择。

# 那么，每个选项对谁最有利呢？

埃隆·马斯克有一句名言:在特斯拉工作不需要学位。虽然这可能是一种情绪，但我不会放弃大学，而赞成在 codeacademy 上自学 Python。如果你看一下特斯拉网站上的招聘要求，他们会提到类似*“对基本计算机科学概念的出色掌握”*这样的事情，你可能很难与大学毕业生的理论知识竞争。

## 在线课程

对于那些可能已经就业并希望利用自己的时间发展特定技能的人来说，在线课程是一个很好的选择，这些人对自己获取信息的能力充满信心，并愿意付出额外的努力来证明自己接受了高质量的教育。

## **训练营**

如果你在大学主修其他专业，并且已经是一名正在寻找职业改变的职业人士，回到大学并不总是必要的，训练营可能是你的完美选择。(不过，你也可以考虑计算机科学硕士学位)

## 大学

在我看来，如果你还年轻，而且负担得起，大学仍然是获得你梦想中的科技工作的最可靠途径——更不用说大学经历的乐趣了，你可能会买也可能不会买。作为候选人，你将有机会参加交流项目、俱乐部、社团和实习，这些都会增加你的价值。你只需要决定这是否值得每年几万英镑/美元/欧元。*
