滚动到页面顶部的CSS和JavaScript解决方案

CSS解决方案

1.基本CSS解决方案

<html>
  <body>
     <a id="top"></a>
     <!-- the entire document -->
     <a href="#top">Jump to top of page</a>
  </body>
</html>

下面是效果，你可以注意到它直接跳到页面顶部，没有任何平滑效果。

2.平滑滚动的CSS解决方案

如果你想让它平滑地滚动到顶部，你可以在CSS中这样做:

html { scroll-behavior: smooth; }

下面是平滑滚动效果的结果。

但是，请确保您知道并非所有浏览器都支持scroll-behavior

JavaScript解决方案

您可能需要用JavaScript触发滚动，在这种情况下，有一些选项:

  1. window.scrollTo()
  2. document.documentElement.scrollTop()
  3. window.scroll()
  4. document.documentElement.scrollIntoView()

1.window.scrollTo()

window.scrollTo(0, 0)肯定会将窗口(或任何其他元素)滚动回顶部。的滚动行为也是由CSS决定的。

例如没有CSS平滑效果:

<html>
  <body>
    <a onClick="window.scrollTo(0, 0)">Top</a>
  </body>
</html>

结果是:

以CSS scroll-behavior为例:

<html>
  <style>
    html { scroll-behavior: smooth; }
  </style>
  <body>
    <a onClick="window.scrollTo(0, 0)">Top</a>
  </body>
</html>

会给你平滑的滚动效果，但是要注意scroll-behavior的浏览器支持问题(如前所示)。

2.窗口.滚动

如果你没有使用CSS来获得平滑的滚动效果，你可以在JavaScript window.scroll()中强制平滑

window.scroll({
   top: 0,
   left: 0,
   behavior: 'smooth'
});

例如

<html>
  <body>
    <a onClick="window.scroll({ top: 0,left: 0,behavior: 'smooth' });">Top</a>
  </body>
</html>

也会给你带来流畅的滚动效果:

但是要注意浏览器对ScrollToOptions的支持

3.document . document element . scroll top

属性获取或设置元素内容垂直滚动的像素数。document.documentElement.scrollTop正在根元素上使用scrollTop(<html>元素)。

<a onClick="document.documentElement.scrollTop = 0;">Top</a>

这是结果，显然没有任何平滑效果。但是几乎所有的浏览器都支持这个属性。

4.document . document element . scrollintoview()

Element接口的scrollIntoView()方法滚动元素的父容器，这样用户就可以看到调用scrollIntoView()的元素。document.documentElement.scrollIntoView()是scrollIntoView用在根元素(<html>元素)上的特例。

一些浏览器支持scrollIntoViewOptions，允许过渡动画。

例如:

<a onClick="document.documentElement.scrollIntoView({ behavior: 'smooth' });">Top</a>
