成为一名优秀的远程工作者需要很多训练。我们必须自己做每件事，没有人看着。

在本文中，我们将研究一些可以成为优秀远程开发者的习惯。

# 知道什么时候下班

因为我们在家里工作，我们不必从办公室回家。

所以，远程办公的时候要知道什么时候应该停止工作。

我们应该争取在办公室的同一时间下班。这样，我们就可以休息一下了。

# 享受远程工作的好处

远程工作很棒，因为我们可以控制我们的工作场所，我们可以做任何我们想做的事情，只要我们完成我们的工作。

因此，我们应该通过保持灵活的时间表来接受远程工作的好处，并在工作之外以不同的方式享受在办公室工作的乐趣。

现在我们可以做的事情太多了，我们不再局限在办公室里，所以我们应该在办公室里尽情享受。

# 远程工作自动化

开发人员都想让事情自动化。因此，我们也应该使我们的工作自动化。

我们可以在 Slack 等大多数消息系统上设置自动通知时间表，这样我们都知道我们离开了 Zapier 等键盘。

此外，我们可以使用 Zapier 自动发送视频会议链接。

它还可以在 Slack 上自动跟人打招呼。

我们也可以用 IFTTT 做同样的事情，自动创建食谱，当有事情发生时自动发送通知，还有很多我们可能还没有想到的事情。

作为开发人员，我们也可以编写自己的脚本来实现自动化，只要我们能够使用库和 API 来帮助我们实现工作自动化，这就开辟了无限的可能性。

# 创造我们自己的水冷却器

聊天室是饮水机聊天的好地方。我们可以在不同的人之间建立聊天室，这比在办公室的饮水机旁聊天更私密。

这是因为我们可以创建私人或公共聊天室，只在我们聊天的人之间进行。

此外，我们可以在网上发送 gif 和谈论天气，以及我们可以聊的其他事情。

# 使用在线白板

在线白板和真正的白板一样好，如果不是更好的话。

在线白板可以是任意大小，而实体白板只能有这么大。

我们可以用它来写任何东西，上传图片，添加链接，附加文件，以及做更多的事情，这以常规白板无法做到的方式增加了对话。

此外，我们可以保存在线白板的快照，以便稍后可以返回到我们所拥有的内容。

对于普通的白板，我们能做的最好的事情就是给它拍张照片，这并不是很好。

因此，我们应该利用物理白板不具备的所有功能。

# 独自头脑风暴

因为我们在自己的家里工作，我们可以安静地单独进行头脑风暴，而不是在嘈杂的开放式办公室里思考。

这是一个关上我们房间的门，集思广益的好机会。

许多人独自头脑风暴更好，所以我们可以通过安静地独自头脑风暴来充分利用我们的家庭办公室。

# 决定我们是否需要开会

如果真的需要，我们应该只召集会议。如果聊天信息和电子邮件足够好，那么我们就可以使用它们，不要用额外的会议打扰别人。

然而，如果我们根本没有会议，也许我们可以创建一个可选的会议，以便人们可以在线社交。

站立会议可以在聊天上进行，因为它只是用来更新人们对我们的进展，例如。

# 记录会议

我们可以记录会议，而不会让人在远程会议中感到毛骨悚然，因为人们不会看着摄像机记录我们自己。

这很好，因为我们可以记录我们的会议，而不是让人们做笔记。

如果我们想回忆会议中的一些信息，我们也可以以我们想要的速度倒回去观看记录的会议。

# 结论

我们应该充分利用远程工作的优势，比如有一个丰富的白板来进行头脑风暴和记录会议，以便事后回忆信息。

此外，我们需要设定界限，以便我们都可以休息一下。
