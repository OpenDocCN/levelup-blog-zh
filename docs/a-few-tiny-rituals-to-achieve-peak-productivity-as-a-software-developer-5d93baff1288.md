# 作为一名软件开发人员，实现最高生产力的一些小仪式

## 每天都有小进步，日积月累

![](img/0e9f5778c076c91439f7cd7238b3b25d.png)

凯瑟琳·拉威利在 [Unsplash](https://unsplash.com?utm_source=medium&utm_medium=referral) 上拍摄的照片

***一杯咖啡*** 是当我们感到无精打采时想到的东西。咖啡让人清醒，但它不是我们每天面对的生产力的解决方案。

经常发生的情况是，你知道解决方案，但你需要从别人那里听到。我没有什么不同。我们将讨论一些小的行动或习惯，它们能在很大程度上提高你的日常工作效率

# 分析你在 EOD 的成就

> “如果你不能衡量它，你就不能改善它。”—彼得·德鲁克

从自省开始。每天结束时，衡量一下你完成了什么。这个想法是你的一天是多么富有成效。 ***还能更好吗？你能做更多的事情吗？你能以更好的方式做事吗？***

这是一个反馈循环，可以帮助你学习几个月。你开始了解你需要改进的地方。如何改善后面来，先搞清楚自己做的对与错。

# 计划你的一天

> "如果你不计划你的一天，别人或其他事情会的."—尼尔·埃亚尔

一旦你知道如何衡量你的效率和生产力，你需要计划你的一天，使之更有效率和生产力。

如今有太多的干扰，社交媒体、视频、新闻、人等等。你口袋里的矩形设备总是不断发出通知，渴望引起你的注意。

所以，如果你不计划你的一天，你会让这些干扰为你计划。每天结束时，反思一下第二天的计划。

# 喝一杯咖啡因的最佳时间

我每天喝一杯咖啡和两杯茶。一杯咖啡因帮助我保持清醒，专注于我想做的事情。

一杯黑咖啡需要大约 8-14 个小时才能从你的身体里冲出来。作为我日常生活的一部分，我喜欢在 11-11:30 之间喝杯咖啡。我们通常在上午开会，然后我休息一下，喝杯咖啡。

它帮助我专注于我的工作，至少我是这样认为的。在我晚上睡觉前，它已经离开了我的系统。晚上喝咖啡可能会打扰你的睡眠，从而影响你第二天的工作效率。

所以，计划一下你什么时候想喝一杯咖啡因。

# 牢记你的长期目标

> "永远不要让短期的欲望阻碍长期目标的实现."柯蒂斯·马丁

定期回顾你的长期目标是让你走上正轨的必要条件。这个仪式有两个好处，它让你不断提醒自己你的目标，并帮助你衡量你的长期进展。

这与你的日常任务计划密切相关。一些专家说你应该每天回顾你的长期目标。我建议从每周至少做一次开始。

# 管理会议

> "会议越长，完成的就越少。"蒂姆·库克

在工作中，我们参加很多会议。它们是我们工作生活的一部分，也很重要。自从疫情事件爆发以来，随着在家工作成为强制性的，我们花在会议上的时间急剧增加。音频/视频电话会议成为与同事联系的唯一方式。

这些会议给我们一种成就感，但是大多数会议都是冗长且不必要的。学会让会议简短，并专注于议程。这可能会节省你很多时间。

# 集中编码时间

> "专注于高效而不是忙碌."—蒂姆·费里斯

这个想法是让 2-3 个小时的专注工作完全远离干扰。工作中有不同种类的干扰——你的电话、电子邮件、聊天、同事寻求帮助、会议等等。

拥有专注、不分心、不被打扰的时间可以帮助你完成比一天所能完成的更多的事情。编码需要专门的时间，没有干扰，也没有上下文切换。给自己这段时间，你会在短时间内完成更多。

# 根据任务带来的价值来区分任务的优先级

> "关键不在于优先处理你日程表上的事情，而在于安排好你的优先事项."—斯蒂芬·柯维

根据什么对组织的业务重要来确定任务的优先级。我们经常试图完成任务清单上的任何事情，而没有着眼于大局。

知道哪个任务应该优先于对业务发展非常重要的任务。

# 每工作 25 分钟，休息 3-5 分钟

这种技术叫做[番茄工作法](https://en.wikipedia.org/wiki/Pomodoro_Technique)。人类通常的注意力持续时间大约是 25 分钟。所以试着最大化你的注意力，在 25 分钟的训练后休息一下。

我通常会在每次会话时使用闹铃。这个警报给你一个提示，让你分析你是专注于手头的问题还是偏离了它。它给你一个机会回到重要的任务上。

# 打盹

午饭后，我会在躺椅上小睡 15 分钟来给自己充电。这个午睡的工作原理类似于 ***垃圾收集*** 。你的大脑会把所有正在发生的事情冲洗出来，从而提高你的表现。

# 包扎

我已经把这些习惯融入了我的生活。这些习惯需要时间来培养。你可能需要一段时间才能在工作中看到它们的好处。但是如果你坚持这样做几天，你会更专注，更有效率，最重要的是你会有更多的时间可以支配。
