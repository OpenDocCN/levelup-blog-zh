函数式编程是一种编程范式，侧重于使用函数来解决问题。JavaScript 是一种支持函数式编程的多范例编程语言，具有以下函数属性:

1.  **一等函数**:在 JavaScript 中，函数是一等公民，这意味着它们可以像其他任何值一样被对待。这意味着函数可以赋给变量，作为参数传递给其他函数，或者从函数返回。
2.  **高阶函数**:在 JavaScript 中，函数可以作为参数传递给其他函数，或者从函数返回。这允许您编写高阶函数，即操作或返回其他函数的函数。
3.  **Lambda 函数**:在 JavaScript 中，你可以使用 Lambda 函数(也称为匿名函数)来定义函数，而无需给它们命名。这对于内联定义函数很有用，不需要创建单独的函数声明。
4.  闭包:在 JavaScript 中，闭包是可以访问定义它们的外部函数的变量和参数的函数。这允许您创建能够访问全局范围内不可用的数据的函数。
5.  **Currying** :在 JavaScript 中，Currying 是将一个接受多个参数的函数分解成一系列接受单个参数的函数的过程。这对于创建具有固定数量参数的函数或实现函数的部分应用非常有用。

这些是支持函数式编程的 JavaScript 的主要功能属性。通过使用这些属性，您可以用 JavaScript 编写函数式程序，并利用函数式编程的优势。
