问题陈述

当您在Visual Studio代码IDE中工作时，有时会需要您所工作的计算机上的IP地址。根据你的操作系统，打开浏览器太麻烦了；在互联网上搜索会显示一长串IP地址的相关命令，并最终挑选出您需要的一个。

此外，如果IP地址不是静态的，手动重复这个过程是一件令人沮丧的工作。作为一名开发人员，您不希望整天兜圈子，浪费宝贵的时间。

对于MICROSOFT WINDOWS

如果你的机器运行在微软视窗操作系统上，你可以打开命令提示符并输入“ipconfig ”,它会给出一个详细的结果，如下所示。

对于MAC OS

如果你在Macintosh笔记本上工作，你可以在终端中输入“ifconfig ”,产生一个综合的输出，如下图所示。

对于LINUX操作系统

如果您使用基于Linux的操作系统，您将导航到bash终端，键入' ifconfig ',并产生如下图所示的输出。
