# 如果你没有这样做过，千万不要申请工作面试

> 原文：<https://levelup.gitconnected.com/never-apply-for-a-job-interview-if-you-havent-done-this-bf20f1bedbcf>

## 这是一场灾难——你可能会不断被拒绝

![](img/2e6a69d4c47cb6c90f16e5a7dd6a3d4b.png)

图片由[穆罕默德·哈桑](https://pixabay.com/users/mohamed_hassan-5229782/?utm_source=link-attribution&utm_medium=referral&utm_campaign=image&utm_content=3531130)来自 [Pixabay](https://pixabay.com//?utm_source=link-attribution&utm_medium=referral&utm_campaign=image&utm_content=3531130)

有些人认为他们受到了歧视。他们已经申请了几次工作面试，但不断遭到拒绝。他们根本得不到回音，或者最终被拒绝。

他们不知道原因，开始怀疑自己，认为自己就是问题所在。

当他们看着他们的同龄人时，他们正在茁壮成长。他们从一份工作跳到另一份工作。面试和工作邀请不断向他们涌来。

对于一个不断被拒绝的人来说，这是令人沮丧的。但现在不是了。我会让你知道这个职业秘密。这会让你的求职游戏更上一层楼。

一旦我透露这一点，你可以期待更多的采访邀请。得到你想要的工作的可能性会大大增加。

拒绝是痛苦的，多次拒绝把你压垮，让你的自信心跌到谷底。

如果你被多次拒绝，问题不在你。问题是你如何做这件事。

不管你认为自己有多聪明，冷冰冰地申请工作是绝对不行的。你根本就不应该这么做。无论是大的科技公司还是小公司。没关系。

你将扮演另一个试图找工作的 *x 人*。你的申请很可能在 HR 里丢失或遗忘。

还有另一种方法。正确的和推荐的方式。不要申请编码面试，如果你没有…..寻找推荐或内部推荐。

等等！我知道问题不断涌现，我会努力解决其中的大部分问题。

这合乎道德吗？是啊！这是完全合法和道德的。事实上，这是值得推荐的。如果你没有参加过面试，你就不应该申请。它让你得到了展示你的个性和技能所需要的面试机会。

谁也不认识怎么办？没问题。这是我将在下一节讨论的内容。我还会告诉你怎么做。

有推荐人会增加你申请的份量。你可以确定你的申请不会被 HR 遗忘，你从陌生人转变为推荐人。

那么当你知道没有人在公司工作时，你是如何做到的呢？它简单且可重复。这是你应该做的。

**一、**去 Linkedin 找你想合作的公司。

![](img/892317d143d594315339f6ceab88e3f6.png)

图片来源:LinkedIn

**二。**转到*人员选项卡*并在你的关系中找到为该公司工作的朋友。

![](img/7fedb086bd5badf36950a1a10d9707ca.png)

图片来源:LinkedIn(来自搜索)

您可以如上图所示直接从初始搜索中访问*人员选项卡*,或者如下图所示直接从公司的 LinkedIn 页面访问。

![](img/0b6e8cc6f480cb5a3e177423ef8b0095.png)

图片来源:LinkedIn

如果你有相当多的人脉，或者你已经在这个行业呆了很长时间，你很可能会和那个公司的某个人有联系。

它会显示你学校的人，你所在地区的人，以及为那家公司工作的二级和三级关系。有了所有可用的选项，你就不能缺少一个你可以要求推荐你的人。

**三世。**最后一步是直接给其中一个人发信息，请他们推荐你。

他们为什么要这么做？如果你被录用，他们会得到报酬。大多数公司会为推荐的员工发放奖金。

如果没被录用呢？没关系，他们不会受到惩罚或承担后果。你要求推荐的那个人不会有任何损失。

盲目求职是永远得不到回音和面试失败的秘诀。你不应该这样做。

在提交你的申请之前，确保你已经被公司内部的人推荐了。这样做会大大增加你获得第一次面试的机会，这是非常重要的一步。

如果你足够优秀，你被聘用的机会也会大大增加。
